Detailed Description

Device command to instruct the underlying platform to pair a user account to the targeted device.

If successful, the platform should then send an DeviceConfigurationEvent to signal that the device configuration has been changed. In response, a QueryUserIdCommand may be sent to fetch the paired user ID from the device.
