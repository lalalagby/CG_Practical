void BakeConfiner (float maxComputationTimePerFrameInSeconds)
 Creates shrinkable polygons from input parameters. The algorithm is divide and conquer. It iteratively shrinks down the input polygon towards its shrink directions. If the polygon intersects with itself, then we divide the polygon into two polygons at the intersection point, and continue the algorithm on these two polygons separately. We need to keep track of the connectivity information between sub-polygons.
