Detailed Description

Scale a float value by a constant factor.

This processor is registered (see InputSystem.RegisterProcessor<T>) under the name "scale".

// Bind to left trigger on the gamepad such that its values are scaled by a factor of 2. new InputAction(binding: "&lt;Gamepad&gt;/leftTrigger", processors: "scale(factor=2)");
