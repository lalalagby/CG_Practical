Detailed Description

Write accessor to a binding on either an InputAction or an InputActionMap.

Both InputAction.bindings and InputActionMap.bindings are read-only. To modify bindings (other than setting overrides which you can do through InputActionRebindingExtensions.ApplyBindingOverride(InputAction,int,InputBinding)), it is necessary to gain indirect write access through this structure.

playerInput.actions["fire"]
.ChangeBinding(0)
.WithPath("<Keyboard>/space");
See also
AddBinding(InputAction,InputBinding), ChangeBinding(InputAction,int)

void UnityEngine.InputSystem.InputActionSetupExtensions.BindingSyntax.Erase ( )
inline

Remove the binding.

If the binding is a composite (see InputBinding.isComposite), part bindings of the composite will be removed as well.

Note that the accessor will not necessarily be invalidated. Instead, it will point to what used to be the next binding in the array (though that means the accessor will be invalid if the binding that got erased was the last one in the array).

◆ NextPartBinding()

BindingSyntax UnityEngine.InputSystem.InputActionSetupExtensions.BindingSyntax.NextPartBinding ( string partName)
inline

Iterate to the next part binding of the current composite with the given part name.

Parameters
partNameName of the part of the binding, such as "Positive".
Returns
An accessor to the next part binding with the given name or an invalid (see valid) accessor if there is no such binding.
Exceptions
ArgumentNullExceptionpartName is null or empty.

Each binding that is part of a composite is marked with InputBinding.isPartOfComposite set to true. The name of the part is determined by InputBinding.name (comparison is case-insensitive). Which parts are relevant to a specific composite is determined by the type of composite. An Composites.AxisComposite, for example, has "Negative" and a "Positive" part.

// Delete first "Positive" part of "Axis" composite.
action.ChangeCompositeBinding("Axis")
.NextPartBinding("Positive").Erase();
See also
InputBinding.isPartOfComposite, InputBinding.isComposite, InputBindingComposite
