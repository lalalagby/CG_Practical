Detailed Description

Wraps around mechanisms for listening in on the InputEvent stream made available through InputSystem.onEvent.

This struct can be used to add (op_Addition) or remove (op_Subtraction) callbacks directly to/from the event pipeline.

Alternatively, it can be used as an IObservable to Subscribe observers to the event stream. See Observable for extension methods to set up various observer mechanisms.

InputSystem.onEvent
.ForDevice(Mouse.current)
.Call(evt =>
{
foreach (var control in evt.EnumerateChangedControls())
Debug.Log($"Control {control} on mouse has changed value");
});
