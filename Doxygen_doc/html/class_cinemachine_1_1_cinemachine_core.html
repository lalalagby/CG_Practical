Detailed Description

A singleton that manages complete lists of CinemachineBrain and, Cinemachine Virtual Cameras, and the priority queue. Provides services to keeping track of whether Cinemachine Virtual Cameras have been updated each frame.

Member Enumeration Documentation

◆ Stage

enum Cinemachine.CinemachineCore.Stage

Stages in the Cinemachine Component pipeline, used for UI organization>. This enum defines the pipeline order.

Enumerator
Body 

Second stage: position the camera in space.

Aim 

Third stage: orient the camera to point at the target.

Noise 

Final pipeline stage: apply noise (this is done separately, in the Correction channel of the CameraState)

Finalize 

Post-correction stage. This is invoked on all virtual camera types, after the pipeline is complete.

◆ OnTargetObjectWarped()

void Cinemachine.CinemachineCore.OnTargetObjectWarped ( Transform target,
Vector3 positionDelta )
inline

Call this to notify all virtual camewras that may be tracking a target that the target's position has suddenly warped to somewhere else, so that the virtual cameras can update their internal state to make the camera warp seamlessy along with the target.

All virtual cameras are iterated so this call will work no matter how many are tracking the target, and whether they are active or inactive.

Parameters
targetThe object that was warped
positionDeltaThe amount the target's position changed
