Detailed Description

An add-on module for Cinemachine Virtual Camera that places an image in screen space over the camera's output.

Member Enumeration Documentation

◆ FillStrategy

enum Cinemachine.CinemachineStoryboard.FillStrategy

How to fit the image in the frame, in the event that the aspect ratios don't match.

Enumerator
BestFit 

Image will be as large as possible on the screen, without being cropped.

CropImageToFit 

Image will be cropped if necessary so that the screen is entirely filled.

StretchToFit 

Image will be stretched to cover any aspect mismatch with the screen.

◆ StoryboardRenderMode

enum Cinemachine.CinemachineStoryboard.StoryboardRenderMode

Camera render modes supported by CinemachineStoryboard.

Enumerator
ScreenSpaceOverlay 

Renders in camera screen space. This means, that the storyboard is going to be displayed in front of any objects in the scene. Equivalent to Unity's RenderMode.ScreenSpaceOverlay.

ScreenSpaceCamera 

Render using the vcam on which the storyboard is on. This is useful, if you'd like to render the storyboard at a specific distance from the vcam. Equivalent to Unity's RenderMode.ScreenSpaceCamera.
