int Cinemachine.ICameraOverrideStack.SetCameraOverride ( int overrideId,
ICinemachineCamera camA,
ICinemachineCamera camB,
float weightB,
float deltaTime )

Override the current camera and current blend. This setting will trump any in-game logic that sets virtual camera priorities and Enabled states. This is the main API for the timeline.

Parameters
overrideIdId to represent a specific client. An internal stack is maintained, with the most recent non-empty override taking precenence. This id must be > 0. If you pass -1, a new id will be created, and returned. Use that id for subsequent calls. Don't forget to call ReleaseCameraOverride after all overriding is finished, to free the OverideStack resources.
camAThe camera to set, corresponding to weight=0.
camBThe camera to set, corresponding to weight=1.
weightBThe blend weight. 0=camA, 1=camB.
deltaTimeOverride for deltaTime. Should be Time.FixedDelta for time-based calculations to be included, -1 otherwise.
Returns
The override ID. Don't forget to call ReleaseCameraOverride after all overriding is finished, to free the OverideStack resources.
