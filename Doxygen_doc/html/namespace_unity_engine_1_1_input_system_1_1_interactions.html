Enumerator
PressOnly 

Perform the action when the button is pressed.

Triggers InputAction.performed when a control crosses the button press threshold.

ReleaseOnly 

Perform the action when the button is released.

Triggers InputAction.started when a control crosses the button press threshold and InputAction.performed when the control goes back below the button press threshold.

PressAndRelease 

Perform the action when the button is pressed and when the button is released.

Triggers InputAction.performed when a control crosses the button press threshold and triggers InputAction.performed again when it goes back below the button press threshold.
