Returns the dot product, or scalar product, of the two input vectors {\bfseries{A}} and {\bfseries{B}}.

The dot product is a value equal to the magnitudes of the two vectors multiplied together and then multiplied by the cosine of the angle between them.

For normalized input vectors, the {\bfseries{Dot Product}} node returns 1 if they point in exactly the same direction, -\/1 if they point in completely opposite directions and 0 if the vectors are perpendicular.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_dot-_product-_node_autotoc_md3180}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_dot-_product-_node_autotoc_md3180}
