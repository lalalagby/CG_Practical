Generates a rectangle shape based on input {\bfseries{UV}} at the size specified by inputs {\bfseries{Width}} and {\bfseries{Height}}. The generated shape can be offset or tiled by connecting a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_tiling-_and-_offset-_node}{Tiling And Offset Node}{0}. Note that in order to preserve the ability to offset the shape within the UV space the shape will not automatically repeat if tiled. To achieve a repeating rectangle effect first connect your input through a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_fraction-_node}{Fraction Node}{0}.
