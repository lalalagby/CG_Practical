\doxysubsection{Detailed Description}
Inverts the {\ttfamily x} and/or {\ttfamily y} and/or {\ttfamily z} channel of a {\ttfamily Vector3}. 

This process is registered (see Input\+System.\+Register\+Processor$<$\+T$>$ as "{}invert\+Vector3"{} by default.
