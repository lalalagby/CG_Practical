 The {\bfseries{Slider}} control allows the user to select a numeric value from a predetermined range by dragging the mouse. Note that the similar \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_scrollbar}{Scroll\+Bar}{0} control is used for scrolling rather than selecting numeric values. Familiar examples include difficulty settings in games and brightness settings in image editors.

The value of a Slider is determined by the position of the handle along its length. The value increases from the {\itshape Min Value} up to the {\itshape Max Value} in proportion to the distance the handle is dragged. The default behaviour is for the slider to increase from left to right but it is also possible to reverse this behavior using the {\itshape Direction} property. You can also set the slider to increase vertically by selecting {\itshape Bottom To Top} or {\itshape Top To Bottom} for the {\itshape Direction} property.

The slider has a single event called {\itshape On Value Changed} that responds as the user drags the handle. The current numeric value of the slider is passed to the function as a {\ttfamily float} parameter. Typical use cases include\+:


\begin{DoxyItemize}
\item Choosing a level of difficulty in a game, brightness of a light, etc.
\item Setting a distance, size, time or angle. 
\end{DoxyItemize}
