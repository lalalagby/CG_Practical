\doxysubsection{Detailed Description}
Keep a list of \doxylink{class_unity_engine_1_1_input_system_1_1_input_control}{Input\+Control}s without allocating managed memory. 

This struct is mainly used by methods such as Input\+System.\+Find\+Controls(string) or Input\+Control\+Path.\+Try\+Find\+Controls$<$\+TControl$>$ to store an arbitrary length list of resulting matches without having to allocate GC heap memory.

Requires the control setup in the system to not change while the list is being used. If devices are removed from the system, the list will no longer be valid. Also, only works with controls of devices that have been added to the system (\doxylink{class_unity_engine_1_1_input_system_1_1_input_device_a4ff5e7586f91a321108317341566d4eb}{Input\+Device.\+added}). The reason for these constraints is that internally, the list only stores integer indices that are translates to \doxylink{class_unity_engine_1_1_input_system_1_1_input_control}{Input\+Control} references on the fly. If the device setup in the system changes, the indices may become invalid.

This struct allocates unmanaged memory and thus must be disposed or it will leak memory. By default allocates {\ttfamily Allocator.\+Persistent} memory. You can direct it to use another allocator by passing an Allocator value to one of the constructors.
