This control enables you to manually select your preferred preview mode for a node that has a preview.

When you select {\bfseries{Inherit}} in the Preview Mode Control, the Editor automatically selects the preview mode to use.

That decision is determined by either the type of the node you are previewing, the Sub Graph setting (if this node is in a Sub Graph) or other upstream nodes. To override the inheritance mode, select {\bfseries{Preview 2D}} or {\bfseries{Preview 3D}}.

This mode control functionality also applies to Sub Graph previews.
