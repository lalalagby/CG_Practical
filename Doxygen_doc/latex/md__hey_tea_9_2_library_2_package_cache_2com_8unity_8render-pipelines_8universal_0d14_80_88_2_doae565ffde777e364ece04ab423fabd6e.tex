Unityâ€™s Scriptable Render Pipeline (SRP) includes the Lens Flare (SRP) component which renders a lens flare in your scene. This is the SRP equivalent of the Built-\/in Render Pipeline\textquotesingle{}s \href{https://docs.unity3d.com/Manual/class-LensFlare.html}{\texttt{ Lens Flare}} component, which is incompatible with SRPs. You can attach a Lens Flare (SRP) component to any Game\+Object, but some properties only appear when you attach a Lens Flare (SRP) component to a light.

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_doae565ffde777e364ece04ab423fabd6e_autotoc_md2510}{}\doxysection{\texorpdfstring{Creating lens flares in SRP}{Creating lens flares in SRP}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_doae565ffde777e364ece04ab423fabd6e_autotoc_md2510}
The Lens Flare (SRP) component controls where the lens flare is as well as properties such as attenuation and whether the lens flare considers occlusion. For properties that define how the lens flare looks, SRP uses the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do2133c5043828aab4ed9305186b8574f7}{Lens Flare (SRP) Data}{0} asset. Each Lens Flare (SRP) component must reference a Lens Flare (SRP) data asset to display a lens flare on-\/screen.

To create a lens flare in a scene\+:


\begin{DoxyEnumerate}
\item Create or select a Game\+Object to attach the lens flare too.
\item In the Inspector, click {\bfseries{Add Component}}.
\item Select {\bfseries{Rendering}} \texorpdfstring{$>$}{>} {\bfseries{Lens Flare (SRP)}}. Currently, the lens flare doesn\textquotesingle{}t render in the scene because the component doesn\textquotesingle{}t reference a Lens Flare (SRP) Data asset in its {\bfseries{Lens Flare Data}} property.
\item Create a new Lens Flare (SRP) Data asset (menu\+: {\bfseries{Assets}} \texorpdfstring{$>$}{>} {\bfseries{Create}} \texorpdfstring{$>$}{>} {\bfseries{Lens Flare (SRP)}}).
\item In the Lens Flare (SRP) component Inspector, assign the new Lens Flare (SRP) Data asset to the {\bfseries{Lens Flare Data}} property.
\item Select the Lens Flare (SRP) Data asset and, in the Inspector, add a new element to the {\bfseries{Elements}} list. A default white lens flare now renders at the position of the Lens Flare (SRP) component. For information on how to customize how the lens flare looks, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do2133c5043828aab4ed9305186b8574f7}{Lens Flare (SRP) Data}{0}.
