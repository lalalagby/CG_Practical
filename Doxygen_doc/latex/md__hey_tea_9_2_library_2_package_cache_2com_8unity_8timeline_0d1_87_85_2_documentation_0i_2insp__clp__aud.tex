 Use the Inspector window to change the properties of an Audio clip. These properties include the name, timing, play speed, blend properties, audio media, and loop option.



{\itshape Inspector window when selecting an Audio clip in the \doxylink{namespace_timeline}{Timeline} window}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2insp__clp__aud_autotoc_md4668}{}\doxysection{\texorpdfstring{Display Name}{Display Name}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2insp__clp__aud_autotoc_md4668}
The name of the Audio clip shown in the \doxylink{namespace_timeline}{Timeline} window. This is not the name of the audio file that \doxylink{namespace_unity}{Unity} uses for the waveform. For information on audio file properties, see {\bfseries{Audio Playable Asset}} below.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2insp__clp__aud_autotoc_md4669}{}\doxysection{\texorpdfstring{Clip Timing properties}{Clip Timing properties}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2insp__clp__aud_autotoc_md4669}
Use the {\bfseries{Clip Timing}} properties to position, change the duration, change the ease-\/in and ease-\/out duration, and adjust the play speed of the Audio clip.

Most timing properties are expressed in both seconds (s) and frames (f). When specifying seconds, a {\bfseries{Clip Timing}} property accepts decimal values. When specifying frames, a property only accepts integer values. For example, if you attempt to enter 12.\+5 in a frames (f) field, the Inspector window sets the value to 12 frames.

Depending on \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2clp__about}{the selected Clip Edit mode}{0}, changing the {\bfseries{Start}}, {\bfseries{End}}, or {\bfseries{Duration}} may blend, ripple, or replace Audio clips on the same track.

Use the {\bfseries{Blend Curves}} to customize the fade-\/in and fade-\/out between the outgoing and incoming Audio clips. See \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2clp__blend}{Blending clips}{0} for details on how to blend clips and customize blend curves.

When easing-\/in or easing-\/out Audio clips, use the {\bfseries{Blend Curves}} to customize the curve that fades-\/in or fades-\/out an Audio clip. See \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2clp__ease}{Easing-\/in and Easing-\/out clips}{0} for details.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2insp__clp__aud_autotoc_md4671}{}\doxysection{\texorpdfstring{Audio Playable Asset}{Audio Playable Asset}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2insp__clp__aud_autotoc_md4671}
Use the {\bfseries{Audio Playable Asset}} properties to select the Audio file used by the Audio clip and to set whether the selected Audio clip loops ({\bfseries{Loop}} enabled) or plays once ({\bfseries{Loop}} disabled).
