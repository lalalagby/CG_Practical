You can use a Keyword node to create a static branch in your Shader Graph that references a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_keywords}{Keyword}{0} on the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_blackboard}{Blackboard}{0}. The appearance of a Keyword node, including its available ports, changes based on the Keyword it references.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_keyword-_node_autotoc_md3405}{}\doxysection{\texorpdfstring{Creating new Keyword Nodes}{Creating new Keyword Nodes}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_keyword-_node_autotoc_md3405}
Because each Keyword node references a specific Keyword, you must first define at least one Keyword on the Blackboard. Drag a Keyword from the Blackboard to the workspace to make a Keyword node that corresponds to that Keyword.

You can also right-\/click anywhere on the workspace, and use the {\bfseries{Create Node}} menu to make a new Keyword node. Under {\bfseries{Keywords}}, there is a list of Keywords that you defined on the Blackboard. Click on a Keyword in that list to create a corresponding Keyword node.
