 The \doxylink{namespace_timeline}{Timeline} window supports different methods of adding clips to tracks, depending on the type of track, where you click, and whether a clip or track is already selected.

The quickest method to add a clip is to right-\/click on an empty area within a track and select the appropriate Add option from the context menu. Depending on the track, the options for adding a clip change.



{\itshape Context menu for adding an Activation clip.}

There are other ways to add clips\+:


\begin{DoxyItemize}
\item Select a clip option from the Track menu in the Track Header to add a clip at the location of the \doxylink{namespace_timeline}{Timeline} Playhead.
\item Drag an animation Source Asset from the Project window to an empty area in the \doxylink{namespace_timeline}{Timeline} window to automatically create an Animation track and add an Animation clip.
\item Drag an animation Source Asset from the Project window to an existing track in the \doxylink{namespace_timeline}{Timeline} window to add an Animation clip to the same track.
\item Drag an audio Source Asset from the Project window to an empty area in the \doxylink{namespace_timeline}{Timeline} window to automatically create an Audio track and add an Audio clip.
\item Drag a Game\+Object with a Playable\+Director component to create a nested \doxylink{namespace_timeline}{Timeline} instance. This automatically creates a Control track and adds a Control clip for the nested \doxylink{namespace_timeline}{Timeline} instance.
\item Drag a Prefab from the Project window to an empty area in the \doxylink{namespace_timeline}{Timeline} window to add a Prefab instance to your \doxylink{namespace_timeline}{Timeline} instance. This automatically creates a Control track and adds a Control clip for the Prefab instance.
\item Drag a Game\+Object with a Particle component to add a particle effect to your \doxylink{namespace_timeline}{Timeline} instance. This automatically creates a Control track and adds a Control clip for the duration of the Particle effect.
\end{DoxyItemize}

When you add a clip, the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2clp__about}{selected Clip Edit mode}{0} determines how the added clip interacts with surrounding clips. For example, if you add an Animation clip or an Audio clip in Mix mode and the added clip intersects a clip on the same track, \doxylink{namespace_timeline}{Timeline} \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2clp__blend}{creates a blend}{0}.
