Use the {\bfseries{Clip Timing}} properties to position, change the duration, change the ease-\/in and ease-\/out duration, choose the extrapolation mode, and adjust the play speed of the Animation clip.

Most timing properties are expressed in both seconds (s) and frames (f). When specifying seconds, a {\bfseries{Clip Timing}} property accepts decimal values. When specifying frames, a property only accepts integer values. For example, if you attempt to enter 12.\+5 in a frames (f) field, the Inspector window sets the value to 12 frames.

Depending on the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2clp__about}{selected Clip Edit mode}{0}, changing the {\bfseries{Start}}, {\bfseries{End}}, or {\bfseries{Duration}} may blend, ripple, or replace Animation clips on the same track.

Use the {\bfseries{Animation Extrapolation}} properties to set the gap extrapolation before and after an Animation clip. The term {\bfseries{gap extrapolation}} refers to how an Animation track approximates or extends animation data in the gaps before, between, and after the Animation clips on a track.

There are two properties for \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2clp__gap__extrap}{setting the gap extrapolation}{0} between Animation clips. The {\bfseries{Pre-\/\+Extrapolate}} property only appears for Animation clips.
