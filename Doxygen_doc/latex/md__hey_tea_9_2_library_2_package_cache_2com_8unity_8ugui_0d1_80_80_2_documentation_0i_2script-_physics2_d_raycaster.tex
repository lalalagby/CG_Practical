 The 2D Raycaster raycasts against 2D objects in the scene. This allows messages to be sent to 2D physics objects that implement event interfaces. The Camera Game\+Object needs to be used and will be added to the Game\+Object if the Physics 3D Raycaster is not added to the Camera Game\+Object.
