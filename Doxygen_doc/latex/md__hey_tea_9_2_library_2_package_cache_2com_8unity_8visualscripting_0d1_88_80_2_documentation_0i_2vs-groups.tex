To create a new group\+:


\begin{DoxyEnumerate}
\item \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-open-graph-edit}{Open the graph}{0} where you want to create a new group.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item In an empty area of the Graph Editor, Ctrl+click (mac\+OS\+: Cmd+click) and drag to create a selection.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Release the mouse to create the group.
\end{DoxyEnumerate}

After you create a group, you can\+:


\begin{DoxyItemize}
\item Change the group name
\item Add a comment to the group
\item Change the color of the group
\item Resize the group
\item Move the group
\end{DoxyItemize}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-groups_autotoc_md5374}{}\doxysection{\texorpdfstring{Change a group name}{Change a group name}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-groups_autotoc_md5374}
To change the name of the group in your graph\+:


\begin{DoxyEnumerate}
\item In the group header, click {\bfseries{Group}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Enter a new name for the group.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Press {\bfseries{Enter}} to save your changes.
\end{DoxyEnumerate}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-groups_autotoc_md5375}{}\doxysection{\texorpdfstring{Add a comment to a group}{Add a comment to a group}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-groups_autotoc_md5375}
To add a comment to a group in a graph\+:


\begin{DoxyEnumerate}
\item Select the group where you want to add a comment.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Open the \href{vs-interface-overview.md\#the-graph-inspector}{\texttt{ Graph Inspector}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item In the \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}(Comment)\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} field in the Graph Inspector, enter comments or information about the group.
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
Group comments are only visible in the Graph Inspector. To add comments to a graph that are always visible, use a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-sticky-notes}{Sticky Note}{0}.
\end{DoxyNote}
\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-groups_autotoc_md5376}{}\doxysection{\texorpdfstring{Change the color of a group}{Change the color of a group}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-groups_autotoc_md5376}
To change the color of a group in a graph\+:


\begin{DoxyEnumerate}
\item Select the group you want to edit.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Open the \href{vs-interface-overview.md\#the-graph-inspector}{\texttt{ Graph Inspector}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item In the Graph Inspector, select the {\bfseries{Color}} field.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select a new color for your group through one of the following methods\+:
\begin{DoxyItemize}
\item Use the color picker.
\item Use the sliders or RGBA value fields.
\item Enter a hexadecimal color value.
\item Select the eyedropper icon and select a color from anywhere on your screen.
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-groups_autotoc_md5377}{}\doxysection{\texorpdfstring{Resize a group}{Resize a group}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-groups_autotoc_md5377}
To resize a group in a graph\+:


\begin{DoxyEnumerate}
\item Click and drag from any edge or corner on the group.
\end{DoxyEnumerate}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-groups_autotoc_md5378}{}\doxysection{\texorpdfstring{Move a group}{Move a group}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-groups_autotoc_md5378}
To move a group and its nodes\+:


\begin{DoxyEnumerate}
\item Click and drag the group\textquotesingle{}s header to a new location in the Graph Editor.
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
You can also move a group without moving any of the nodes inside, but the required input changes based on your chosen control scheme. \end{DoxyNote}
