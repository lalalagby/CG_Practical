Provides access to the current {\bfseries{Camera}}\textquotesingle{}s color buffer using input {\bfseries{UV}}, which is expected to be normalized screen coordinates.

The behavior of the Scene Color node isn\textquotesingle{}t defined globally. The executed HLSL code for the Scene Color node is defined per {\bfseries{Render Pipeline}}, and different {\bfseries{Render Pipelines}} can produce different results. Custom {\bfseries{Render Pipelines}} that wish to support the Scene Color node need to explicitly define the behavior for it. If the behavior is undefined, the Scene Color node returns 0 (black).

In the {\bfseries{Universal Render Pipeline}} the Scene Color node returns the value of the {\bfseries{Camera Opaque Texture}}. Refer to the {\bfseries{Universal Render Pipeline}} for more documentation on this feature. The contents of this texture are only available for {\bfseries{Transparent}} objects. Set the {\bfseries{Surface Type}} dropdown on the {\bfseries{Material Options}} panel of the Master Node to {\bfseries{Transparent}} to receive the correct values from this node.
