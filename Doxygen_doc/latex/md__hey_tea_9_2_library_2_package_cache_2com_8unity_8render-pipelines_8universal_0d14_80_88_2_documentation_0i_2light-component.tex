{\bfseries{Color}}   &Use the color picker to set the color of the emitted light.    \\\cline{1-2}
{\bfseries{Intensity}}   &Set the brightness of the light. The default value for a {\bfseries{Directional}} light is 0.\+5. The default value for a {\bfseries{Point}}, {\bfseries{Spot}} or {\bfseries{Area}} light is 1.    \\\cline{1-2}
{\bfseries{Indirect Multiplier}}   &Use this value to vary the intensity of indirect light. Indirect light is light that has bounced from one object to another. The {\bfseries{Indirect Multiplier}} defines the brightness of bounced light calculated by the global illumination (GI) system. If you set {\bfseries{Indirect Multiplier}} to a value lower than {\bfseries{1,}} the bounced light becomes dimmer with every bounce. A value higher than {\bfseries{1}} makes light brighter with each bounce. This is useful, for example, when a dark surface in shadow (such as the interior of a cave) needs to be brighter in order to make detail visible.    \\\cline{1-2}
{\bfseries{Range}}   &Define how far the light emitted from the center of the object travels ({\bfseries{Point}} and {\bfseries{Spot}} lights only).    \\\cline{1-2}
{\bfseries{Cookie}}   &The RGB texture this Light projects into the scene. Use cookies to create silhouettes or patterned illumination. The texture format to use depends on the type of Light\+:~\newline
 \&\#8226; Directional\+: 2D texture~\newline
 \&\#8226; Spot\+: 2D texture~\newline
 \&\#8226; Point\+: \href{https://docs.unity3d.com/Manual/class-Cubemap.html}{\texttt{ cubemap texture}}~\newline
~\newline
{\bfseries{Note}}\+: URP doesn\textquotesingle{}t support light cookies for Area lights.~\newline
~\newline
For more information about light cookies, see \href{https://docs.unity3d.com/Manual/Cookies.html}{\texttt{ Cookies}}.    \\\cline{1-2}
~~{\bfseries{Cookie Size}}   &The per-\/axis scale \doxylink{namespace_unity}{Unity} applies to the cookie texture. Use this property to set the size of the cookie.~\newline
~\newline
This property is available only if you set {\bfseries{Type}} to {\bfseries{Directional}} and assign a texture to {\bfseries{Cookie}}.    \\\cline{1-2}
~~{\bfseries{Cookie Offset}}   &The per-\/axis offset \doxylink{namespace_unity}{Unity} applies to the cookie texture. Use this property to move the cookie without moving the light itself. You can also animate this property to scroll the cookie. ~\newline
~\newline
This property is available only if you set {\bfseries{Type}} to {\bfseries{Directional}} and assign a texture to {\bfseries{Cookie}}.   \\\cline{1-2}

~~~~Important   &The light is always rendered at per-\/pixel quality. Use {\bfseries{Important}} mode only for the most noticeable visual effects (for example, the headlights of a player’s car).    \\\cline{1-2}
~~~~Not Important   &The light is always rendered in a faster, vertex/object light mode.    \\\cline{1-2}

{\bfseries{Shadow Type}}   &Determine whether this Light casts Hard Shadows, Soft Shadows, or no shadows at all. See the page \href{https://docs.unity3d.com/Manual/class-Light.html}{\texttt{ Lights}} for information on hard and soft shadows.    \\\cline{1-2}
~~~~Baked Shadow Angle   &If {\bfseries{Type}} is set to {\bfseries{Directional}} and {\bfseries{Shadow Type}} is set to {\bfseries{Soft Shadows}}, this property adds some artificial softening to the edges of shadows and gives them a more natural look.    \\\cline{1-2}
~~~~Baked Shadow Radius   &If {\bfseries{Type}} is set to {\bfseries{Point}} or {\bfseries{Spot}} and {\bfseries{Shadow Type}} is set to {\bfseries{Soft Shadows}}, this property adds some artificial softening to the edges of shadows and gives them a more natural look.    \\\cline{1-2}
~~~~Realtime Shadows   &These properties are available when {\bfseries{Shadow Type}} is set to {\bfseries{Hard Shadows}} or {\bfseries{Soft Shadows}}. Use these properties to control real-\/time shadow rendering settings.    \\\cline{1-2}
~~~~~~~~Strength   &Use the slider to control how dark the shadows cast by this Light are, represented by a value between 0 and 1. This is set to 1 by default.    \\\cline{1-2}
~~~~~~~~Bias   &Controls whether to use shadow bias settings from the URP Asset, or whether to define custom shadow bias settings for this Light. Possible values are {\bfseries{Use Pipeline Settings}} or {\bfseries{Custom}}.    \\\cline{1-2}
~~~~~~~~Depth   &Controls the distance at which the shadows will be pushed away from the light. Useful for avoiding false self-\/shadowing artifacts. This property is visible only when {\bfseries{Bias}} is set to {\bfseries{Custom}}.    \\\cline{1-2}
~~~~~~~~Normal   &Controls the distance at which the shadow casting surfaces will be shrunk along the surface normal. Useful for avoiding false self-\/shadowing artifacts. This property is visible only when {\bfseries{Bias}} is set to {\bfseries{Custom}}.    \\\cline{1-2}
~~~~~~~~Near Plane   &Use the slider to control the value for the near clip plane when rendering shadows, defined as a value between 0.\+1 and 10. This value is clamped to 0.\+1 units or 1\% of the light’s {\bfseries{Range}} property, whichever is lower. This is set to 0.\+2 by default.    \\\cline{1-2}
~~~~~~~~Soft~Shadows~Quality   &Select the soft shadows quality. With the {\bfseries{Use Pipeline Settings}} option selected \doxylink{namespace_unity}{Unity} uses the value from the URP Asset. Options {\bfseries{Low}}, {\bfseries{Medium}}, and {\bfseries{High}} let you specify the soft shadow quality value for this Light. For more information on the values, see the \href{universalrp-asset.md\#soft-shadows}{\texttt{ Soft Shadows}} section.   \\\cline{1-2}
