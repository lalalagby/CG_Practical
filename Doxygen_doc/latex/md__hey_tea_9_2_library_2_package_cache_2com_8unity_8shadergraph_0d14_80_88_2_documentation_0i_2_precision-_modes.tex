Shader Graph provides specific \href{https://en.wikipedia.org/wiki/Precision_(computer_science)}{\texttt{ data precision modes}} for nodes, graphs, and Sub Graphs to help you optimize your content for different platforms.

To set the precision of an entire graph, select the \mbox{[}{\bfseries{Graph Settings}}\mbox{]}(Graph-\/\+Settings-\/\+Menu) tab in the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_internal-_inspector}{Graph Inspector}{0} and adjust the {\bfseries{Precision}} control. Select a node in your graph and select the {\bfseries{Node Settings}} tab in the Graph Inspector to adjust the precision of individual nodes.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3618}{}\doxysection{\texorpdfstring{Precision mode settings}{Precision mode settings}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3618}

To set the default precision for the entire graph to {\bfseries{Single}} or {\bfseries{Half}}, open the {\bfseries{Graph Settings}} and set the Precision property. Newly-\/created nodes in a graph default to the {\bfseries{Inherit}} precision mode, and inherit the graph\textquotesingle{}s precision.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3622}{}\doxysubsection{\texorpdfstring{Setting node Precision}{Setting node Precision}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3622}
Select a node to access its precision setting. The precision you set for a node determines the precision of the data types which that node uses for its calculations.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3623}{}\doxysubsection{\texorpdfstring{Precision Inheritance}{Precision Inheritance}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3623}
All nodes use the {\bfseries{Inherit}} precision mode by default. In this mode, a node that has one or more edge connections takes on the precision mode of an incoming \mbox{[}edge\mbox{]}(Edge). Nodes that do not have any edge connections take on {\bfseries{Graph Precision}}. If you change the {\bfseries{Graph Precision}} mode, the precision of those nodes also changes.

Simple inheritance refers to the inheritance behaviour of a node with only one precision type on its inputs.

In the figure below, Node A has the {\bfseries{Inherit}} mode. Because it has no incoming edge, it takes the {\bfseries{Graph Precision}}, which is {\bfseries{Half}}. Node B also has the {\bfseries{Inherit}} mode, so it inherits the {\bfseries{Half}} precision mode from Node A.

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3625}{}\doxysubsubsection{\texorpdfstring{Complex inheritance}{Complex inheritance}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3625}
Complex inheritance refers to the inheritance behaviour of a node with multiple precision types on its inputs.

A node reads precision settings from each input port. If you connect a node to several others with a variety of precision modes, the node with the highest resolution determines the precision mode for the group.

In the figure below, node D has the {\bfseries{Inherit}} mode. It receives input from the adjacent edges via inputs 1 and 2. Node B passes the {\bfseries{Half}} mode through input 1. Node C passes the {\bfseries{Single}} mode through input 2. Because {\bfseries{Single}} is 32-\/bit and {\bfseries{Half}} only 16-\/bit, {\bfseries{Single}} takes precedence, so Node D uses {\bfseries{Single}} precision.

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3626}{}\doxysubsubsection{\texorpdfstring{Mixed inheritance}{Mixed inheritance}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3626}
Mixed inheritance refers to the inheritance behaviour on a node with both simple and complex inheritance types.

Nodes with no input ports, such as \mbox{[}Input nodes\mbox{]}(Input-\/\+Nodes), inherit the {\bfseries{Graph Precision}}. However, complex inheritance rules still affect other nodes in the same group, as illustrated in the figure below.

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3628}{}\doxysubsection{\texorpdfstring{Sub Graph precision}{Sub Graph precision}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3628}
Precision behavior and user interface elements for \mbox{[}Sub Graphs\mbox{]}(Sub-\/graph) and their nodes do not differ from other graphs and nodes. Sub Graphs represent a function, and you can affect that function\textquotesingle{}s inputs, outputs, and operators by modifying the relevant set of precision settings.


\begin{DoxyItemize}
\item The Sub Graph properties correspond to the function\textquotesingle{}s inputs.
\item The internal node properties correspond to the function\textquotesingle{}s operators.
\item The output node corresponds to the function\textquotesingle{}s outputs.
\end{DoxyItemize}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3629}{}\doxysubsubsection{\texorpdfstring{Outputs}{Outputs}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3629}
To manually determine the precision of a Sub Graph\textquotesingle{}s output, modify the {\bfseries{Output}} nodeâ€™s {\bfseries{Precision Mode}} setting.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3630}{}\doxysubsubsection{\texorpdfstring{Inputs}{Inputs}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3630}
To manually determine the precision of {\bfseries{Sub Graph Inputs}}, open the \mbox{[}Graph Inspector\mbox{]}(Internal-\/\+Inspector) and set precision modes for each individual \mbox{[}Property\mbox{]}(Property-\/\+Types). Properties that use the Inherit option take on the {\bfseries{Graph Precision}} you set for the Sub Graph.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3631}{}\doxysubsubsection{\texorpdfstring{Sub Graph Precision within other graphs}{Sub Graph Precision within other graphs}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes_autotoc_md3631}
By default, a Sub Graph has a Precision Mode of {\ttfamily Switchable}. You can modify Precision Mode of any \mbox{[}Sub Graph node\mbox{]}(Sub-\/graph-\/\+Node) for that Sub Graph, as long as you set the Precision Mode on the Sub Graph as {\ttfamily Switchable}.

Shader Graph won\textquotesingle{}t allow you to change the Precision Mode for any Sub Graph node that doesn\textquotesingle{}t have its Sub Graph set to {\ttfamily Switchable}. This is because the input and output precision you set in a Sub Graph define the precision of its associated Sub Graph Node.

For example, let\textquotesingle{}s say that Sub Graph A is {\bfseries{Switchable}}. You open Graph 1, which includes a Sub Graph Node referencing Sub Graph A. Like all other nodes, Sub Graph Node A defaults to {\bfseries{Inherit}}. You change the precision of Sub Graph Node A to {\bfseries{Half}}. The precision of Sub Graph A also becomes {\bfseries{Half}}.
