 \doxylink{namespace_burst}{Burst} is a compiler that you can use with \doxylink{namespace_unity}{Unity}\textquotesingle{}s \href{https://docs.unity3d.com/Manual/JobSystem.html}{\texttt{ job system}} to create code that enhances and improves your application\textquotesingle{}s performance. It translates your code from IL/.NET bytecode to optimized native CPU code that uses the \href{https://llvm.org/}{\texttt{ LLVM compiler}}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2index_autotoc_md407}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2index_autotoc_md407}
To install this package, follow the instructions in the \href{https://docs.unity3d.com/Manual/upm-ui-actions.html}{\texttt{ Package Manager documentation}}.

If you change the \doxylink{namespace_burst}{Burst} package version (for example, via Update), you need to close and restart the Editor.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2index_autotoc_md408}{}\doxysection{\texorpdfstring{Further resources}{Further resources}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2index_autotoc_md408}
