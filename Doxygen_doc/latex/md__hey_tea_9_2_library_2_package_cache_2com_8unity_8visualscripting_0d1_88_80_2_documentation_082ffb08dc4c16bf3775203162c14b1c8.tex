In the following example, the On Keyboard Input node listens for when the user presses the {\bfseries{Space}} key. When the user presses {\bfseries{Space}}, the On Keyboard Input triggers the Transform Translate node and lifts the Game\+Object along its {\ttfamily Y} coordinate by {\ttfamily 5} units. This makes the Game\+Object jump.
