 After you configure an SRP, you can create a new Shader Graph Asset. Right-\/click the Project window, locate {\bfseries{Create}} \texorpdfstring{$>$}{>} {\bfseries{Shader Graph}} in the context menu, then select your desired type of Shader Graph.

The type of Shader Graph available is dependent on the render pipelines present in your project. Some options may or may not be present based on the render pipelines.

The following options are always available\+:

No target is selected and no blocks are added to the Master Stack. \texorpdfstring{$\vert$}{|} \texorpdfstring{$\vert$}{|} Sub Graph \texorpdfstring{$\vert$}{|} A blank sub graph asset. \texorpdfstring{$\vert$}{|}

A sub menu for each installed render pipeline may be present containing template stacks for standard shading models ( Lit, Unlit, etc ).

For a full list of provided options, refer to the \href{https://docs.unity3d.com/Packages/com.unity.render-pipelines.universal@latest}{\texttt{ Universal Render Pipeline}} and \href{https://docs.unity3d.com/Packages/com.unity.render-pipelines.high-definition@latest}{\texttt{ High Definition Render Pipeline}} documentation.

For this example, Universal is installed so a Unversal Lit Shader Graph has been created.

Double-\/click your newly created Shader Graph Asset to open it in the Shader Graph window.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_create-_shader-_graph_autotoc_md3094}{}\doxysection{\texorpdfstring{Shader Graph window}{Shader Graph window}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_create-_shader-_graph_autotoc_md3094}
The Shader Graph window consists of the Master Stack, the Preview Window, the Blackboard, and the Graph Inspector.
