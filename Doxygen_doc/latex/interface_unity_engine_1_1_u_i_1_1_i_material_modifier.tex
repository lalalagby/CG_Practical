\doxysubsection{Detailed Description}
Use this interface to modify a Material that renders a Graphic. The Material is modified before the it is passed to the Canvas\+Renderer. 

When a Graphic sets a material that is passed (in order) to any components on the Game\+Object that implement \doxylink{interface_unity_engine_1_1_u_i_1_1_i_material_modifier}{IMaterial\+Modifier}. This component can modify the material to be used for rendering.
