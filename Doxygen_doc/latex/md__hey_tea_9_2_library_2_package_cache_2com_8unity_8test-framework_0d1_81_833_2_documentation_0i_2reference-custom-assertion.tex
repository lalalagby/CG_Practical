 A test fails if \doxylink{namespace_unity}{Unity} logs a message other than a regular log or warning message. Use Log\+Assert to check for an expected message in the log so that the test does not fail when \doxylink{namespace_unity}{Unity} logs the message.

Use {\ttfamily Log\+Assert.\+Expect} before running the code under test, as the check for expected logs runs at the end of each frame.

A test also reports a failure, if an expected message does not appear, or if \doxylink{namespace_unity}{Unity} does not log any regular log or warning messages.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2reference-custom-assertion_autotoc_md4303}{}\doxysection{\texorpdfstring{Example}{Example}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2reference-custom-assertion_autotoc_md4303}
