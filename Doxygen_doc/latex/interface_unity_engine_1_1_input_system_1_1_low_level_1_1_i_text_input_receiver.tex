This interface should be implemented by devices that are meant to receive text input through \doxylink{struct_unity_engine_1_1_input_system_1_1_low_level_1_1_text_event}{Text\+Event}.

This method is called on a device when a \doxylink{struct_unity_engine_1_1_input_system_1_1_low_level_1_1_text_event}{Text\+Event} is received for the device. {\itshape character}  is the \doxylink{struct_unity_engine_1_1_input_system_1_1_low_level_1_1_text_event_a345ffba87df93f61c16a85c244669063}{Text\+Event.\+character} from the event.

Note that this method will be called {\itshape twice} for a single \doxylink{struct_unity_engine_1_1_input_system_1_1_low_level_1_1_text_event}{Text\+Event} in case the given UTF-\/32 (encoding in the event) needs to be represented as UTF-\/16 (encoding of {\ttfamily char} in C\#) surrogate.
