 The {\bfseries{Render Pipeline Converter}} converts assets made for a Built-\/in Render Pipeline project to assets compatible with URP.

\begin{quote}
{\bfseries{NOTE\+:}} The conversion process makes irreversible changes to the project. Back up your project before the conversion. \end{quote}
\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do54fb4e3619bda36fb863b138677b0598_autotoc_md2155}{}\doxysection{\texorpdfstring{How to use the Render Pipeline Converter}{How to use the Render Pipeline Converter}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do54fb4e3619bda36fb863b138677b0598_autotoc_md2155}
To convert project assets\+:


\begin{DoxyEnumerate}
\item Select {\bfseries{Window}} \texorpdfstring{$>$}{>} {\bfseries{Rendering}} \texorpdfstring{$>$}{>} {\bfseries{Render Pipeline Converter}}. \doxylink{namespace_unity}{Unity} opens the Render Pipeline Converter window.


\item Select the conversion type.


\item Depending on the conversion type, the dialog shows the available converters. Select or clear the check boxes next to converter names to enable or disable the converters.



For the list of available converters, see the section Converters.
\item Click {\bfseries{Initialize Converters}}. The Render Pipeline Converter preprocesses the assets in the project and shows the list of elements to convert. Select or clear check boxes next to assets to include or exclude them from the conversion process.



The following illustration shows initialized converters.



Click a converter to see the list of items that a converter is about to convert.



{\bfseries{Yellow icon}}\+: a yellow icon next to an element indicates that a user action might be required to run the conversion. Hover the mouse pointer over the icon to see the description of the issue.
\item Click {\bfseries{Convert Assets}} to start the conversion process.
\end{DoxyEnumerate}

\begin{quote}
{\bfseries{NOTE\+:}} The conversion process makes irreversible changes to the project. Back up your project before the conversion. \end{quote}


When the conversion process finishes, the window shows the status of each converter.



{\bfseries{Green check mark}}\+: the conversion went without issues.

{\bfseries{Yellow icon}}\+: the conversion finished with warnings and might require user action.

{\bfseries{Red icon}}\+: the conversion failed.


\begin{DoxyEnumerate}
\item Click a converter to see the list of processed items in that converter.



After reviewing the converted project, close the Render Pipeline Converter window.

The Render Pipeline Converter let\textquotesingle{}s you select one of the following conversion types\+:


\begin{DoxyItemize}
\item Built-\/in Render Pipeline to URP
\item Built-\/in Render Pipeline 2D to URP 2D
\item Upgrade 2D URP Assets
\end{DoxyItemize}

When you select on of the conversion types, the tool shows you the available converters.

The following sections describe the converters available for each conversion type.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do54fb4e3619bda36fb863b138677b0598_autotoc_md2157}{}\doxysubsection{\texorpdfstring{Built-\/in Render Pipeline to URP}{Built-\/in Render Pipeline to URP}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do54fb4e3619bda36fb863b138677b0598_autotoc_md2157}
This conversion type converts project elements from the Built-\/in Render Pipeline to URP.

Available converters\+:


\begin{DoxyItemize}
\item {\bfseries{Rendering Settings}}

This converter creates the URP Asset and Renderer assets. Then the converter evaluates the settings in the Built-\/in Render Pipeline project and converts them into equivalent properties in the URP assets.
\item {\bfseries{Material Upgrade}}

This converter converts the Materials. The converter works on pre-\/built Materials that are supplied by \doxylink{namespace_unity}{Unity}, it does not support Materials with custom shaders.
\item {\bfseries{Animation Clip Converter}}

This converter converts the animation clips. It runs after the {\bfseries{Material Upgrade}} converter finishes.
\end{DoxyItemize}

\begin{quote}
{\bfseries{NOTE\+:}} This converter is available only if the project contains animations that affect the properties of Materials, or Post-\/processing Stack v2 properties. \end{quote}



\begin{DoxyItemize}
\item {\bfseries{Read-\/only Material Converter}}

This converter converts the pre-\/built read-\/only Materials, where the {\bfseries{Material Upgrade}} converter cannot replace the shader. This converter indexes the project and creates a temporary {\ttfamily .index} file, which might take a significant time.

Examples of read-\/only Materials\+: {\ttfamily Default-\/\+Diffuse}, {\ttfamily Default-\/\+Line}, {\ttfamily Dafault-\/\+Terrain-\/\+Diffuse}, etc.
\item {\bfseries{Post-\/\+Processing Stack v2 Converter}}

This converter converts PPv2 Volumes, Profiles, and Layers to URP Volumes, Profiles, and Cameras. This converter indexes the project and creates a temporary {\ttfamily .index} file, which might take a significant time.
\end{DoxyItemize}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do54fb4e3619bda36fb863b138677b0598_autotoc_md2158}{}\doxysubsection{\texorpdfstring{Built-\/in Render Pipeline 2D to URP 2D}{Built-\/in Render Pipeline 2D to URP 2D}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do54fb4e3619bda36fb863b138677b0598_autotoc_md2158}
This conversion type converts elements of a project from Built-\/in Render Pipeline 2D to URP 2D.

Available converters\+:


\begin{DoxyItemize}
\item {\bfseries{Material and Material Reference Upgrade}}

This converter converts all Materials and Material references from Built-\/in Render Pipeline 2D to URP 2D.
\end{DoxyItemize}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do54fb4e3619bda36fb863b138677b0598_autotoc_md2159}{}\doxysubsection{\texorpdfstring{Upgrade 2D URP Assets}{Upgrade 2D URP Assets}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do54fb4e3619bda36fb863b138677b0598_autotoc_md2159}
This conversion type upgrades assets of a 2D project from an earlier URP version to the current URP version.

Available converters\+:


\begin{DoxyItemize}
\item {\bfseries{Parametric to Freeform Light Upgrade}}

This converter converts all parametric lights to freeform lights.
