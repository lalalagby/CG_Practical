Returns the logarithm of input {\bfseries{In}}. {\bfseries{Log}} is the inverse operation to the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_exponential-_node}{Exponential Node}{0}.

For example, the result of a base-\/2 {\bfseries{Exponential}} using an input value of 3 is 8.



Therefore the result of a base-\/2 {\bfseries{Log}} using an input value of 8 is 3.

The logarithmic base can be switched between base-\/e, base-\/2 and base-\/10 from the {\bfseries{Base}} dropdown on the node.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_log-_node_autotoc_md3427}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_log-_node_autotoc_md3427}
