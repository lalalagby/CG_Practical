\doxysubsection{Detailed Description}
Contextual data made available when processing values of composite bindings. 

An instance of this struct is passed to \doxylink{class_unity_engine_1_1_input_system_1_1_input_binding_composite_a2c23901dcc819e76109feadf9171fad0}{Input\+Binding\+Composite$<$\+TValue$>$.\+Read\+Value(ref Input\+Binding\+Composite\+Context)}. Use it to access contextual data such as the value for individual part bindings.

Note that an instance of this struct should never be held on to past the duration of the call to {\ttfamily Read\+Value}. The data it retrieves is only valid during the callback.

If no control is bound to the given part, the return value will always be {\ttfamily default(\+TValue)}. If a single control is bound to the part, the value will be that of the control. If multiple controls are bound to a part, the return value will be that greatest one according to {\ttfamily IComparable} implemented by {\itshape TValue} .

Note that this method only works with values that are {\ttfamily IComparable}. To read a value type that is not {\ttfamily IComparable} or to supply a custom comparer, use \doxylink{struct_unity_engine_1_1_input_system_1_1_input_binding_composite_context_ae2acf37a03df15776e87289318ee302c}{Read\+Value$<$\+TValue,\+TComparer$>$(int,\+TComparer)}.

If an invalid {\itshape part\+Number}  is supplied, the return value will simply be {\ttfamily default(\+TValue)}. No exception is thrown.
