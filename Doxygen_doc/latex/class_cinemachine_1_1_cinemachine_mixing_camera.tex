\doxysubsection{Detailed Description}
\doxylink{class_cinemachine_1_1_cinemachine_mixing_camera}{Cinemachine\+Mixing\+Camera} is a "{}manager camera"{} that takes on the state of the weighted average of the states of its child virtual cameras. 

A fixed number of slots are made available for cameras, rather than a dynamic array. We do it this way in order to support weight animation from the \doxylink{namespace_timeline}{Timeline}. \doxylink{namespace_timeline}{Timeline} cannot animate array elements.
