 Tonemapping is the process of remapping the HDR values of an image to a new range of values. Its most common purpose is to make an image with a low dynamic range appear to have a higher range. See Wikipedia\+: Tone mapping.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do8b33578e617f892d2d6e5a8ea0aeda08_autotoc_md2337}{}\doxysection{\texorpdfstring{Using Tonemapping}{Using Tonemapping}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do8b33578e617f892d2d6e5a8ea0aeda08_autotoc_md2337}
{\bfseries{Tonemapping}} uses the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2_volumes}{Volume}{0} system, so to enable and modify {\bfseries{Tonemapping}} properties, you must add a {\bfseries{Tonemapping}} override to a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2_volumes}{Volume}{0} in your Scene.

To add {\bfseries{Tonemapping}} to a Volume\+:


\begin{DoxyEnumerate}
\item In the Scene or Hierarchy view, select a Game\+Object that contains a Volume component to view it in the Inspector.
\item In the Inspector, navigate to {\bfseries{Add Override}} \texorpdfstring{$>$}{>} {\bfseries{Post-\/processing}}, and click on {\bfseries{Tonemapping}}. The Universal Render Pipeline applies {\bfseries{Tonemapping}} to any Camera this Volume affects.

{\bfseries{Mode}}   &Select a tonemapping algorithm to use for color grading. The options are\+:
\begin{DoxyItemize}
\item {\bfseries{None}}\+: Use this option if you do not want to apply tonemapping.
\item {\bfseries{Neutral}}\+: Use this option if you only want range-\/remapping with minimal impact on color hue \& saturation. It is generally a good starting point for extensive color grading.
\item {\bfseries{ACES}}\+: Use this option to apply a close approximation of the reference ACES tonemapper, for a more cinematic look. It is more contrasted than Neutral, and has an effect on actual color hue \& saturation. If you use this tonemapper, \doxylink{namespace_unity}{Unity} does all the grading operations in the ACES color spaces, for optimal precision and results.~\newline
{\bfseries{Note}}\+: ACES HDR tonemapping is not supported on Android devices with Adreno 300 series GPU.
\end{DoxyItemize}\\\cline{1-2}
