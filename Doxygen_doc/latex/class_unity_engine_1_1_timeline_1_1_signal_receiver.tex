\doxysubsection{Detailed Description}
Listens for emitted signals and reacts depending on its defined reactions. 

A \doxylink{class_unity_engine_1_1_timeline_1_1_signal_receiver}{Signal\+Receiver} contains a list of reactions. Each reaction is bound to a \doxylink{class_unity_engine_1_1_timeline_1_1_signal_asset}{Signal\+Asset}. When a \doxylink{class_unity_engine_1_1_timeline_1_1_signal_emitter}{Signal\+Emitter} emits a signal, the \doxylink{class_unity_engine_1_1_timeline_1_1_signal_receiver}{Signal\+Receiver} invokes the corresponding reaction. \begin{DoxySeeAlso}{See also}
