\doxysubsection{Detailed Description}
A key on a \doxylink{class_unity_engine_1_1_input_system_1_1_keyboard}{Keyboard}. 

This is an extended button control which adds various features to account for the fact that keys have symbols associated with them which may change depending on keyboard layout as well as in combination with other keys.

Note that there is no text input associated with individual keys as text composition is highly layout specific and does not need to be key-\/by-\/key. For general text input, see \doxylink{class_unity_engine_1_1_input_system_1_1_keyboard_a9ce9e5d035216150d0869a90fd1be46f}{Keyboard.\+on\+Text\+Input}. To find the text displayed on a key, use \doxylink{class_unity_engine_1_1_input_system_1_1_input_control_ac31c2a2f570e527aacc86ee8060e2ecc}{Input\+Control.\+display\+Name}.
