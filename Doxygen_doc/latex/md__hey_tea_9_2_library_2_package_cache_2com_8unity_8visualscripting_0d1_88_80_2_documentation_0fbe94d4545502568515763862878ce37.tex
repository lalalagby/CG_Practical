 \begin{DoxyImportant}{Important}
If you don\textquotesingle{}t complete the prerequisite configuration for your project, you can\textquotesingle{}t use the Input \doxylink{namespace_system}{System} package with Visual Scripting. For more information, see \href{vs-capture-player-input.md\#input-system-package-prerequisites}{\texttt{ Input System package prerequisites}}.
\end{DoxyImportant}
To use the Input \doxylink{namespace_system}{System} package with Visual Scripting to capture input in your project\+:


\begin{DoxyEnumerate}

\item \mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0e129af79c68762283d74b648822ac987}{open-\/fuzzy-\/finder}{0}\mbox{]}
\item Go to {\bfseries{Events}} \texorpdfstring{$>$}{>} {\bfseries{Input}} or search for {\bfseries{On Input \doxylink{namespace_system}{System} Event}}.
\item Select an Input \doxylink{namespace_system}{System} Event node. In this example, select the {\bfseries{On Input \doxylink{namespace_system}{System} Event Vector 2}} node to add it to the graph.
\item Set the {\bfseries{On Input \doxylink{namespace_system}{System} Event Vector 2}} node\textquotesingle{}s {\bfseries{Input Action Change Type}} control to {\bfseries{On Hold}}\+:


\item In the {\bfseries{Input Action}} list on the {\bfseries{On Input \doxylink{namespace_system}{System} Event Vector 2}} node, select an Input Action to trigger the node. In this example, select {\bfseries{Move}}.
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
By default, Visual Scripting displays all Input Actions from the Input Action asset attached to your current Game\+Object\textquotesingle{}s \doxylink{class_player}{Player} Input component.
\end{DoxyNote}
