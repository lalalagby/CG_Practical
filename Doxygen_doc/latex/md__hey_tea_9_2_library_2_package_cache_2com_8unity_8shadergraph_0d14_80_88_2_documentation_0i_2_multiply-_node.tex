Returns the result of input {\bfseries{A}} multiplied by input {\bfseries{B}}. If both inputs are a vector type, the output type will be a vector type with the same dimension as the evaluated type of those inputs. If both inputs are a matrix type, the output type will be a matrix type with the same dimension as the evaluated type of those inputs.
