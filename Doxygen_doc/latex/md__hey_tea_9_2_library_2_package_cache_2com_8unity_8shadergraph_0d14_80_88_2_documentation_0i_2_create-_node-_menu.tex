Use the {\bfseries{Create Node Menu}} to create \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node}{nodes}{0} in Shader Graph. To open the {\bfseries{Create Node Menu}}, either right-\/click on the workspace in the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_graph-_window}{Shader Graph Window}{0} and select {\bfseries{Create Node}}, or press the spacebar.

At the top of the {\bfseries{Create Node Menu}} is a search bar. To search for a node, type any part of its name in the search field. The search box gives you autocomplete options, and you can press Tab to accept the predictive text. It highlights matching text in yellow.

The {\bfseries{Create Node Menu}} lists all nodes that are available in Shader Graph, categorized by their function. User-\/created \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sub-graph}{Sub Graphs}{0} are also available in the {\bfseries{Create Node Menu}} under {\bfseries{Sub Graph Assets}}, or in a custom category that you define in the Sub Graph Asset.

To add a node to the workspace, double-\/click it in the {\bfseries{Create Node Menu}}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_create-_node-_menu_autotoc_md3091}{}\doxysubsection{\texorpdfstring{Contextual Create Node Menu}{Contextual Create Node Menu}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_create-_node-_menu_autotoc_md3091}
A contextual {\bfseries{Create Node Menu}} filters the available nodes, and only shows those that use the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_data-_types}{Data Type}{0} of a selected edge. It lists every available \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_port}{Port}{0} on nodes that match that Data Type.

To open a contextual {\bfseries{Create Node Menu}}, click and drag an \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_edge}{Edge}{0} from a Port, and then release it in an empty area of the workspace.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_create-_node-_menu_autotoc_md3092}{}\doxysubsection{\texorpdfstring{Master Stack Create Node Menu}{Master Stack Create Node Menu}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_create-_node-_menu_autotoc_md3092}
To add a new \mbox{[}Block Node\mbox{]}() to the \mbox{[}Master Stack\mbox{]}(), either right click and select {\bfseries{Create Node}} or press spacebar with the stack selected.

The {\bfseries{Create Node Menu}} will display all available blocks for the master stack based on the render pipelines in your project. Any block can be added to the master stack via the {\bfseries{Create Node Menu}}. If the block added is not compatible with the current Graph settings, the block will be disabled until the settings are configured to support it.
