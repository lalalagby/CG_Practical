The Position Node provides drop-\/down options for both {\bfseries{World}} and {\bfseries{Absolute World}} space positions. The {\bfseries{Absolute World}} option always returns the absolute world position of the object in the Scene for all Scriptable Render Pipelines. The {\bfseries{World}} option returns the default world space of the selected Scriptable Render Pipeline.

The \href{https://docs.unity3d.com/Packages/com.unity.render-pipelines.high-definition@latest/index.html}{\texttt{ High Definition Render Pipeline}} uses \href{https://docs.unity3d.com/Packages/com.unity.render-pipelines.high-definition@latest?preview=1&subfolder=/manual/Camera-Relative-Rendering.html}{\texttt{ Camera Relative}} as its default world space.

The \href{https://docs.unity3d.com/Packages/com.unity.render-pipelines.universal@latest/index.html}{\texttt{ Universal Render Pipeline}} uses {\bfseries{Absolute World}} as its default world space.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_position-_node_autotoc_md3607}{}\doxysubsection{\texorpdfstring{Upgrading from previous versions}{Upgrading from previous versions}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_position-_node_autotoc_md3607}
If you use a Position Node in {\bfseries{World}} space on a graph authored in Shader Graph version 6.\+7.\+0 or earlier, it automatically upgrades the selection to {\bfseries{Absolute World}}. This ensures that the calculations on your graph remain accurate to your expectations, since the {\bfseries{World}} output might change.

If you use a Position Node in {\bfseries{World}} space in the High Definition Render Pipeline to manually calculate \href{https://docs.unity3d.com/Packages/com.unity.render-pipelines.high-definition@latest?preview=1&subfolder=/manual/Camera-Relative-Rendering.html}{\texttt{ Camera Relative}} world space, you can now change your node from {\bfseries{Absolute World}} to {\bfseries{World}}, which lets you use \href{https://docs.unity3d.com/Packages/com.unity.render-pipelines.high-definition@latest?preview=1&subfolder=/manual/Camera-Relative-Rendering.html}{\texttt{ Camera Relative}} world space out of the box.
