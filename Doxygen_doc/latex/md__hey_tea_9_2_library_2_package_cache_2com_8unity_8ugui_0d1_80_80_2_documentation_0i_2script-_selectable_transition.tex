 Within a selectable component there are several transition options depending on what state the selectable is currently in. The different states are\+: normal, highlighted, pressed and disabled.

{\bfseries{None}}   &This option is for the button to have no state effects at all.    \\\cline{1-2}
{\bfseries{Color Tint}}   &Changes the colour of the button depending on what state it is in. It is possible to select the colour for each individual state. It is also possible to set the Fade Duration between the different states. The higher the number is, the slower the fade between colors will be.    \\\cline{1-2}
{\bfseries{Sprite Swap}}   &Allows different sprites to display depending on what state the button is currently in, the sprites can be customised.    \\\cline{1-2}
{\bfseries{Animation}}   &Allows animations to occur depending on the state of the button, an animator component must exist in order to use animation transition.
