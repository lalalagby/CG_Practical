The standard UI objects available from the menu ({\bfseries{Game\+Object \texorpdfstring{$>$}{>} Create UI}}) all have Canvas Renderers attached wherever they are required but you may need to add this component manually for custom UI objects. Although there are no properties exposed in the inspector, a few properties and function can be accessed from scripts -\/ see the \href{https://docs.unity3d.com/ScriptReference/CanvasRenderer.html.md}{\texttt{ Canvas\+Renderer}} page in the Script Reference for full details.
