\doxysubsection{Detailed Description}
Attribute that specifies a class as an editor for an extended \doxylink{namespace_unity_editor_1_1_timeline}{Timeline} type. 

Use this attribute on a class that extends \doxylink{class_unity_editor_1_1_timeline_1_1_clip_editor}{Clip\+Editor}, \doxylink{class_unity_editor_1_1_timeline_1_1_track_editor}{Track\+Editor}, or \doxylink{class_unity_editor_1_1_timeline_1_1_marker_editor}{Marker\+Editor} to specify either the Playable\+Asset, Marker, or Track\+Asset derived classes for associated customization.
