 The Sample Texture 2D Array node samples a {\bfseries{Texture 2D Array}} asset and returns a {\bfseries{Vector 4}} color value. You can specify the {\bfseries{UV}} coordinates for a texture sample and use a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sampler-_state-_node}{Sampler State node}{0} to define a specific Sampler State. The node\textquotesingle{}s {\bfseries{Index}} input port specifies which index of a Texture 2D Array to sample.

For more information about Texture 2D Arrays, see \href{https://docs.unity3d.com/Manual/class-Texture2DArray.html}{\texttt{ Texture Arrays}} in the \doxylink{namespace_unity}{Unity} User manual.

In the following example, the Sample Texture 2D Array node samples a Texture array that has 4 different cloth normal maps. Change the number given to the {\bfseries{Index}} port as an input, and the Sample Texture 2D Array node can sample a specific normal map from the array. The {\bfseries{Index}} value changes the output the node sends to the Normal Unpack node, and the Normal (Tangent Space) Block node in the Master Stack.
