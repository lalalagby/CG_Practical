bool Unity\+Engine.\+Timeline.\+Timeline\+Asset.\+Editor\+Settings.\+scene\+Preview\hspace{0.3cm}{\ttfamily [get]}, {\ttfamily [set]}}



Set to false to ignore scene preview when this timeline is played by the \doxylink{namespace_unity_engine_1_1_timeline}{Timeline} window. 

When set to false, this setting will
\begin{DoxyItemize}
\item Disable scene preview when this timeline is played by the \doxylink{namespace_unity_engine_1_1_timeline}{Timeline} window.
\item Disable recording for all recordable tracks.
\item Disable play range in the \doxylink{namespace_unity_engine_1_1_timeline}{Timeline} window.
\item {\ttfamily Stop()} is not called on the {\ttfamily Playable\+Director} when switching between different {\ttfamily \doxylink{class_unity_engine_1_1_timeline_1_1_timeline_asset}{Timeline\+Asset}}s in the Timeline\+Window.
\end{DoxyItemize}

{\ttfamily scene\+Preview} will only be applied if the asset is the master timeline.
