The HD Scene Depth node uses a UV input to access the current Camera\textquotesingle{}s depth buffer. \doxylink{namespace_unity}{Unity} expects normalized screen coordinates for this value. You can also use this node to access the mipmaps in the depth buffer.

You can only use the HD Scene Depth node in the Fragment Shader Stage and with non-\/opaque materials.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_h_d-_scene-_depth-_node_autotoc_md3326}{}\doxysection{\texorpdfstring{Render pipeline compatibility}{Render pipeline compatibility}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_h_d-_scene-_depth-_node_autotoc_md3326}
