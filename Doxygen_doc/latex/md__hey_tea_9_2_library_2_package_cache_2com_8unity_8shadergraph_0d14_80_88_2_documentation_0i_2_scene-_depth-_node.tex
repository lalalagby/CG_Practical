Provides access to the current {\bfseries{Camera}}\textquotesingle{}s depth buffer using input {\bfseries{UV}}, which is expected to be normalized screen coordinates.

Note\+: Depth buffer access requires depth buffer to be enabled on the active {\bfseries{Render Pipeline}}. This process is different per {\bfseries{Render Pipeline}}. It is recommended you read the documentation of your active {\bfseries{Render Pipeline}} for information on enabling the depth buffer. If the depth buffer is unavailable this \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node}{Node}{0} will return mid grey.

Note\+: The executed HLSL code for this \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node}{Node}{0} is defined per {\bfseries{Render Pipeline}}, and different {\bfseries{Render Pipelines}} may produce different results. Custom {\bfseries{Render Pipelines}} that wish to support this \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node}{Node}{0} will also need to explicitly define the behaviour for it. If undefined this \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node}{Node}{0} will return 1 (white).
