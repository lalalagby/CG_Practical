\doxysubsection{Detailed Description}
A four-\/character code. 

A four-\/character code is a struct containing four byte characters totalling a single {\ttfamily int}. Four\+CCs are frequently used in the input system to identify the format of data sent to or from the native backend representing events, input device state or commands sent to input devices.
