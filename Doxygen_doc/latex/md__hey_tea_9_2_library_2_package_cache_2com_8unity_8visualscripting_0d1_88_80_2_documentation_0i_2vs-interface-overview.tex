 Visual Scripting\textquotesingle{}s main window is the Graph window.



The Graph window has five main elements\+:


\begin{DoxyItemize}
\item The \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Graph Editor\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}, where you create, arrange, and connect nodes.
\item The \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}fuzzy finder\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}, which you can use to find nodes and add them to your graph.
\item The \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Graph toolbar\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}, where you can change settings specific to your view in the Graph Editor and perform some common layout operations.
\item The \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Graph Inspector\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}, where you can view detailed information about your nodes and configure additional settings for your graph.
\item The \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Blackboard\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}, where you can define and edit variables to use in your graphs. ~\newline

\end{DoxyItemize}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-interface-overview_autotoc_md5385}{}\doxysection{\texorpdfstring{The Graph Editor}{The Graph Editor}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-interface-overview_autotoc_md5385}
The Graph Editor is the center editing area of the Graph window.



You can use the Graph Editor to create your Visual Scripting graphs. You can create nodes and connect them with edges.

You can change some default shortcuts and behaviors in the Graph Editor through your control scheme. For more information on the available control schemes in Visual Scripting, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-control-schemes}{Choose a control scheme}{0}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-interface-overview_autotoc_md5386}{}\doxysection{\texorpdfstring{The fuzzy finder}{The fuzzy finder}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-interface-overview_autotoc_md5386}
The fuzzy finder is a searchable menu that lists every node available in Visual Scripting. Right-\/click anywhere in the Graph Editor to open the fuzzy finder.



Search for a node by name with the Search bar, or open a category from the list to view related nodes. For example, nodes related to the creation or manipulation of variables are in the {\bfseries{Variables}} category.

You can add new nodes to Visual Scripting from your own code, from other packages, or from other \doxylink{namespace_unity}{Unity} features. For more information on how to add nodes to the fuzzy finder, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-configuration}{Configure project settings}{0}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-interface-overview_autotoc_md5387}{}\doxysection{\texorpdfstring{The Graph toolbar}{The Graph toolbar}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-interface-overview_autotoc_md5387}
The Graph toolbar lets you display or hide the Graph Inspector and Blackboard. The Graph toolbar also includes a breadcrumb trail for navigation through nested graphs that displays your current location. Select a graph from the trail to return to that graph file.

You can also configure some additional settings that control how nodes display in the Graph Editor.

The Graph Inspector provides additional information about an open graph, or about any node you select in the Graph Editor.



If a node requires additional configuration, you can use the Graph Inspector to set these values.

To display or hide the Graph Inspector, select {\bfseries{Graph Inspector}} (\texorpdfstring{$<$}{<}img src="{}images\textbackslash{}vs-\/graph-\/inspector-\/icon.\+png"{} alt="{}\+The Graph Inspector icon, of a grey circle with a black, lower case letter "{}i"{} in the center."{}/\texorpdfstring{$>$}{>}) from the toolbar.

To move the Graph Inspector to the other side of the Graph window, select either {\bfseries{Dock Right}} () or {\bfseries{Dock Left}} ().\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-interface-overview_autotoc_md5389}{}\doxysection{\texorpdfstring{The Blackboard}{The Blackboard}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-interface-overview_autotoc_md5389}
The Blackboard provides options to configure and manage variables in a graph. The Blackboard divides variables into five distinct scopes, across five tabs\+: {\bfseries{Graph}}, {\bfseries{Object}}, {\bfseries{Scene}}, {\bfseries{App}}, and {\bfseries{Saved}}.



For more information on the available variable scopes in Visual Scripting, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-variables}{Variables}{0}.

To display or hide the Blackboard, select {\bfseries{Blackboard}} (\texorpdfstring{$<$}{<}img src="{}images\textbackslash{}vs-\/blackboard-\/icon.\+png"{} alt="{}\+The Blackboard icon. A letter "{}x"{} is surrounded by angled brackets."{}/\texorpdfstring{$>$}{>}) from the toolbar.

To move the Blackboard to the other side of the Graph window, select either {\bfseries{Dock Right}} () or {\bfseries{Dock Left}} ().
