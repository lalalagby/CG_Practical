The Scriptable Render Pipeline (SRP) is a \doxylink{namespace_unity}{Unity} feature that allows you to write C\# scripts to control the way \doxylink{namespace_unity}{Unity} renders each frame. SRP \doxylink{namespace_core}{Core} is a package that makes it easier to create or customize an SRP.

SRP \doxylink{namespace_core}{Core} contains reusable code, including boilerplate code for working with platform-\/specific graphics APIs, utility functions for common rendering operations, and the shader libraries used in the High Definition Render Pipeline (HDRP) and Universal Render Pipeline (URP).

If you are creating a custom SRP from scratch or customizing a prebuilt SRP, using SRP \doxylink{namespace_core}{Core} will save you time.

For more information on SRP, including a guide to getting started with a custom SRP, see the \href{https://docs.unity3d.com/Manual/ScriptableRenderPipeline.html}{\texttt{ SRP documentation}}. For more information on \doxylink{namespace_unity}{Unity}\textquotesingle{}s prebuilt SRPs, see the \href{https://docs.unity3d.com/Packages/com.unity.render-pipelines.universal@latest}{\texttt{ Universal Render Pipeline (URP) documentation}}, or the \href{https://docs.unity3d.com/Packages/com.unity.render-pipelines.high-definition@latest}{\texttt{ High Definition Render Pipeline (HDRP) documentation}}.
