To report documentation problems, please use the feedback section at the bottom of the page containing the problem.

To report bugs related to the Input \doxylink{namespace_system}{System} Please follow \doxylink{namespace_unity}{Unity}\textquotesingle{}s standard \href{https://unity3d.com/unity/qa/bug-reporting}{\texttt{ bug reporting guidelines}}. Don\textquotesingle{}t forget to submit a Project that the developer who picks up your report can use to reproduce the issue. Be sure to mention that the bug is specific to the Input \doxylink{namespace_system}{System} package in the description, so it gets forwarded to the correct team at \doxylink{namespace_unity}{Unity}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8inputsystem_0d1_86_83_2_documentation_0i_2_contributing_autotoc_md1296}{}\doxysection{\texorpdfstring{Discussion}{Discussion}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8inputsystem_0d1_86_83_2_documentation_0i_2_contributing_autotoc_md1296}
To ask questions or discuss the Input \doxylink{namespace_system}{System}, see the \href{https://forum.unity.com/forums/new-input-system.103/}{\texttt{ dedicated section on Unity\textquotesingle{}s forum}}. This is also the best place to post feature requests.
