 The Version Control package will allow you to use \doxylink{namespace_unity}{Unity} version control for your projects in the \doxylink{namespace_unity}{Unity} Editor.

\doxylink{namespace_unity}{Unity} VC integrates version control in \doxylink{namespace_unity}{Unity} that will abstract version control complexity. It will also enable you to work collaboratively on more complex projects by providing additional VCS features such as branching, locking, merging, and a standalone GUI.

The Version Control package follows the \doxylink{namespace_unity}{Unity} support schedule. Currently, supported versions are\+:


\begin{DoxyItemize}
\item 2020.\+3
\item 2021.\+3
\item 2022.\+2
\item 2023.\+1
\item 2023.\+2
\end{DoxyItemize}
