\doxysubsection{Detailed Description}
The \doxylink{class_stove_counter}{Stove\+Counter} class is responsible for handling interactions with the stove, including cooking progress and managing objects on the stove. 

\begin{DoxyAuthor}{Author}
Yong Wu, Bingyu Guo
\end{DoxyAuthor}


This class extends the \doxylink{class_base_counter}{Base\+Counter} and implements the \doxylink{interface_i_has_progress}{IHas\+Progress} interface. It manages the cooking process, handles player interactions, and updates the progress state.

\begin{DoxyNote}{Note}
milk\+Tea\+Material\+Type contains six types\+: tea\+Base, basic\+Add, ingredients, fruit, none, un treat. \doxylink{class_hey_tea_object}{Hey\+Tea\+Object} includes the objects contained in tea\+Base, basic\+Add, ingredients, fruit, none, un treat.
\begin{DoxyItemize}
\item tea\+Base\+: tea, milk, milktea
\item basic\+Add\+: sugar, ice
\item ingredients\+: red bean cooked, pearl cooked
\item fruit\+: orange slice, grape slice, strawberry slice
\item none\+: pot, cup
\item un treat\+: bagged pearl, bagged red bean, bagged sugar, grape, orange, strawberry 
\end{DoxyItemize}
\end{DoxyNote}
