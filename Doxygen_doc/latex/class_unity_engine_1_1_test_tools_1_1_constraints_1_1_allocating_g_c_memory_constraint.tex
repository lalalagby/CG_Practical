\doxysubsection{Detailed Description}
An NUnit test constraint class to test whether a given block of code makes any GC allocations. 

Use this class with NUnit\textquotesingle{}s Assert.\+That() method to make assertions about the GC behaviour of your code. The constraint executes the delegate you provide, and checks if it has caused any GC memory to be allocated. If any GC memory was allocated, the constraint passes; otherwise, the constraint fails.

Usually you negate this constraint to make sure that your delegate does not allocate any GC memory. This is easy to do using the \doxylink{class_unity_engine_1_1_test_tools_1_1_constraints_1_1_is}{Is} class\+:
