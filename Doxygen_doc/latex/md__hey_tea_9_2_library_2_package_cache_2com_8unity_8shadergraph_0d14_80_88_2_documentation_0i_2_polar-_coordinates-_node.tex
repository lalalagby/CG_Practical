Converts the value of input {\bfseries{UV}} to polar coordinates. In mathematics, the polar coordinate system is a two-\/dimensional coordinate system in which each point on a plane is determined by a distance from a reference point and an angle from a reference direction.

The resulting effect is that the x channel of the input to {\bfseries{UV}} is converted to a distance value from the point specified by the value of input {\bfseries{Center}} and the y channel of same input is converted to the value of an angle of rotation around that point.

These values can be scaled by the values of inputs {\bfseries{Radial Scale}} and {\bfseries{Length Scale}} respectively.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_polar-_coordinates-_node_autotoc_md3590}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_polar-_coordinates-_node_autotoc_md3590}
