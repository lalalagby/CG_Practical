\doxysubsection{Detailed Description}
This behaviour can be attached to any object to make it shake in response to Impulses. 

This behaviour can be attached to the main Camera with the \doxylink{class_cinemachine_1_1_cinemachine_brain}{Cinemachine\+Brain}, to allow the main camera to shake without putting Listeners on the virtual cameras. In this case, camera shake is not dependent on the active virtual camera.

It is also possible to put this behaviour on other scene objects to shake them in response to impulses.
