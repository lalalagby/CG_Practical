\doxysubsection{Detailed Description}
This class is used to hold default values for an implementation of \doxylink{interface_unity_editor_1_1_timeline_1_1_i_sequence_state}{ISequence\+State}. It could be removed in a phase 2, but it is currently used to limit the scope of this refactoring\+: it allows client code to access sequence info without having to worry about {\ttfamily current\+Sequence} being null. In the future this should be removed and we should pass around the correct data structure (i.\+e. \doxylink{class_unity_editor_1_1_timeline_1_1_sequence_state}{Sequence\+State} OR \doxylink{class_unity_editor_1_1_timeline_1_1_window_state}{Window\+State}) based on the situation.
