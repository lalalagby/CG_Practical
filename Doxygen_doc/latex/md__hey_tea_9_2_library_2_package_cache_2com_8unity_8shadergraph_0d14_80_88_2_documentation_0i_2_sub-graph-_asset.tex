The {\bfseries{Subgraph Asset}} is a new {\bfseries{Asset}} type introduced with the Shader Graph. A {\bfseries{Subgraph Asset}} defines a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sub-graph}{Subgraph}{0}. This is different to a Shader Graph. You can create a {\bfseries{Subgraph Asset}} from the \href{https://docs.unity3d.com/Manual/ProjectView.html.md}{\texttt{ Project window}} from the {\bfseries{Create}} menu via {\bfseries{Subgraph}} in the {\bfseries{Shader}} sub-\/menu.

You can open the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_graph-_window}{Shader Graph Window}{0} by double clicking a {\bfseries{Subgraph Asset}} or by clicking {\bfseries{Open Shader Editor}} in the \href{https://docs.unity3d.com/Manual/UsingTheInspector.html}{\texttt{ Inspector}} when the {\bfseries{Subgraph Asset}} is selected.
