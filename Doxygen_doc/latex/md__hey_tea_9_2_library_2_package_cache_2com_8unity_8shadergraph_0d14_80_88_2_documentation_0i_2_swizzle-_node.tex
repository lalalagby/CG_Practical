Creates a new \href{https://docs.unity3d.com/Manual/VectorCookbook.html}{\texttt{ vector}} from the reordered elements of the input vector. This is called swizzling.

To specify how input elements should be swizzled, enter a formatting string in the input mask. To invert the order of the input elements, for example, use the string "{}wzyx"{} or "{}abgr"{}.

The length of the input mask determines the dimensions of the output vector. The error "{}\+Invalid Mask"{} indicates an input mask value which includes one or more channels that do not exist in the input vector.

To output a vector3 with the x, y and z elements of the input vector, for example, use the input mask “xyz” or “rgb”.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_swizzle-_node_autotoc_md4004}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_swizzle-_node_autotoc_md4004}
