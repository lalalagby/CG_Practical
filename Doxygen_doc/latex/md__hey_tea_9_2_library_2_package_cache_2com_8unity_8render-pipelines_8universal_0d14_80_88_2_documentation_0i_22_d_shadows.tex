The {\bfseries{Composite Shadow Caster 2D}} merges the shape of multiple {\bfseries{Shadow Caster 2Ds}} together as a single {\bfseries{Shadow Caster 2D}}. Add the {\bfseries{Composite Shadow Caster 2D}} component to a Game\+Object by going to menu\+: {\bfseries{Component \texorpdfstring{$>$}{>} Rendering \texorpdfstring{$>$}{>} 2D \texorpdfstring{$>$}{>} Composite Shadow Caster 2D}}, then parent Game\+Objects with the {\bfseries{Shadow Caster 2D}} component to it. The Composite component merges all Shadow Caster 2Ds within this hierarchy, including any Shadow Caster 2Ds on the parent as well.
