Returns the linear parameter that produces the interpolant specified by input {\bfseries{T}} within the range of input {\bfseries{A}} to input {\bfseries{B}}.

{\bfseries{Inverse Lerp}} is the inverse operation of the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_lerp-_node}{Lerp Node}{0}. It can be used to determine what the input to a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_lerp-_node}{Lerp}{0} was based on its output.

For example, the value of a {\bfseries{Lerp}} between 0 and 2 with a {\bfseries{T}} value of 0.\+5 is 1.
