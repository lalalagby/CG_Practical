\doxysubsection{Detailed Description}
A 2D planar motion vector computed from an up+down button pair and a left+right button pair. 

This composite allows to grab arbitrary buttons from a device and arrange them in a D-\/\+Pad like configuration. Based on button presses, the composite will return a normalized direction vector (normalization can be turned off via \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_vector2_composite_a0f1cf5d34efe319affccb88a969028f7}{mode}).

Opposing motions cancel each other out. This means that if, for example, both the left and right horizontal button are pressed, the resulting horizontal movement value will be zero.
