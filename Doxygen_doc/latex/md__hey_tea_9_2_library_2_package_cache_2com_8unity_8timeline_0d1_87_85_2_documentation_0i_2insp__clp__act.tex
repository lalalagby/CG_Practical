By default, each Activation clip is named "{}\+Active"{}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2insp__clp__act_autotoc_md4655}{}\doxysection{\texorpdfstring{Clip Timing properties}{Clip Timing properties}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2insp__clp__act_autotoc_md4655}
Use the {\bfseries{Clip Timing}} properties to change the position and duration of the Activation clip.

Most timing properties are expressed in both seconds (s) and frames (f). When specifying seconds to modify a {\bfseries{Clip Timing}} property, all decimal values are accepted. When specifying frames, only integer values are accepted. For example, if you attempt to enter 12.\+5 in a frames (f) field, it is set to 12 frames.

Depending on \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2clp__about}{the selected Clip Edit mode}{0}, changing the {\bfseries{Start}}, {\bfseries{End}}, or {\bfseries{Duration}} may ripple or replace Activation clips on the same track.
