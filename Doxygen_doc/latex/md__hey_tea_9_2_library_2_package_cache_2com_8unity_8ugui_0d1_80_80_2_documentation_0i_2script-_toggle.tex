The Toggle control allows the user to switch an option on or off. You can also combine several toggles into a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_toggle_group}{Toggle Group}{0} in cases where only one of a set of options should be on at once.

The Toggle has a single event called {\itshape On Value Changed} that responds when the user changes the current value. The new value is passed to the event function as a {\ttfamily boolean} parameter. Typical use cases for Toggles include\+:


\begin{DoxyItemize}
\item Switching an option on or off (eg, playing music during a game).
\item Letting the user confirm they have read a legal disclaimer.
\item Choosing one of a set of options (eg, a day of the week) when used in a Toggle Group.
\end{DoxyItemize}

Note that the {\bfseries{Toggle}} is a parent that provides a clickable area to children. If the {\bfseries{Toggle}} has no children (or they are disabled) then it is not clickable.
