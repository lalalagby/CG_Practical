\doxysubsection{Detailed Description}
Use this attribute to add a menu item to a context menu. Used to indicate path and priority that are auto added to the menu (examples can be found on ). 

Unlike Menu item, \doxylink{class_unity_editor_1_1_timeline_1_1_actions_1_1_menu_entry_attribute}{Menu\+Entry\+Attribute} doesn\textquotesingle{}t handle shortcuts in the menu name. See \doxylink{class_unity_editor_1_1_timeline_1_1_actions_1_1_timeline_shortcut_attribute}{Timeline\+Shortcut\+Attribute}.
