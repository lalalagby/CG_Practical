 Use the {\ttfamily Burst\+Compile} attribute on an assembly to set options for all \doxylink{namespace_burst}{Burst} jobs and function-\/pointers within the assembly\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\ \#}}
\DoxyCodeLine{[assembly:\ BurstCompile(CompileSynchronously\ =\ \textcolor{keyword}{true})]}

\end{DoxyCode}


For example, if an assembly just contains game code which needs to run quickly, you can use\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\ \#}}
\DoxyCodeLine{[assembly:\ BurstCompile(OptimizeFor\ =\ \mbox{\hyperlink{namespace_unity_1_1_burst_aeae1a1ecc1c6ea7b35442c0175b325ad}{OptimizeFor}}.FastCompilation)]}

\end{DoxyCode}


This means that \doxylink{namespace_burst}{Burst} compiles the code as fast as it possibly can, which means that you can iterate on the game code much more quickly. It also means that other assemblies compile as they did before, which gives you more control on how \doxylink{namespace_burst}{Burst} works with your code.

Assembly-\/level {\ttfamily Burst\+Compile} attributes iterate with any job or function-\/pointer attribute, and also with any globally set options from the \doxylink{namespace_burst}{Burst} Editor menu. \doxylink{namespace_burst}{Burst} prioritizes assembly level attributes in the following order\+:


\begin{DoxyEnumerate}
\item \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2editor-burst-menu}{Editor menu settings}{0} take precedence. For example, if you enable {\bfseries{Native Debug Compilation}} from the \doxylink{namespace_burst}{Burst} menu, \doxylink{namespace_burst}{Burst} always compiles your code ready for debugging.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item \doxylink{namespace_burst}{Burst} checks any {\ttfamily Burst\+Compile} attribute on a job or function-\/pointer. If you have {\ttfamily Compile\+Synchronously = true} in {\ttfamily Burst\+Compile}, then \doxylink{namespace_burst}{Burst} compiles synchronously
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Otherwise, \doxylink{namespace_burst}{Burst} sources any remaining settings from any assembly level attribute.
\end{DoxyEnumerate}
