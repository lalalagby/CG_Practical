Provides a reference to a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sub-graph-_asset}{Subgraph Asset}{0}. All ports on the reference node are defined by the properties and outputs defined in the Subgraph Asset. This is useful for sharing functionality between graphs or duplicating the same functionality within a graph.

The preview used for a Subgraph Node is determined by the first port of that \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sub-graph}{Subgraph}{0} Output node. Valid \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_data-_types}{Data Types}{0} for the first port are {\ttfamily Float}, {\ttfamily Vector 2}, {\ttfamily Vector 3}, {\ttfamily Vector 4}, {\ttfamily Matrix2}, {\ttfamily Matrix3}, {\ttfamily Matrix4}, and {\ttfamily Boolean}. Any other data type will produce an error in the preview shader and the Subgraph will become invalid.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sub-graph-_node_autotoc_md3989}{}\doxysection{\texorpdfstring{Subgraph Nodes and Shader Stages}{Subgraph Nodes and Shader Stages}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sub-graph-_node_autotoc_md3989}
If a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node}{Node}{0} within a Subgraph specifies a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_stage}{Shader Stage}{0}, such as how \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sample-_texture-2_d-_node}{Sample Texture 2D Node}{0} specifies the {\bfseries{fragment}} Shader Stage, then that entire Subgraph) is now locked to that stage. As such a Subgraph node that references the graph will also be locked to that Shader Stage.

Furthermore, when an \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_edge}{Edge}{0} connected to an output \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_port}{Port}{0} on a {\bfseries{Subgraph Node}} flows into a port on the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_master-_stack}{Master Stack}{0} that {\bfseries{Subgraph Node}} is now locked to the Shader Stage of that \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_block-_node}{Block Node}{0} in the Master Stack.
