{\bfseries{LOD}} 

Float 

LOD 

\doxysubsubsubsubsection*{NOTE}



The {\bfseries{LOD}} Input port only displays if {\bfseries{Mip Sampling Mode}} is {\bfseries{LOD}}. For more information, see \href{\#additional-node-settings}{\texttt{ Additional Node Settings}}.

 The specific mip that the node uses when sampling the Texture.  

{\bfseries{Bias}} 

Float 

Bias 

\doxysubsubsubsubsection*{NOTE}



The {\bfseries{Bias}} Input port only displays if {\bfseries{Mip Sampling Mode}} is {\bfseries{Bias}}. For more information, see \href{\#additional-node-settings}{\texttt{ Additional Node Settings}}.

 If {\bfseries{Use Global Mip Bias}} is enabled, \doxylink{namespace_unity}{Unity} adds this Bias amount to the Global Mip Bias for a texture\textquotesingle{}s mip calculation. If {\bfseries{Global Mip Bias}} is disabled, \doxylink{namespace_unity}{Unity} uses this Bias amount instead of the Global Mip Bias.  

{\bfseries{DDX}} 

Float 

DDX 

\doxysubsubsubsubsection*{NOTE}



The {\bfseries{DDX}} Input port only displays if {\bfseries{Mip Sampling Mode}} is {\bfseries{Gradient}}. For more information, see \href{\#additional-node-settings}{\texttt{ Additional Node Settings}}.

 The specific DDX value to use to calculate the Texture\textquotesingle{}s mip when sampling. For more information on DDX values for mipmaps, see \href{https://docs.unity3d.com/Documentation/Manual/texture-mipmaps-introduction.html}{\texttt{ Mipmaps introduction}} in the \doxylink{namespace_unity}{Unity} User Manual..
