 Connections control the flow of logic and data in a Script Graph\textquotesingle{}s nodes.

To connect nodes in a Script Graph\+:


\begin{DoxyEnumerate}
\item With a Script Graph \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-open-graph-edit}{open in the Graph window}{0}, either find an existing node where you want to make a connection, or \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-add-node-to-graph}{add a new node to your Script Graph}{0}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Do one of the following\+:
\begin{DoxyItemize}
\item Connect to a new node.
\item Connect to an existing node.
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-creating-connections_autotoc_md5298}{}\doxysubsection{\texorpdfstring{Connect to a new node}{Connect to a new node}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-creating-connections_autotoc_md5298}

\begin{DoxyEnumerate}
\item Select a port and point to a blank area in your graph to start the connection.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select again to open the fuzzy finder.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select an entry to automatically add that node at the end of your connection.
\end{DoxyEnumerate}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-creating-connections_autotoc_md5299}{}\doxysubsection{\texorpdfstring{Connect to an existing node}{Connect to an existing node}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-creating-connections_autotoc_md5299}

\begin{DoxyEnumerate}
\item Select a port and point to an existing port on another node.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select the port to make the connection.
\end{DoxyEnumerate}

To delete a connection between two nodes\+:


\begin{DoxyEnumerate}
\item With a Script Graph open in the Graph window, right-\/click the port at either end of a connection.
\end{DoxyEnumerate}
