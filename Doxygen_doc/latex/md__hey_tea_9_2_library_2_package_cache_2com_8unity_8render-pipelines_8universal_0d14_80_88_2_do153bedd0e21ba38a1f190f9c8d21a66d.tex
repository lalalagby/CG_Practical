 URP uses the Terrain Lit shader for \doxylink{namespace_unity}{Unity} Terrain. This shader is a simpler version of the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2lit-shader}{Lit shader}{0}. A Terrain can use a Terrain Lit Material with up to eight \href{https://docs.unity3d.com/Manual/class-TerrainLayer.html}{\texttt{ Terrain Layers}}.

~\newline
{\itshape A Terrain Game\+Object rendered with the Terrain Lit shader.}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do153bedd0e21ba38a1f190f9c8d21a66d_autotoc_md2475}{}\doxysection{\texorpdfstring{Terrain Lit Material properties}{Terrain Lit Material properties}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do153bedd0e21ba38a1f190f9c8d21a66d_autotoc_md2475}

{\bfseries{Enable Height-\/based Blend}}   &Enable to have \doxylink{namespace_unity}{Unity} take the height values from the blue channel of the {\bfseries{Mask Map}} Texture.~\newline
~\newline
If you do not enable this property, \doxylink{namespace_unity}{Unity} blends the Terrain Layers based on the weights painted in the splatmap Textures. When you disable this property and the Terrain Lit Shader Material is assigned to a Terrain, URP adds an additional option {\bfseries{Opacity as Density Blend}} for each Terrain Layer that is added to that Terrain in the Paint Texture Tool Inspector.~\newline
~\newline
{\bfseries{Note}}\+: \doxylink{namespace_unity}{Unity} ignores this option when more than four Terrain Layers are on the Terrain.    \\\cline{1-2}
\&\#160;\&\#160;\&\#160;\&\#160;{\itshape Height Transition}   &Select the size in world units of the smooth transition area between Terrain Layers.    \\\cline{1-2}
{\bfseries{Enable Per-\/pixel Normal}}   &Enable to have \doxylink{namespace_unity}{Unity} sample the normal map Texture on a per-\/pixel level, preserving more geometry details for distant terrain parts. \doxylink{namespace_unity}{Unity} generates a geometry normal map at runtime from the heightmap, rather than the Mesh geometry. This means you can have high-\/resolution Mesh normals, even if your Mesh is low resolution.~\newline
~\newline
{\bfseries{Note}}\+: This option only works if you enable {\bfseries{Draw Instanced}} on the Terrain.   \\\cline{1-2}

To create a Material compatible with a Terrain Game\+Object\+:


\begin{DoxyEnumerate}
\item Create a new Material ({\bfseries{Assets}} \texorpdfstring{$>$}{>} {\bfseries{Create}} \texorpdfstring{$>$}{>} {\bfseries{Material}}).
\item Select the new Material.
\item In the Inspector, click the {\bfseries{Shader}} drop-\/down, and select {\bfseries{Universal\&\#160;Render\&\#160;Pipeline}} \texorpdfstring{$>$}{>} {\bfseries{Terrain}} \texorpdfstring{$>$}{>} {\bfseries{Lit}}.
\end{DoxyEnumerate}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do153bedd0e21ba38a1f190f9c8d21a66d_autotoc_md2477}{}\doxysection{\texorpdfstring{Assign a Terrain Lit Material to a Terrain Game\+Object}{Assign a Terrain Lit Material to a Terrain Game\+Object}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do153bedd0e21ba38a1f190f9c8d21a66d_autotoc_md2477}
To assign a Terrain Lit Material to a Terrain Game\+Object\+:


\begin{DoxyEnumerate}
\item Select a Terrain Game\+Object.
\item In the Inspector, click the gear icon on the right side of the Terrain Inspector toolbar to open the {\bfseries{Terrain Settings}} section.
\item In the {\bfseries{Material}} property, select a Terrain Lit Material. Either use the Object picker (circle icon), or drag and drop the Material onto the property.
\end{DoxyEnumerate}

To use the {\bfseries{Paint Holes}} tool on a Terrain, ensure that the {\bfseries{Terrain Holes}} check box in your project\textquotesingle{}s URP Asset is checked. Otherwise, the Terrain holes are absent when you build the application.
