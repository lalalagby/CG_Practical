{\bfseries{{\bfseries{Depth Test}}}}&\multicolumn{3}{l|}{{\bfseries{This property is only exposed if {\bfseries{Allow Material Override}} is enabled for this Shader Graph. ~\newline
Use this property to set the conditions under which pixels pass or fail depth testing. The GPU does not draw pixels that fail a depth test. If you choose anything other than {\bfseries{LEqual}} (the default setting for this property), consider also changing the rendering order of this material. ~\newline
 ~\newline
Options\+: 
\begin{DoxyItemize}
\item {\bfseries{LEqual}} (default)\+: \doxylink{namespace_unity}{Unity} draws the pixel, if its depth value is less than or equal to the value on the depth texture. Less\+: \doxylink{namespace_unity}{Unity} draws pixels of the affected surface if their coordinates are less than the current depth buffer value.  
\item {\bfseries{Never}}\+: \doxylink{namespace_unity}{Unity} never draws the pixels of the affected surface.  
\item {\bfseries{Less}}\+: \doxylink{namespace_unity}{Unity} draws pixels of the affected surface if their coordinates are less than the current depth buffer value.  
\item {\bfseries{Greater}}\+: \doxylink{namespace_unity}{Unity} draws pixels of the affected surface if their coordinates are greater than the current depth buffer value.  
\item {\bfseries{GEqual}}\+: \doxylink{namespace_unity}{Unity} draws pixels of the affected surface if their coordinates are greater than or equal to the current depth buffer value.  
\item {\bfseries{Equal}}\+: \doxylink{namespace_unity}{Unity} draws pixels of the affected surface if their coordinates are equal to the current depth buffer value.  
\item {\bfseries{Not\+Equal}}\+: \doxylink{namespace_unity}{Unity} draws pixels of the affected surface if their coordinates are not the same as the current depth buffer value.  
\item {\bfseries{Always}}\+: \doxylink{namespace_unity}{Unity} draws this surface to your screen regardless of its z-\/coordinate.  
\end{DoxyItemize}This option\textquotesingle{}s functionality corresponds to the command \href{https://docs.unity3d.com/Manual/SL-ZTest.html}{\texttt{ ZTest}} in \href{https://docs.unity3d.com/Manual/SL-Reference.html}{\texttt{ Shader\+Lab}}. To override this setting in a \href{https://docs.unity3d.com/ScriptReference/Rendering.RenderStateBlock.html}{\texttt{ Render\+State\+Block}}, set the \href{https://docs.unity3d.com/ScriptReference/Rendering.RenderStateBlock-depthState.html}{\texttt{ depth\+State}} property. }}
