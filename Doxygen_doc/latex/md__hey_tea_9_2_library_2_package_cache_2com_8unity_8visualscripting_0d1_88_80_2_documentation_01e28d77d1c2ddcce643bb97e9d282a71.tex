In the following example, the On Button Input node listens for the user to press the button or key assigned to the {\bfseries{Jump}} axes in the Input Manager. When the user presses the button, the On Button Input node triggers the Rigidbody Add Force node, which adds an {\bfseries{Impulse}} Force to the Rigidbody\textquotesingle{}s {\bfseries{Y}} axis\+:



The Add Force node makes the {\bfseries{Target}} Rigidbody lift into the air.
