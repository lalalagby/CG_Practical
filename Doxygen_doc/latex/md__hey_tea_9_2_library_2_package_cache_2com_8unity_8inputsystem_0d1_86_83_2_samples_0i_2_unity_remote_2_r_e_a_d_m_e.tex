 This sample is just a simple scene that lets you see try out the \doxylink{namespace_unity}{Unity} Remote app. The app is useful for quickly testing your project in the \doxylink{namespace_unity}{Unity} Editor without having to build and deploy to the device. This is supported for i\+OS and Android.

More detailed information about the \doxylink{namespace_unity}{Unity} Remote can be found in the \href{https://docs.unity3d.com/Manual/UnityRemote5.html}{\texttt{ Unity manual}}.

Instructions\+:


\begin{DoxyEnumerate}
\item Install the \doxylink{namespace_unity}{Unity} Remote app on your Android or i\+OS device.
\item Connect the device to your computer via USB. On Android, make sure you have the Android SDK installed and configured appropriately in \doxylink{namespace_unity}{Unity}. Also, USB debugging needs to be enabled.
\item In \doxylink{namespace_unity}{Unity}, go to {\ttfamily Edit \texorpdfstring{$>$}{>} Project Settings \texorpdfstring{$>$}{>} Editor} and select the device in the {\ttfamily \doxylink{namespace_unity}{Unity} Remote} section. 
\item Open the \doxylink{namespace_unity}{Unity} Remote app on the device.
\item Open the \href{UnityRemoteTest.unity}{\texttt{ {\ttfamily Unity\+Remote\+Test.\+unity}}} scene from this sample.
\item Enter play mode. After a short delay, the \doxylink{namespace_unity}{Unity} Remote app on your device should switch to display the scene and you should be able to interact with the scene in the editor using the device. 
\end{DoxyEnumerate}
