This User Guide was designed to provide first time users of Text\+Mesh Pro with a basic overview of the features and functionality of the tool.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8textmeshpro_0d3_80_86_2_documentation_0i_2_text_mesh_pro_autotoc_md4499}{}\doxysubsubsection{\texorpdfstring{{\bfseries{Installation}}}{{\bfseries{Installation}}}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8textmeshpro_0d3_80_86_2_documentation_0i_2_text_mesh_pro_autotoc_md4499}
The Text\+Mesh Pro UPM package is already included with the \doxylink{namespace_unity}{Unity} Editor and as such does not require installation. Text\+Mesh Pro "{}\+TMP"{} does however require adding resources to your project which are essential for using Text\+Mesh Pro.

To import the "{}\texorpdfstring{$\ast$}{*}\+TMP Essential Resources\texorpdfstring{$\ast$}{*}"{}, please use the "{}\texorpdfstring{$\ast$}{*}\+Window -\/$>$ Text\+Mesh\+Pro -\/$>$ Import TMP Essential Resources\texorpdfstring{$\ast$}{*}"{} menu option. These resources will be added at the root of your project in the "{}\texorpdfstring{$\ast$}{*}\+Text\+Mesh Pro\texorpdfstring{$\ast$}{*}"{} folder.

The Text\+Mesh Pro package also includes additional resources and examples that will make discovering and learning about Text\+Mesh Pro\textquotesingle{}s powerful features easier. It is strongly recommended that first time users import these additional resources.

To import the "{}\texorpdfstring{$\ast$}{*}\+TMP Examples \& Extras\texorpdfstring{$\ast$}{*}"{}, please use the "{}\texorpdfstring{$\ast$}{*}\+Window -\/$>$ Text\+Mesh\+Pro -\/$>$ Import TMP Examples \& Extras\texorpdfstring{$\ast$}{*}"{} menu option. These resources will also be added in the same "{}\texorpdfstring{$\ast$}{*}\+Text\+Mesh Pro\texorpdfstring{$\ast$}{*}"{} folder inside your project.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8textmeshpro_0d3_80_86_2_documentation_0i_2_text_mesh_pro_autotoc_md4500}{}\doxysubsubsection{\texorpdfstring{{\bfseries{Quick Start}}}{{\bfseries{Quick Start}}}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8textmeshpro_0d3_80_86_2_documentation_0i_2_text_mesh_pro_autotoc_md4500}
There are two Text\+Mesh Pro components available. The first TMP text component is of type \texorpdfstring{$<$}{<}Text\+Mesh\+Pro\texorpdfstring{$>$}{>} and designed to work with the Mesh\+Renderer. This component is an ideal replacement for the legacy Text\+Mesh component.

To add a new \texorpdfstring{$<$}{<}Text\+Mesh\+Pro\texorpdfstring{$>$}{>} text object, go to\+: “\texorpdfstring{$\ast$}{*}\+Game\+Object-\/\texorpdfstring{$>$}{>}3D Object-\/\texorpdfstring{$>$}{>}Text\+Mesh\+Pro Text\texorpdfstring{$\ast$}{*}”.

The second TMP text component is of type \texorpdfstring{$<$}{<}Text\+Mesh\+Pro\+UGUI\texorpdfstring{$>$}{>} and designed to work with the Canvas\+Renderer and Canvas system. This component is an ideal replacement for the UI.\+Text component.

To add a new \texorpdfstring{$<$}{<}Text\+Mesh\+Pro\+UGUI\texorpdfstring{$>$}{>} text object, go to\+: “\texorpdfstring{$\ast$}{*}\+Game\+Object-\/\texorpdfstring{$>$}{>}UI-\/\texorpdfstring{$>$}{>}Text\+Mesh\+Pro Text\texorpdfstring{$\ast$}{*}”.

You may also wish to watch this \href{https://youtu.be/olnxlo-Wri4}{\texttt{ Getting Started}} short video which covers this topic.

We strongly recommend that you also watch the \href{https://youtu.be/qzJNIGCFFtY}{\texttt{ Font Asset Creation}} video as well as the \href{https://youtu.be/d2MARbDNeaA}{\texttt{ Working with Material Presets}} as these two topics is also key to working and getting the most out of Text\+Mesh Pro.

As mentionned in the Installation section of this guide, it is recommended that you import the "{}\texorpdfstring{$\ast$}{*}\+TMP Examples \& Extras\texorpdfstring{$\ast$}{*}"{} and take the time to explore each of the examples as they provide a great overview of the functionality of the tool and the many text layout and \href{http://digitalnativestudios.com/textmeshpro/docs/rich-text/}{\texttt{ rich text tags}} available in Text\+Mesh Pro.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8textmeshpro_0d3_80_86_2_documentation_0i_2_text_mesh_pro_autotoc_md4501}{}\doxysubsubsection{\texorpdfstring{{\bfseries{Support \& API Documentation}}}{{\bfseries{Support \& API Documentation}}}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8textmeshpro_0d3_80_86_2_documentation_0i_2_text_mesh_pro_autotoc_md4501}
