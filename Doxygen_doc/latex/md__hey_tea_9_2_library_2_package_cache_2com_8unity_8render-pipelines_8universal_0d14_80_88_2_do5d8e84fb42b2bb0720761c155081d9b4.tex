 {\bfseries{Volume Overrides}} let you change or extend the default properties in a Volume Profile.

URP implements post-\/processing effects as Volume Overrides. For example, the following image shows the Vignette post-\/processing effect in the URP Template Sample\+Scene.



In a the Volume Override, checkboxes to the left of each property let you enable or disable specific properties. If you disable a property, URP uses the Volumeâ€™s default value for that property instead. To turn all properties on or off, use the {\bfseries{All}} or {\bfseries{None}} shortcuts above the property list.
