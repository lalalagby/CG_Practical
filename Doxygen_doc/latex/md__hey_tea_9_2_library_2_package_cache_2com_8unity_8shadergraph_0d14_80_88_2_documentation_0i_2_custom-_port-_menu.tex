The Custom \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_port}{Port}{0} Menu is displayed in the {\bfseries{Node Settings}} tab of the \mbox{[}Graph Inspector\mbox{]}(Internal-\/\+Inspector) by clicking on the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_custom-_function-_node}{Custom Function Node}{0} and \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sub-graph}{Sub Graph}{0} output node. This menu allows you to add, remove, rename, reorder, and define the types of your custom input and output ports.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_custom-_port-_menu_autotoc_md3128}{}\doxysection{\texorpdfstring{How to Use}{How to Use}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_custom-_port-_menu_autotoc_md3128}
Select the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_custom-_function-_node}{Custom Function Node}{0} or the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sub-graph}{Sub Graph}{0} output node to view the Custom Port Menu in the Inspector. To close the menu, click anywhere in the graph or on another graph-\/element.

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_custom-_port-_menu_autotoc_md3129}{}\doxysubsection{\texorpdfstring{Adding and Removing Ports}{Adding and Removing Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_custom-_port-_menu_autotoc_md3129}
To add ports, click the {\ttfamily +} icon at the bottom right corner of the port list.



To remove ports, select a port using the hamburger icon on the left, and click the {\ttfamily -\/} icon at the bottom right corner of the port list.

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_custom-_port-_menu_autotoc_md3130}{}\doxysubsection{\texorpdfstring{Renaming Ports}{Renaming Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_custom-_port-_menu_autotoc_md3130}
To rename a port, double-\/click its text field and enter the new name. Currently, only the following characters are valid for port names\+: A-\/Z, a-\/z, 0-\/9, \+\_\+, ( ), and whitespace. If the name contains an invalid character, an error badge appears.

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_custom-_port-_menu_autotoc_md3131}{}\doxysubsection{\texorpdfstring{Reordering Ports}{Reordering Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_custom-_port-_menu_autotoc_md3131}
To reorder ports, click and hold the hamburger icon on the left, and drag the port to your desired place in the list.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_custom-_port-_menu_autotoc_md3132}{}\doxysubsection{\texorpdfstring{Changing Port Types}{Changing Port Types}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_custom-_port-_menu_autotoc_md3132}
To change a port type, use the Type drop-\/down menu on the right. See the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_data-_types}{Data Types}{0} page for a list of currently valid port types.
