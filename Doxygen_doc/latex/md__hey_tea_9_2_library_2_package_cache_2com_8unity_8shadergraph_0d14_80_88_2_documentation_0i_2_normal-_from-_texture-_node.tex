Converts a height map defined by input {\bfseries{Texture}} into a normal map. UV values and sampler state can be defined by inputs {\bfseries{UV}} and {\bfseries{Sampler}} respectively. If nothing is connected to these ports they will use default values from the inputs. See \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_port-_bindings}{Port Bindings}{0} for more information.

The strength of the created normal map can be defined by inputs {\bfseries{Offset}} and {\bfseries{Strength}}, where {\bfseries{Offset}} defines the maximum distance of a normal detail and {\bfseries{Strength}} acts as a multiplier to the result.
