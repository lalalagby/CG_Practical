The On Input \doxylink{namespace_system}{System} Event Vector 2 node listens for a specific Input Action from a \doxylink{class_player}{Player} Input component. The node can output two values as a Vector 2.

Use this node when you want to read input and return two values, such as a joystick or mouse position.

In the following example, an On Input \doxylink{namespace_system}{System} Event Vector 2 node uses the bindings assigned to the {\bfseries{Move}} Input Action. When a user presses a button from the {\bfseries{Move}} binding, Visual Scripting takes the Vector 2 value it receives from the Input \doxylink{namespace_system}{System} and sends it as an input to the Transform Set Position node\textquotesingle{}s {\bfseries{Value}} input port. The Vector 2 value changes the position of the Game\+Object associated with the {\bfseries{Target}} transform.



For this example, {\bfseries{Move}} uses the W, A, S, and D keys. The Game\+Object moves up in the scene when the user presses W, moves down when the user presses S, and moves left or right when the user presses A or D.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_099c24c63f7e03b902caa3d20cbbc815c_autotoc_md5456}{}\doxysection{\texorpdfstring{Related nodes}{Related nodes}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_099c24c63f7e03b902caa3d20cbbc815c_autotoc_md5456}
