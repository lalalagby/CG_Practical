Provides a dynamic branch to the shader. If input {\bfseries{Predicate}} is true the return output will be equal to input {\bfseries{True}}, otherwise it will be equal to input {\bfseries{False}}. This is determined per vertex or per pixel depending on shader stage. Both sides of the branch will be evaluated in the shader, and the branch not used will be discarded.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_branch-_node_autotoc_md2985}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_branch-_node_autotoc_md2985}
