URP 10.\+0.\+x implements the Screen Space Ambient Occlusion (SSAO) effect.

If you intend to use the SSAO effect with your custom shaders, consider the following entities related to SSAO\+:


\begin{DoxyItemize}
\item The {\ttfamily \+\_\+\+SCREEN\+\_\+\+SPACE\+\_\+\+OCCLUSION} keyword.
\item {\ttfamily Input.\+hlsl} contains the new declaration {\ttfamily float2 normalized\+Screen\+Space\+UV} in the {\ttfamily Input\+Data} struct.

To support SSAO in custom shader, add the {\ttfamily Depth\+Normals} Pass and the {\ttfamily \+\_\+\+SCREEN\+\_\+\+SPACE\+\_\+\+OCCLUSION} keyword the the shader. For example, see {\ttfamily Lit.\+shader}.
