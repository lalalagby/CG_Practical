\doxysubsection{Detailed Description}
Normalizes input values in the range \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_normalize_processor_aaa33b42272a4e6ee030ad9387edabea0}{min} and \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_normalize_processor_a4b7d6f0f16a3b116145ea4b603bc1491}{max} to unsigned normalized form \mbox{[}0..1\mbox{]} if \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_normalize_processor_af2430a68328562e7a240ed392de0e181}{zero} is placed at (or below) \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_normalize_processor_aaa33b42272a4e6ee030ad9387edabea0}{min} or to signed normalized form \mbox{[}-\/1..1\mbox{]} if \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_normalize_processor_af2430a68328562e7a240ed392de0e181}{zero} is placed in-\/between \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_normalize_processor_aaa33b42272a4e6ee030ad9387edabea0}{min} and \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_normalize_processor_a4b7d6f0f16a3b116145ea4b603bc1491}{max}. 

This processor is registered (see Input\+System.\+Register\+Processor$<$\+T$>$) under the name "{}normalize"{}.

Note that this processor does not clamp the incoming value to \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_normalize_processor_aaa33b42272a4e6ee030ad9387edabea0}{min} and \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_normalize_processor_a4b7d6f0f16a3b116145ea4b603bc1491}{max}. To achieve this, either add a \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_clamp_processor}{Clamp\+Processor} or use \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_axis_deadzone_processor}{Axis\+Deadzone\+Processor} which combines clamping and normalizing.
