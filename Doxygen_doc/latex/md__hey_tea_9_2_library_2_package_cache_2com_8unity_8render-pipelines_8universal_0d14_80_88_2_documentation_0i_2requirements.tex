Since the release of \doxylink{namespace_unity}{Unity} 2021.\+1, graphics packages are \href{https://docs.unity3d.com/2021.2/Documentation/Manual/pack-core.html}{\texttt{ core Unity packages}}.

For each release of \doxylink{namespace_unity}{Unity} (alpha, beta, patch release), the main \doxylink{namespace_unity}{Unity} installer contains the up-\/to-\/date versions of the following graphics packages\+: SRP \doxylink{namespace_core}{Core}, URP, HDRP, Shader Graph, VFX Graph. Since the release of \doxylink{namespace_unity}{Unity} 2021.\+1, the Package Manager shows only the major revisions of the graphics packages (version 11.\+0.\+0 for all \doxylink{namespace_unity}{Unity} 2021.\+1.\+x releases, version 12.\+0.\+0 for all \doxylink{namespace_unity}{Unity} 2021.\+2.\+x releases).

You can install a different version of a graphics package from disk using the Package Manager, or by modifying the {\ttfamily manifest.\+json} file.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2requirements_autotoc_md2451}{}\doxysection{\texorpdfstring{Render pipeline compatibility}{Render pipeline compatibility}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2requirements_autotoc_md2451}
Projects made using URP are not compatible with the High Definition Render Pipeline (HDRP) or the Built-\/in Render Pipeline. Before you start development, you must decide which render pipeline to use in your Project. For information on choosing a render pipeline, see the \href{https://docs.unity3d.com/2019.3/Documentation/Manual/render-pipelines.html}{\texttt{ Render Pipelines}} section of the \doxylink{namespace_unity}{Unity} Manual.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2requirements_autotoc_md2452}{}\doxysection{\texorpdfstring{Unity Player system requirements}{Unity Player system requirements}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2requirements_autotoc_md2452}
This package does not add any extra platform-\/specific requirements. General system requirements for the \doxylink{namespace_unity}{Unity} \doxylink{class_player}{Player} apply. For more information on \doxylink{namespace_unity}{Unity} system requirements, see \href{https://docs.unity3d.com/Manual/system-requirements.html}{\texttt{ System requirements for Unity}}.
