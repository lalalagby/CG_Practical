\item The Universal Renderer asset contains the property {\bfseries{Rendering Path}} that lets you select the Forward or the Deferred Rendering Path.
\item The method {\ttfamily Clear\+Flag.\+Depth} does not implicitly clear the Stencil buffer anymore. Use the new method {\ttfamily Clear\+Flag.\+Stencil}.
