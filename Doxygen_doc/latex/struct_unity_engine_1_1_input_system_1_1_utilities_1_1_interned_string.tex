\doxysubsection{Detailed Description}
Wraps around a string to allow for faster case-\/insensitive string comparisons while preserving original casing. 

Unlike {\ttfamily string}, Interned\+Strings can be compared with a quick {\ttfamily Object.\+Reference\+Equals} comparison and without actually comparing string contents.

Also, unlike {\ttfamily string}, the representation of an empty and a {\ttfamily null} string is identical.

Note that all string comparisons using Interned\+Strings are both case-\/insensitive and culture-\/insensitive.

There is a non-\/zero cost to creating an \doxylink{struct_unity_engine_1_1_input_system_1_1_utilities_1_1_interned_string}{Interned\+String}. The first time a new unique \doxylink{struct_unity_engine_1_1_input_system_1_1_utilities_1_1_interned_string}{Interned\+String} is encountered, there may also be a GC heap allocation.
