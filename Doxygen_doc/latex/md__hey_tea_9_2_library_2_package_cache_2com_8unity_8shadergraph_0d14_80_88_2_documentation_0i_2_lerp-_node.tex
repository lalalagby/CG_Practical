Returns the result of linearly interpolating between input {\bfseries{A}} and input {\bfseries{B}} by input {\bfseries{T}}.

For example, when the value of input {\bfseries{T}} is 0 the return value is equal to the value of input {\bfseries{A}}, when it is 1 the return value is equal to the value of input {\bfseries{B}} and when it is 0.\+5 the return value is the midpoint of the two inputs {\bfseries{A}} and {\bfseries{B}}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_lerp-_node_autotoc_md3420}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_lerp-_node_autotoc_md3420}
