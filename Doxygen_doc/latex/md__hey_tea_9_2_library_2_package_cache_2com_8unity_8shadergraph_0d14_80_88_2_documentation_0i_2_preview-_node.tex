This node enables you to inspect a preview at a specific point in a Shader Graph. It does not modify any input values.

By default, the Editor automatically selects a preview mode. That decision is determined by both the type of the node you are previewing and other upstream nodes. With \mbox{[}Preview Mode Control\mbox{]}(Preview-\/\+Mode-\/\+Control), you can manually select your preferred preview mode.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_preview-_node_autotoc_md3637}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_preview-_node_autotoc_md3637}
