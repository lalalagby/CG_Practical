 A {\bfseries{manager}} camera oversees many Virtual Cameras but acts as a single Virtual Camera from the point of view of \doxylink{namespace_cinemachine}{Cinemachine} Brain and \doxylink{namespace_timeline}{Timeline}.

\doxylink{namespace_cinemachine}{Cinemachine} includes these manager cameras\+:


\begin{DoxyItemize}
\item \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_free_look}{Free Look Camera}{0}\+: an enhanced \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_d1eecda77a41f7373053e4f2c0a56bf7}{Orbital Transposer}{0}. It manages three horizontal orbits, arranged vertically to surround an avatar.
\item \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_mixing_camera}{Mixing Camera}{0}\+: uses the weighted average of up to eight child Virtual Cameras.
\item \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_blend_list_camera}{Blend List Camera}{0}\+: executes a sequence of blends or cuts of its child Virtual Cameras.
\item \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_clear_shot}{Clear Shot Camera}{0}\+: picks the child Virtual Camera with the best view of the target.
\item \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_state_driven_camera}{State-\/\+Driven Camera}{0}\+: picks a child Virtual Camera in reaction to changes in animation state.
\end{DoxyItemize}

Because manager cameras act like normal Virtual Cameras, you can nest them. In other words, create arbitrarily complex camera rigs that combine regular Virtual Cameras and manager cameras.
