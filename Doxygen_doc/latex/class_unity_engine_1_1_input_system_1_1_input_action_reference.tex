\doxysubsection{Detailed Description}
References a specific \doxylink{class_unity_engine_1_1_input_system_1_1_input_action}{Input\+Action} in an \doxylink{class_unity_engine_1_1_input_system_1_1_input_action_map}{Input\+Action\+Map} stored inside an \doxylink{class_unity_engine_1_1_input_system_1_1_input_action_asset}{Input\+Action\+Asset}. 

The difference to a plain reference directly to an \doxylink{class_unity_engine_1_1_input_system_1_1_input_action}{Input\+Action} object is that an \doxylink{class_unity_engine_1_1_input_system_1_1_input_action_reference}{Input\+Action\+Reference} can be serialized without causing the referenced \doxylink{class_unity_engine_1_1_input_system_1_1_input_action}{Input\+Action} to be serialized as well. The reference will remain intact even if the action or the map that contains the action is renamed.

References can be set up graphically in the editor by dropping individual actions from the project browser onto a reference field.
