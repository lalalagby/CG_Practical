 Use this attribute to define a specific set of platforms you want or do not want your test(s) to run on.

You can use this attribute on the test method, test class, or test assembly level. Use the supported \href{https://docs.unity3d.com/ScriptReference/RuntimePlatform.html}{\texttt{ Runtime\+Platform}} enumeration values to specify the platforms. You can also specify which platforms to test by passing one or more {\ttfamily Runtime\+Platform} values along with or without the include or exclude properties as parameters to the \href{https://github.com/nunit/docs/wiki/Platform-Attribute}{\texttt{ Platform}} attribute constructor.

The test(s) skips if the current target platform is\+:


\begin{DoxyItemize}
\item Not explicitly specified in the included platforms list
\item In the excluded platforms list
\end{DoxyItemize}
