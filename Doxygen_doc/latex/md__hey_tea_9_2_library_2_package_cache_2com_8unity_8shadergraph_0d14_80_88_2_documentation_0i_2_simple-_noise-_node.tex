Generates a simple, or \href{https://en.wikipedia.org/wiki/Value_noise}{\texttt{ Value}}, noise based on input {\bfseries{UV}}. The scale of the generated noise is controlled by input {\bfseries{Scale}}.

You can also choose to use two different hashing methods for calculating the noise. As of \doxylink{namespace_unity}{Unity} version 2021.\+2, the Simple Noise node defaults to the {\bfseries{Deterministic}} hash, to ensure consistent results for noise generation across platforms.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_simple-_noise-_node_autotoc_md3865}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_simple-_noise-_node_autotoc_md3865}
