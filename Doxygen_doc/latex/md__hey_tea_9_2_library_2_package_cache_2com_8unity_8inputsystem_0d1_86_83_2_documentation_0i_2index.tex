 The {\bfseries{Input \doxylink{namespace_system}{System}}} allows your users to control your game or app using a device, touch, or gestures.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8inputsystem_0d1_86_83_2_documentation_0i_2index_autotoc_md1397}{}\doxysection{\texorpdfstring{Introduction}{Introduction}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8inputsystem_0d1_86_83_2_documentation_0i_2index_autotoc_md1397}
\doxylink{namespace_unity}{Unity} supports input through two separate systems, one older, and one newer.

The older system, which is built-\/in to the editor, is called the \href{https://docs.unity3d.com/Manual/class-InputManager.html}{\texttt{ Input Manager}}. The Input Manager is part of the core \doxylink{namespace_unity}{Unity} platform and is the default, if you do not install this Input \doxylink{namespace_system}{System} Package.

This {\bfseries{Input \doxylink{namespace_system}{System} package}} is a newer, more flexible system, which allows you to use any kind of Input Device to control your \doxylink{namespace_unity}{Unity} content. It\textquotesingle{}s intended to be a replacement for \doxylink{namespace_unity}{Unity}\textquotesingle{}s classic Input Manager. It iss referred to as "{}\+The Input System Package"{}, or just \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}"{}\+The Input System"{}\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}. To use it, you must \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8inputsystem_0d1_86_83_2_documentation_0i_2_installation}{install it into your project using the Package Manager}{0}.

During the installation process for the Input \doxylink{namespace_system}{System} package, the installer offers to automatically deactivate the older built-\/in system. (\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8inputsystem_0d1_86_83_2_documentation_0i_2_installation}{Read more}{0})

To get started, see the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8inputsystem_0d1_86_83_2_documentation_0i_2_installation}{Installation}{0} and \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8inputsystem_0d1_86_83_2_documentation_0i_2_workflows}{Workflows}{0} sections.
