\doxylink{namespace_samples}{Samples} a Cubemap and returns a Vector 4 color value for use in the shader. Requires a Direction ({\bfseries{Dir}}) input in world space to sample the Cubemap. You can achieve a blurring effect by using the {\bfseries{LOD}} input to sample at a different Level of Detail. You can also use the {\bfseries{Sampler}} input to define a custom Sampler State.
