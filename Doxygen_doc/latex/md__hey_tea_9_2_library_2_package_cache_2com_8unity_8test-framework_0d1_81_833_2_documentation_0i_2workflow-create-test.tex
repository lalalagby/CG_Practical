\begin{DoxyEnumerate}
\item Create your {\itshape Test} \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_04872722a20000e06b1b28d5347f2329d}{assembly folder}{0} and select it in the {\bfseries{Project}} window.
\item Click the button {\bfseries{Create Test Script in current folder}} option in the {\bfseries{Test Runner}} window. 
\item It creates a {\itshape New\+Test\+Script.\+cs} file in the {\itshape \doxylink{namespace_tests}{Tests}} folder. Change the name of the script, if necessary, and press Enter to accept it.

To create another set of tests\+:


\begin{DoxyEnumerate}
\item In the {\bfseries{Project}} window, select {\bfseries{Assets}}.
\item Create a new test assembly folder (menu\+: {\bfseries{Assets \texorpdfstring{$>$}{>} Create\texorpdfstring{$>$}{>} Testing \texorpdfstring{$>$}{>} \doxylink{namespace_tests}{Tests} Assembly Folder}}).
\item In the {\bfseries{Project}} window, select the new folder.
\item Create a new test script in the folder (menu\+: {\bfseries{Assets \texorpdfstring{$>$}{>} Create \texorpdfstring{$>$}{>} Testing \texorpdfstring{$>$}{>} C\# Test Script}}).
\end{DoxyEnumerate}

The assembly definition is assigned the same name as your new asset. To rename it, change the {\bfseries{Name}} in the {\bfseries{Insepctor}} window. Assembly definition names must be unique.

\begin{quote}
{\bfseries{Note}}\+: Changing the file name of the assembly definition file does not affect the value of the {\bfseries{Name}} property in the file. Use the {\bfseries{Inspector}} window to make sure the name is properly changed. \end{quote}


By default {\bfseries{Any Platform}} is preselected as the target platform for the new assembly, which means the test script appears as a Play\+Mode test in the \doxylink{namespace_test_runner}{Test\+Runner} window. To change it to an Edit\+Mode test, in the {\bfseries{Inspector}} window select {\bfseries{Editor}} only under {\bfseries{Platforms}}.

New assemblies created through the {\bfseries{Assets}} menu should automatically include references to {\ttfamily \doxylink{namespace_unity_engine_1_1_test_runner}{Unity\+Engine.\+Test\+Runner}} and {\ttfamily \doxylink{namespace_unity_editor_1_1_test_runner}{Unity\+Editor.\+Test\+Runner}}.
