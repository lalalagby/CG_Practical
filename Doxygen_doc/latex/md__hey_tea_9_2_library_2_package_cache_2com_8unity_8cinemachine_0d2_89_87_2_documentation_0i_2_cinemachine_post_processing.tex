 Use the \doxylink{namespace_cinemachine}{Cinemachine} Post Processing \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_414e015286be387e4fe85aa25dd01df7}{extension}{0} to attach a Postprocessing V2 profile to a Virtual Camera.

{\bfseries{Note 1}}\+: \doxylink{namespace_unity}{Unity} recommends using Postprocessing V2 instead of Postprocessing V1.

{\bfseries{Note 2}}\+: With HDRP and URP 7 and up, The Post\+Processing package is deprecated, and is implemented natively by HDRP and URP. In that case, please see the {\bfseries{Cinemachine\+Volume\+Settings}} extension.

The \doxylink{namespace_cinemachine}{Cinemachine} Post Processing extension holds a Post-\/\+Processing Profile asset to apply to a Virtual Camera when it is activated. If the camera is blending with another Virtual Camera, then the blend weight is applied to the Post Process effects also.

Before attaching post processing profiles to Virtual Cameras, you first need to set up your project to use post processing.

To set up project to use Post Processing V2 with \doxylink{namespace_cinemachine}{Cinemachine}\+:


\begin{DoxyEnumerate}
\item \href{https://docs.unity3d.com/Packages/com.unity.package-manager-ui@latest/index.html}{\texttt{ Install}} the Postprocessing V2 package.
\item Select your \doxylink{namespace_unity}{Unity} camera with \doxylink{namespace_cinemachine}{Cinemachine} Brain in the \href{https://docs.unity3d.com/Manual/UsingTheSceneView.html}{\texttt{ Scene}} view.
\item \href{https://docs.unity3d.com/Manual/UsingComponents.html}{\texttt{ Add the component}} named {\bfseries{Post-\/\+Process Layer}}. This will enable Post Process profiles to affect the Camera.
\end{DoxyEnumerate}

To add a Post Process profile to a Virtual Camera


\begin{DoxyEnumerate}
\item Select your Virtual Camera in the \href{https://docs.unity3d.com/Manual/UsingTheSceneView.html}{\texttt{ Scene}} view or \href{https://docs.unity3d.com/Manual/Hierarchy.html}{\texttt{ Hierarchy}} window.
\item In the \href{https://docs.unity3d.com/Manual/UsingTheInspector.html}{\texttt{ Inspector}}, choose {\bfseries{Add Extension \texorpdfstring{$>$}{>} Cinemachine\+Post\+Processing}}, then configre the Profile asset to have the effects you want when this virtual camera is live.
