You can use Exposure Weight to determine how exposure affects emission. It is a value between {\bfseries{0}} and {\bfseries{1}} where. A value of {\bfseries{0}} means that exposure does not effect this part of the emission. A value of {\bfseries{1}} means that exposure fully affects this part of the emission.
