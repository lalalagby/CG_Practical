The On Mouse Exit node listens for the user\textquotesingle{}s mouse pointer location to exit the Collider of a specified Game\+Object. When the mouse exits the Collider or GUI element, the node triggers the next node connected to it. It doesn\textquotesingle{}t send or receive any other data.

In the following example, continued from the example from the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_026476a4595e3c6bbd99201d4e87c8469}{On Mouse Enter node}{0}, the On Mouse Exit node triggers a Destroy Game\+Object node when the user\textquotesingle{}s mouse exits the Collider on the Script Machine\textquotesingle{}s Game\+Object. When the user\textquotesingle{}s mouse leaves the Collider, the {\bfseries{Target}} Game\+Object no longer has a spotlight.
