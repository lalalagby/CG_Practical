\doxysubsection{Detailed Description}
A \doxylink{namespace_cinemachine}{Cinemachine} Virtual Camera Body component that constrains camera motion to a \doxylink{class_cinemachine_1_1_cinemachine_path}{Cinemachine\+Path}. The camera can move along the path. 

This behaviour can operate in two modes\+: manual positioning, and Auto-\/\+Dolly positioning. In Manual mode, the camera\textquotesingle{}s position is specified by animating the Path Position field. In Auto-\/\+Dolly mode, the Path Position field is animated automatically every frame by finding the position on the path that\textquotesingle{}s closest to the virtual camera\textquotesingle{}s Follow target.
