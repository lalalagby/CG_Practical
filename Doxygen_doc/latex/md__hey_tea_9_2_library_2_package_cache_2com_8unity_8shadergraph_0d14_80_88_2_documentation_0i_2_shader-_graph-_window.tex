The {\bfseries{Shader Graph Window}} contains the workspace for creating shaders using the {\bfseries{Shader Graph}} system. To open the {\bfseries{Shader Graph Window}} you must first create a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2index}{Shader Graph Asset}{0}. For more information see the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_getting-_started}{Getting Started}{0} section.

The {\bfseries{Shader Graph}} window contains various individual elements such as the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_blackboard}{Blackboard}{0}, \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_internal-_inspector}{Graph Inspector}{0}, and \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_main-_preview}{Main Preview}{0}. These elements can be moved inside the workspace. They will automatically anchor to the nearest corner when scaling the {\bfseries{Shader Graph Window}}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_graph-_window_autotoc_md3849}{}\doxysection{\texorpdfstring{Title Bar}{Title Bar}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_graph-_window_autotoc_md3849}
