Fade Transition is a method of adding noise to add variation while a function transitions from on to off. This node takes in a fade value and remaps it using the noise value (usually from a texture). When Fade\+Value is 0, the output is always 0, and when Fade\+Value is 1, the output is always exactly 1. In between 0 and 1 the transition will follow the pattern in the noise.
