Gets the first instantiated \doxylink{class_unity_engine_1_1_rendering_1_1_volume_profile}{Volume\+Profile} assigned to the \doxylink{class_unity_engine_1_1_rendering_1_1_volume}{Volume}. If another \doxylink{class_unity_engine_1_1_rendering_1_1_volume}{Volume} uses the same Profile, this clones the shared Profile and starts using it from now on. 

This property automatically instantiates the Profile and make it unique to this \doxylink{class_unity_engine_1_1_rendering_1_1_volume}{Volume} so you can safely edit it via scripting at runtime without changing the original Asset in the Project. Note that if you pass your own Profile, you must destroy it when you finish using it.
