\begin{DoxyItemize}
\item Added Timeline\+Playback\+Controls Editor API\+:
\begin{DoxyItemize}
\item The Playback controls API lets you drive the \doxylink{namespace_timeline}{Timeline} Window playback controls from code. From this API, you can
\begin{DoxyItemize}
\item Change the current Time/\+Frame
\item Query the current Time/\+Frame
\item Start/\+Stop playback of the currently shown \doxylink{namespace_timeline}{Timeline}
\item Go to First or Last frame
\item Go to previous or next frame.
\end{DoxyItemize}
\item Use it to accelerate your workflow, or build your own workflows on top of \doxylink{namespace_timeline}{Timeline}.
\end{DoxyItemize}
