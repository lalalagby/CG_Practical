 Before you begin, make sure that your project is set up properly, and the graphs are loading correctly. See \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_getting-_started}{Getting started with Shader Graph}{0} for more information.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3235}{}\doxysection{\texorpdfstring{Create a New Graph}{Create a New Graph}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3235}
Use the Project Browser to create a new \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_graph-_asset}{Shader Graph Asset}{0} in your project. The {\bfseries{Create \texorpdfstring{$>$}{>} Shader Graph}} will display the various creation options.

A {\bfseries{Blank Shader Graph}} will create a Shader Graph with no selected active targets or \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_block-_node}{block nodes}{0}. You will need to select a target via the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_graph-_settings-_tab}{Graph Settings Menu}{0} to continue.

Certain integrations, like Render Pipelines, can also provide pre-\/configured options for Shader Graphs. For this example, a {\bfseries{Universal \texorpdfstring{$>$}{>} Lit}} Shader Graph has been created and opened.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3236}{}\doxysection{\texorpdfstring{Create a new node}{Create a new node}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3236}
Use the {\bfseries{Create Node}} menu to create new nodes. There are two ways to open the menu\+:


\begin{DoxyEnumerate}
\item Right click, and select {\bfseries{Create Node}} from the context menu.
\item Press the spacebar.
\end{DoxyEnumerate}

In the menu, you can type in the search bar to look for specific nodes, or browse all nodes in the library. In this example, we\textquotesingle{}ll create a Color node. First, type "{}color"{} in the {\bfseries{Create Node}} menu\textquotesingle{}s search bar. Then, click {\bfseries{Color}}, or highlight {\bfseries{Color}} and press Enter to create a Color node.

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3237}{}\doxysection{\texorpdfstring{Connect nodes}{Connect nodes}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3237}
To build a graph, you need to connect nodes together. To do so, click the {\bfseries{Output Slot}} of a node, and drag that connection into the {\bfseries{Input Slot}} of another node.

Start by connecting the Color node to the {\bfseries{Base Color}} block of our Fragment Stack.

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3238}{}\doxysection{\texorpdfstring{Change node output}{Change node output}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3238}
Notice that the connection updated the main preview, and the 3D Object in the {\bfseries{Main Preview}} is now black, which is the color specified in the Color node. You can click on the color bar in that node, and use the color picker to change the color. Any changes you make on the node updates the object in the {\bfseries{Main Preview}} in real time.

For example, if you pick red, the 3D Object immediately reflects this change.

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3239}{}\doxysection{\texorpdfstring{Save the graph}{Save the graph}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3239}
Currently, Shader Graphs do not automatically save. There are two ways to save your changes\+:


\begin{DoxyEnumerate}
\item Click the {\bfseries{Save Asset}} button in the top left corner of the window.
\item Close the graph. If \doxylink{namespace_unity}{Unity} detects any unsaved changes, a pop-\/up window appears, and asks if you want to save those changes.
\end{DoxyEnumerate}

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3240}{}\doxysection{\texorpdfstring{Create a Material}{Create a Material}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3240}
After saving your graph, use the shader to create a new Material. The process of \href{https://docs.unity3d.com/Manual/Materials.html}{\texttt{ creating a new Material}} and assigning it a Shader Graph shader is the same as that for regular shaders. In either the main menu or the Project View context menu, select {\bfseries{Assets \texorpdfstring{$>$}{>} Create \texorpdfstring{$>$}{>} Material}}. Select the Material you just created. In its Inspector window, select the {\bfseries{Shader}} drop-\/down menu, click {\bfseries{Shader Graphs}}, and choose the Shader Graph shader you wish to apply to the Material.

You can also right-\/click the Shader Graph shader, and select {\bfseries{Create \texorpdfstring{$>$}{>} Material}}. This method automatically assigns that Shader Graph shader to the newly created Material.



A Material is also automatically generated as a subasset of the Shader Graph. You can assign it directly to an object in your scene. Modifying a property from the Blackboard on the Shader Graph will update this material in real time, which allows for quick visualization in the scene.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3241}{}\doxysection{\texorpdfstring{Put the Material in the Scene}{Put the Material in the Scene}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3241}
Now that you have assigned your shader to a Material, you can apply it to objects in the Scene. Drag and drop the Material onto an object in the Scene. Alternatively, in the object\textquotesingle{}s Inspector window, locate {\bfseries{Mesh Renderer \texorpdfstring{$>$}{>} Materials}}, and apply the Material to the {\bfseries{Element}}.

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3242}{}\doxysection{\texorpdfstring{Use properties to edit the graph}{Use properties to edit the graph}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_first-_shader-_graph_autotoc_md3242}
You can also use properties to alter your shader\textquotesingle{}s appearance. Properties are options that are visible from the Material\textquotesingle{}s Inspector, which lets others change settings in your shader without the need to open the Shader Graph.

To create a new property, use the {\bfseries{Add (+)}} button on the top right corner of the Blackboard, and select the type of property to create. In this example, we\textquotesingle{}ll select {\bfseries{Color}}.



This adds a new property in the Blackboard with the following options in the {\bfseries{Node Settings}} tab of the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_internal-_inspector}{Graph Inspector}{0} when the property is selected.

There are two ways to reference a property in your graph\+:


\begin{DoxyEnumerate}
\item Drag the property from the Blackboard onto the graph.
\item Right-\/click and select {\bfseries{Create Node}}. The property is listed in the {\bfseries{Properties}} category.
\end{DoxyEnumerate}



Try connecting the property to the {\bfseries{Base Color}} block. The object immediately changes to black.



Save your graph, and return to the Material\textquotesingle{}s Inspector. The property now appears in the Inspector. Any changes you make to the property in the Inspector affects all objects that use this Material.
