A {\bfseries{Node}} defines an input, output or operation on the Shader Graph, depending on its available \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_port}{Ports}{0}. A {\bfseries{Node}} may have any number of input and/or output ports. You create a Shader Graph by connecting these ports with \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_edge}{Edges}{0}. A {\bfseries{Node}} might also have any number of {\bfseries{Controls}}, these are controls on the {\bfseries{Node}} that do not have ports.

You can collapse a {\bfseries{Node}} by clicking the {\bfseries{Collapse}} button in the top-\/right corner of the {\bfseries{Node}}. This will hide all unconnected ports.

For components of a {\bfseries{Node}} see\+:

There are many available {\bfseries{Nodes}} in Shader Graph. For a full list of all available {\bfseries{Nodes}} see the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node-_library}{Node Library}{0}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node_autotoc_md3519}{}\doxysection{\texorpdfstring{Preview}{Preview}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node_autotoc_md3519}
Many nodes include a preview. This preview displays the main output value at that stage in the graph. Hide this preview with the Collapse control that displays when you hover over the node. You can also collapse and expand node previews via the Context Menu in the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_graph-_window}{Shader Graph Window}{0}. To configure the appearance of node previews, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_preview-_mode-_control}{Preview Mode Control}{0}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node_autotoc_md3520}{}\doxysection{\texorpdfstring{Context Menu}{Context Menu}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node_autotoc_md3520}
Right clicking on a {\bfseries{Node}} will open a context menu. This menu contains many operations that can be performed on the {\bfseries{Node}}. Note that when multiple nodes are selected, these operations will be applied to the entire selection.
