 This node provides access to Water\textquotesingle{}s surface foam, surface gradient, and deep foam. You can also use this node to dampen the normals for each \href{https://docs.unity3d.com/Packages/com.unity.render-pipelines.high-definition@14.0/manual/WaterSystem-simulation.html\#simulation-bands}{\texttt{ water band}}.

The High Definition Render Pipeline (HDRP) uses this node in the default water shader graph to fetch data from the water simulation. Don\textquotesingle{}t modify the settings of this node.
