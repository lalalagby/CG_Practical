\doxysubsection{Detailed Description}
\doxylink{class_unity_engine_1_1_input_system_1_1_h_i_d_1_1_h_i_d}{HID} descriptor for a \doxylink{class_unity_engine_1_1_input_system_1_1_h_i_d_1_1_h_i_d}{HID} class device. 

This is a processed view of the combined descriptors provided by a \doxylink{class_unity_engine_1_1_input_system_1_1_h_i_d_1_1_h_i_d}{HID} as defined in the \doxylink{class_unity_engine_1_1_input_system_1_1_h_i_d_1_1_h_i_d}{HID} specification, i.\+e. it\textquotesingle{}s a combination of information from the USB device descriptor, \doxylink{class_unity_engine_1_1_input_system_1_1_h_i_d_1_1_h_i_d}{HID} class descriptor, and \doxylink{class_unity_engine_1_1_input_system_1_1_h_i_d_1_1_h_i_d}{HID} report descriptor.
