\doxysubsection{Detailed Description}
Interface that can be implemented by Mono\+Behaviours indicating that they receive time-\/related control calls from a Playable\+Graph. 

Implementing this interface on Mono\+Behaviours attached to Game\+Objects under control by control-\/tracks will cause them to be notified when associated \doxylink{namespace_unity_engine_1_1_timeline}{Timeline} clips are active.
