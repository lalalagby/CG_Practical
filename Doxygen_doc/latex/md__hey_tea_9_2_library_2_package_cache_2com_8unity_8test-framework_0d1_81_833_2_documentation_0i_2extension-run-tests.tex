Run tests by calling {\ttfamily Execute} on the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2reference-test-runner-api}{Test\+Runner\+Api}{0}, and provide some execution settings that consists of a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2reference-filter}{Filter}{0}. The {\ttfamily Filter} specifies what tests to run.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2extension-run-tests_autotoc_md4209}{}\doxysubsection{\texorpdfstring{Example}{Example}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2extension-run-tests_autotoc_md4209}

It is possible to specify a more specific filter by filling out the fields on the {\ttfamily Filter} class in more detail.

Many of the fields allow for multiple values. The runner tries to match tests against at least one of the values provided and then runs any tests that match.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2extension-run-tests_autotoc_md4211}{}\doxysubsection{\texorpdfstring{Example}{Example}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2extension-run-tests_autotoc_md4211}
In this example, the API runs tests with full names that fit either of the two names provided\+:


\begin{DoxyCode}{0}

\DoxyCodeLine{api.Execute(new\ ExecutionSettings(new\ Filter()}

The execution settings take one or more filters in its constructor. If there is no filter provided, then it runs all {\bfseries{Edit Mode}} tests by default. If there are multiple filters provided, then a test runs if it matches any of the filters.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2extension-run-tests_autotoc_md4215}{}\doxysubsection{\texorpdfstring{Example}{Example}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2extension-run-tests_autotoc_md4215}
