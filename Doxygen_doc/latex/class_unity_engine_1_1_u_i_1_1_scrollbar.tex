\doxysubsection{Detailed Description}
A standard scrollbar with a variable sized handle that can be dragged between 0 and 1. 

The slider component is a \doxylink{class_unity_engine_1_1_u_i_1_1_selectable}{Selectable} that controls a handle which follow the current value and is sized according to the size property. The anchors of the handle Rect\+Transforms are driven by the \doxylink{class_unity_engine_1_1_u_i_1_1_scrollbar}{Scrollbar}. The handle can be a direct child of the Game\+Object with the \doxylink{class_unity_engine_1_1_u_i_1_1_scrollbar}{Scrollbar}, or intermediary Rect\+Transforms can be placed in between for additional control. When a change to the scrollbar value occurs, a callback is sent to any registered listeners of on\+Value\+Changed.
