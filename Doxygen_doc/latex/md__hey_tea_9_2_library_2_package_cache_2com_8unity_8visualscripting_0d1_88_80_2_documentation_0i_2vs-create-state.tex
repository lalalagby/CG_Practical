 You can create three types of State nodes in a State Graph\+: Script States, Any States, and Super States. For more information on the types of State nodes, see State Graphs in \href{vs-graph-types.md\#state-graphs}{\texttt{ Graphs}}.

You can also add a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-sticky-notes}{Sticky Note}{0} to add comments to a graph.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-create-state_autotoc_md5294}{}\doxysection{\texorpdfstring{Create a Script State}{Create a Script State}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-create-state_autotoc_md5294}
To create a new blank Script State\+:


\begin{DoxyEnumerate}
\item 

\mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2snippets_2vs-open-state-menu}{open-\/state-\/menu}{0}\mbox{]}


\item 

Select {\bfseries{Create Script State}}. ~\newline
Visual Scripting creates a new Script State node.


\item 

Open the \href{vs-interface-overview.md\#the-graph-inspector}{\texttt{ Graph Inspector}}.


\item 

In the Graph Inspector, choose a source for the Script State node\+:


\begin{DoxyItemize}
\item 

{\bfseries{Embed}}\+: The graph only exists on the Script State node. You can only modify the graph from the node in its parent State Graph.


\item 

{\bfseries{Graph}}\+: The graph exists in a separate file. You can modify the graph outside of its parent State Graph and reuse the graph in other areas of your application.


\end{DoxyItemize}
\item 

If you chose {\bfseries{Graph}}\+:


\begin{DoxyEnumerate}[label=\enumalphalphcnt*]

\item 

Select {\bfseries{New}}.


\item 

Enter a name for the graph file.


\item 

Choose where you want to save the new graph.


\item 

Select {\bfseries{Save}}.


\end{DoxyEnumerate}
\end{DoxyEnumerate}



To create a Script State from an existing Script Graph\+:


\begin{DoxyEnumerate}
\item 

\mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2snippets_2vs-open-state-menu}{open-\/state-\/menu}{0}\mbox{]}


\item 

Select {\bfseries{Create Script State}}. ~\newline
Visual Scripting creates a new Script State node.


\item 

Open the \href{vs-interface-overview.md\#the-graph-inspector}{\texttt{ Graph Inspector}}.


\item 

In the Graph Inspector, set the source for the Script State node to {\bfseries{Graph}}.


\item 

Do one of the following\+:


\begin{DoxyItemize}
\item 

Select the object picker (circle icon) and choose a compatible Script Graph from your project.


\item 

Click and drag a Script Graph file from your Project window and release on the {\bfseries{Graph}} field.


\end{DoxyItemize}
\end{DoxyEnumerate}



\begin{DoxyRemark}{Remarks}
Click and drag the Script Graph from your Project window into the Graph Editor to automatically create a Script State node.
\end{DoxyRemark}
