\doxysubsection{Detailed Description}
A base class to implement to draw custom editors for custom Volume\+Parameter. You must use a \doxylink{class_unity_editor_1_1_rendering_1_1_volume_parameter_drawer_attribute}{Volume\+Parameter\+Drawer\+Attribute} to let the editor know which parameter this drawer is for. 

If you do not provide a custom editor for a Volume\+Parameter, \doxylink{namespace_unity}{Unity} uses the buil-\/in property drawers to draw the property as-\/is.
