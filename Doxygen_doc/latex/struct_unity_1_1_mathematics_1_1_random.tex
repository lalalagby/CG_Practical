\doxysubsection{Detailed Description}
\doxylink{struct_unity_1_1_mathematics_1_1_random}{Random} Number Generator based on xorshift. Designed for minimal state (32bits) to be easily embeddable into components. \doxylink{namespace_core}{Core} functionality is integer multiplication free to improve vectorization on less capable SIMD instruction sets.
