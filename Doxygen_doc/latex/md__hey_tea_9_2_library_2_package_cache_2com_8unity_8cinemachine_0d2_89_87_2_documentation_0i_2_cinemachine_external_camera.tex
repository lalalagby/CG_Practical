 {\bfseries{Note\+:}} This component is {\bfseries{deprecated}} in favour of using a normal Cinemachine\+Virtual\+Camera with Do Nothing in both Aim and Body.

This component will expose a non-\/cinemachine camera to the cinemachine system, allowing it to participate in blends. Just add it as a component alongside an existing \doxylink{namespace_unity}{Unity} Camera component.
