Override this method if you need to to configure render targets and their clear state, and to create temporary render target textures. If a render pass doesn\textquotesingle{}t override this method, this render pass renders to the active Camera\textquotesingle{}s render target. You should never call Command\+Buffer.\+Set\+Render\+Target. Instead call {\ttfamily Configure\+Target} and {\ttfamily Configure\+Clear}.
