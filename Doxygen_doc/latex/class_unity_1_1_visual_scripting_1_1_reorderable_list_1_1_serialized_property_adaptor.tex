\doxysubsection{Detailed Description}
Reorderable list adaptor for serialized array property. 

\doxylink{class_unity_1_1_visual_scripting_1_1_this}{This} adaptor can be subclassed to add special logic to item height calculation. You may want to implement a custom adaptor class where specialised functionality is needed. 

List elements are {\bfseries{not}} cloned using the System.\+ICloneable interface when using a Unity\+Editor.\+Serialized\+Property to manipulate lists.
