 To use the Input \doxylink{namespace_system}{System} package with Visual Scripting, add a \doxylink{class_player}{Player} Input component to the same Game\+Object as the Script Graph and create an Input Actions asset.

You must add the \doxylink{class_player}{Player} Input component and create the Input Actions asset before you create the Script Graph.

\begin{DoxyNote}{Note}
If the Input \doxylink{namespace_system}{System} package isn\textquotesingle{}t installed in your project, follow the Input \doxylink{namespace_system}{System} documentation\textquotesingle{}s \href{https://docs.unity3d.com/Packages/com.unity.inputsystem@latest/index.html?subfolder=/manual/Installation.html}{\texttt{ Installation guide}} to install the package. Go to {\bfseries{Window}} \texorpdfstring{$>$}{>} {\bfseries{Package Manager}} to check your installed packages.
\end{DoxyNote}
To add a \doxylink{class_player}{Player} Input component to a Game\+Object\+:


\begin{DoxyEnumerate}
\item \mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_01b8576ec7a772dec58bafc7460cd57fb}{open-\/hierarchy-\/window}{0}\mbox{]}
\item In the Hierarchy window, select the Game\+Object that you want to move with the Script Graph.
\item \mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0e9d7825430e3771479edc4004f004d3d}{open-\/inspector-\/window}{0}\mbox{]}
\item Select {\bfseries{Add Component}}. The Components menu opens.
\item In the Components menu, do one of the following\+:
\begin{DoxyItemize}
\item Go to {\bfseries{Input}}.
\item In the Search bar, enter {\ttfamily \doxylink{class_player}{Player} Input}.
\end{DoxyItemize}
\item Select the {\bfseries{\doxylink{class_player}{Player} Input}} component to add it to the Game\+Object.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Add an Input Actions asset to the \doxylink{class_player}{Player} Input component. Do one of the following\+:
\begin{DoxyItemize}
\item Create a new Input Actions asset.
\item Use an existing Input Actions asset.
\end{DoxyItemize}
\end{DoxyEnumerate}

 \hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_076b4cc5c553f7a02f130c785b12d612f_autotoc_md5180}{}\doxysection{\texorpdfstring{Create a new Input Actions asset}{Create a new Input Actions asset}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_076b4cc5c553f7a02f130c785b12d612f_autotoc_md5180}

\begin{DoxyEnumerate}
\item Select {\bfseries{Create Actions}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Choose a location in your project to save the Input Actions asset.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select {\bfseries{Save}}.
\end{DoxyEnumerate}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_076b4cc5c553f7a02f130c785b12d612f_autotoc_md5181}{}\doxysection{\texorpdfstring{Use an existing Input Actions asset}{Use an existing Input Actions asset}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_076b4cc5c553f7a02f130c785b12d612f_autotoc_md5181}

\begin{DoxyEnumerate}
\item Do one of the following\+:
\begin{DoxyItemize}
\item Click the {\bfseries{Actions}} field\textquotesingle{}s object picker (circle icon) and in the {\bfseries{Select\+Input\+Action\+Asset}} window, select the asset.
\item Click and drag a file from your Project window and drop it into the {\bfseries{Actions}} field.
\end{DoxyItemize}
