 Use an Output node to control the flow of logic and data from a Script Graph\textquotesingle{}s Subgraph node. An Output node sends data from a Subgraph and makes it available to a parent graph.



For more information on Subgraphs, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0b052f136142546305a9d053d2653f106}{Subgraphs and State Units}{0} and \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-nesting-subgraph-node}{Subgraph node}{0}. The Output node is in the {\bfseries{Nesting}} category in the fuzzy finder.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-nesting-output-node_autotoc_md5416}{}\doxysection{\texorpdfstring{Available inputs}{Available inputs}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-nesting-output-node_autotoc_md5416}
By default, an Output node has no ports.

The Output node can only have input ports. Define the number and specific data type for the input ports with the \href{vs-interface-overview.md\#the-graph-inspector}{\texttt{ Graph Inspector}}. For more information on how to define ports on a Script Graph, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_06f95905303333ea92719076820d2246b}{Add a Trigger or Data port to a Script Graph}{0}.

In the following example, the {\bfseries{Character Move}} Subgraph uses an Input node to receive data from a parent graph. The Subgraph uses three values from the parent graph and the values from two Input Get Axis nodes to create a new Vector 3 value. The graph sends the new Vector 3 value to the Output node and back to the parent Script Graph. The Subgraph reduces the number of nodes in the parent graph.



The parent graph receives the Vector 3 value from the Output node. The parent graph uses that value to set a new position on the current Game\+Object\textquotesingle{}s Transform component. ~\newline
