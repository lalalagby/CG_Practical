The {\bfseries{Shader Graph Asset}} is the new {\bfseries{Asset}} type introduced with the shader graph. You can create a {\bfseries{Shader Graph Asset}} from the \href{https://docs.unity3d.com/Manual/ProjectView.html}{\texttt{ Project Window}} from the {\bfseries{Create}} menu.

For convenience there is a {\bfseries{Create}} menu entry for {\bfseries{Blank Shader Graph}} and \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sub-graph}{Sub-\/graph}{0}. They can be found in the {\bfseries{Shader}} sub-\/menu. Additional options may be provided by render pipelines. These options will create a new Shader Graph with required settings and \mbox{[}Block\mbox{]}() nodes in the \mbox{[}Master Stack\mbox{]}() for the selected shading model.

You can open the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_graph-_window}{Shader Graph Window}{0} by double clicking a {\bfseries{Shader Graph Asset}} or by clicking {\bfseries{Open Shader Editor}} in the \href{https://docs.unity3d.com/Manual/UsingTheInspector.html}{\texttt{ Inspector}} when the {\bfseries{Shader Graph Asset}} is selected.
