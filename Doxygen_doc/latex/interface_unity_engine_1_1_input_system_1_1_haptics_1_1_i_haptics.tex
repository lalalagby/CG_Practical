\doxysubsection{Detailed Description}
Base interface for haptics on input devices. 

To support haptics, an \doxylink{class_unity_engine_1_1_input_system_1_1_input_device}{Input\+Device} has to implement one or more haptics interfaces.

void Unity\+Engine.\+Input\+System.\+Haptics.\+IHaptics.\+Pause\+Haptics (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Pause haptics playback on the device. 

This should preserve current playback settings (such as motor speed levels or effect playback positions) but shut down feedback effects on the device.

If proper resumption of effects is not possible, playback should be stopped and \doxylink{interface_unity_engine_1_1_input_system_1_1_haptics_1_1_i_haptics_a777a78e91c8120cd4c3e81d14a9d564f}{Resume\+Haptics} is allowed to be a no-\/operation.

Note that haptics playback states are not required to survive domain reloads in the editor.
