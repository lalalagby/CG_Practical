\doxysubsection{Detailed Description}
Layout class to arrange child elements in a grid format. 

The \doxylink{class_unity_engine_1_1_u_i_1_1_grid_layout_group}{Grid\+Layout\+Group} component is used to layout child layout elements in a uniform grid where all cells have the same size. The size and the spacing between cells is controlled by the \doxylink{class_unity_engine_1_1_u_i_1_1_grid_layout_group}{Grid\+Layout\+Group} itself. The children have no influence on their sizes.
