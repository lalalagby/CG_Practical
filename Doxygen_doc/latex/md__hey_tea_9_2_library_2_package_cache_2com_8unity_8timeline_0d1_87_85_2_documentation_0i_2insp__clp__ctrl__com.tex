 Use the Inspector window to change the common properties of a Control clip. You can only create a Control clip in a Control track. A Control clip is a special clip that controls a nested \doxylink{namespace_timeline}{Timeline} instance, Particle \doxylink{namespace_system}{System}, Prefab instance, or ITime\+Control Script, depending on how you create the Control clip\+:


\begin{DoxyItemize}
\item If you create the Control clip from a Game\+Object with a Playable Director component associated with a \doxylink{namespace_timeline}{Timeline} Asset, then the Control clip controls a nested \doxylink{namespace_timeline}{Timeline} instance. If the Game\+Object parents other Game\+Objects associated with many \doxylink{namespace_timeline}{Timeline} Assets, then the Control clip controls multiple \doxylink{namespace_timeline}{Timeline} instances.
\item If you create the Control clip from a Game\+Object with a Particle \doxylink{namespace_system}{System} component, then the Control clip controls a Particle \doxylink{namespace_system}{System}.
\item If you create the Control clip from a Game\+Object linked to a Prefab, then the Control clip controls a Prefab instance.
\item If you create the Control clip from a Game\+Object with a script that implements the ITime\+Control interface, then the Control clip controls an ITime\+Control Script.
\end{DoxyItemize}

The common properties of a Control clip include its name and Clip Timing properties. Not all common properties apply to all types of Control clips.

Use the {\bfseries{Clip Timing}} properties to position and change the duration of the Control clip.

Most timing properties are expressed in both seconds (s) and frames (f). When specifying seconds, a {\bfseries{Clip Timing}} property accepts decimal values. When specifying frames, a property only accepts integer values. For example, if you attempt to enter 12.\+5 in a frames (f) field, the Inspector window sets the value to 12 frames.

Depending on the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2clp__about}{selected Clip Edit mode}{0}, changing the {\bfseries{Start}}, {\bfseries{End}} or {\bfseries{Duration}} of a Control clip may create an insert or replace clips on the same track. You cannot create a blend between Control clips.
