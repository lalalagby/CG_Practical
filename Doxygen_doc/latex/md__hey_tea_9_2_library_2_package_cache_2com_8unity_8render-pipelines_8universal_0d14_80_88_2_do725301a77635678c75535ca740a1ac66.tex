URP uses its own \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do9e1490445e45fd9eef56dbe3e8613a42}{integrated post-\/processing solution}{0}. If you have the Post Processing Version 2 package installed in your Project, you need to delete the Post Processing Version 2 package before you install URP into your Project. When you have installed URP, you can then recreate your post-\/processing effects.

URP does not currently support custom post-\/processing effects. If your Project uses custom post-\/processing effects, these cannot currently be recreated in URP. Custom post-\/processing effects will be supported in a forthcoming release of URP.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do725301a77635678c75535ca740a1ac66_autotoc_md2172}{}\doxysection{\texorpdfstring{Installing URP}{Installing URP}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do725301a77635678c75535ca740a1ac66_autotoc_md2172}

\begin{DoxyEnumerate}
\item In \doxylink{namespace_unity}{Unity}, open your Project.
\item In the top navigation bar, select {\bfseries{Window \texorpdfstring{$>$}{>} Package Manager}} to open the {\bfseries{Package Manager}} window.
\item In the {\bfseries{\doxylink{namespace_packages}{Packages}}} menu, select {\bfseries{\doxylink{namespace_unity}{Unity} Registry}}. This shows the list of available packages for the version of \doxylink{namespace_unity}{Unity} that you are currently running.
\item Select {\bfseries{Universal RP}} from the list of packages.
\item In the bottom right corner of the Package Manager window, select {\bfseries{Install}}. \doxylink{namespace_unity}{Unity} installs URP directly into your Project.

Before you can start using URP, you need to configure it. To do this, you need to create a Scriptable Render Pipeline Asset and adjust your \doxylink{namespace_graphics}{Graphics} settings.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do725301a77635678c75535ca740a1ac66_autotoc_md2174}{}\doxysubsection{\texorpdfstring{Creating the Universal Render Pipeline Asset}{Creating the Universal Render Pipeline Asset}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do725301a77635678c75535ca740a1ac66_autotoc_md2174}
The \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do6a4fd81cefe986341b093c28ca9c1f30}{Universal Render Pipeline Asset}{0} (URP Asset) contains the global rendering and quality settings of your project, and creates the rendering pipeline instance. The rendering pipeline instance contains intermediate resources and the render pipeline implementation.

To create a Universal Render Pipeline Asset\+:


\begin{DoxyEnumerate}
\item In the Editor, go to the Project window.
\item Right-\/click in the Project window, and select {\bfseries{Create \texorpdfstring{$>$}{>} Rendering \texorpdfstring{$>$}{>} URP Asset}}. Alternatively, navigate to the menu bar at the top, and select {\bfseries{Assets \texorpdfstring{$>$}{>} Create \texorpdfstring{$>$}{>} Rendering \texorpdfstring{$>$}{>} URP Asset}}.
\end{DoxyEnumerate}

You can either leave the default name for the new Universal Render Pipeline Asset, or type a new one.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do725301a77635678c75535ca740a1ac66_autotoc_md2175}{}\doxysubsection{\texorpdfstring{\label{_set-urp-active}%
