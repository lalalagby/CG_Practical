A semicolon-\/separated list of test categories to include in the run. A semi-\/colon separated list should be formatted as a string enclosed in quotation marks, e.\+g. {\ttfamily test\+Category "{}first\+Category;second\+Category"{}}. A semicolon-\/separated list of test names to run, or a regular expression pattern to match tests by their full name. A semi-\/colon separated list should be formatted as a string enclosed in quotation marks, e.\+g. {\ttfamily test\+Filter "{}\+Low;\+Medium"{}}. This argument supports negation using \textquotesingle{}!\textquotesingle{}. If using the test filter \textquotesingle{}!\+My\+Namespace.Something.\+My\+Test\textquotesingle{}, then all tests except that test will be run.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2reference-command-line_autotoc_md4256}{}\doxysubsection{\texorpdfstring{test\+Platform}{test\+Platform}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2reference-command-line_autotoc_md4256}

\begin{DoxyItemize}
\item {\bfseries{Edit\+Mode}}

\end{DoxyItemize}
\item {\bfseries{Play\+Mode}}
\begin{DoxyItemize}
\item Play Mode tests that run in the Editor. Equivalent to running tests from the Play\+Mode tab of the Test Runner window.
\end{DoxyItemize}
\item Any value from the \href{https://docs.unity3d.com/ScriptReference/BuildTarget.html}{\texttt{ Build\+Target}} enum.
\begin{DoxyItemize}
\item Play Mode tests that run on a player built for the specified platform. Equivalent to using the \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}\+Run all tests ({\ttfamily \texorpdfstring{$<$}{<}target\+\_\+platform\texorpdfstring{$>$}{>}})\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} dropdown in the Play\+Mode tab of the Test Runner window.
