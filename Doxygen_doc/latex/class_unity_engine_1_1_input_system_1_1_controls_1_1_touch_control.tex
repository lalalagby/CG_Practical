\doxysubsection{Detailed Description}
A control representing a touch contact. 

Note that unlike most other control types, {\ttfamily Touch\+Controls} do not have a flexible memory layout. They are hardwired to Touch\+State and will not work correctly with a different memory layouts. Additional fields may be appended to the struct but what\textquotesingle{}s there in the struct has to be located at exactly those memory addresses.
