\doxysubsection{Detailed Description}
Scope for property that handle\+: 


\begin{DoxyItemize}
\item Layout decorator (Space, Header)
\item Naming decorator (Tooltips, Inspector\+Name)
\item Overridable checkbox if parameter Is\+Auto\+Property
\item disabled GUI if Overridable checkbox (case above) is unchecked
\item additional property scope This is automatically used inside Property\+Field method 
\end{DoxyItemize}
