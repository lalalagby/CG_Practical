\href{xref:Unity.Burst.Intrinsics.Common.Pause}{\texttt{ {\ttfamily Unity.\+Burst.\+Intrinsics.\+Common.\+Pause}}} is an intrinsic that requests that the CPU pause the current thread. It maps to {\ttfamily pause} on x86, and {\ttfamily yield} on ARM.

Use it to stop spin locks over contending on an atomic access, which reduces contention and power on that section of code.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2csharp-burst-intrinsics-common_autotoc_md316}{}\doxysection{\texorpdfstring{Prefetch}{Prefetch}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2csharp-burst-intrinsics-common_autotoc_md316}
The {\ttfamily Unity.\+Burst.\+Intrinsics.\+Common.\+Prefetch} is an experimental intrinsic that provides a hint that \doxylink{namespace_burst}{Burst} should prefetch the memory location into the cache.

Because the intrinsic is experimental, you must use the {\ttfamily UNITY\+\_\+\+BURST\+\_\+\+EXPERIMENTAL\+\_\+\+PREFETCH\+\_\+\+INTRINSIC} preprocessor define to get access to it.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2csharp-burst-intrinsics-common_autotoc_md317}{}\doxysection{\texorpdfstring{umul128}{umul128}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2csharp-burst-intrinsics-common_autotoc_md317}
Use the \href{xref:Unity.Burst.Intrinsics.Common.umul128*}{\texttt{ {\ttfamily Unity.\+Burst.\+Intrinsics.\+Common.\+umul128}}} intrinsic to access 128-\/bit unsigned multiplication. The {\ttfamily Unity.\+Burst.\+Intrinsics.\+Common.\+Interlocked\+And} and {\ttfamily Unity.\+Burst.\+Intrinsics.\+Common.\+Interlocked\+Or} are experimental intrinsics that provides atomic and/or operations on {\ttfamily int}, {\ttfamily uint}, {\ttfamily long}, and {\ttfamily ulong} types.

Because these intrinsics are experimental, you must use the {\ttfamily UNITY\+\_\+\+BURST\+\_\+\+EXPERIMENTAL\+\_\+\+ATOMIC\+\_\+\+INTRINSICS} preprocessor define to get access to them.
