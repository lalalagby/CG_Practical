{\itshape \doxylink{namespace_unity}{Unity}\textquotesingle{}s \doxylink{namespace_timeline}{Timeline}}

Use \doxylink{namespace_unity}{Unity}\textquotesingle{}s \doxylink{namespace_timeline}{Timeline} to create cinematic content, game-\/play sequences, audio sequences, and complex particle effects.

Each cut-\/scene, cinematic, or game-\/play sequence that you create with \doxylink{namespace_unity}{Unity}\textquotesingle{}s \doxylink{namespace_timeline}{Timeline} consists of a \doxylink{namespace_timeline}{Timeline} Asset and a \doxylink{namespace_timeline}{Timeline} instance. The \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2tl__window}{Timeline window}{0} creates and modifies \doxylink{namespace_timeline}{Timeline} Assets and \doxylink{namespace_timeline}{Timeline} instances simultaneously.

The \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2tl__about}{Timeline Overview section}{0} includes details on the relationship between the \doxylink{namespace_timeline}{Timeline} window, \doxylink{namespace_timeline}{Timeline} Assets, and \doxylink{namespace_timeline}{Timeline} instances.

The \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2wf__about}{Using Timeline section}{0} shows how to create \doxylink{namespace_timeline}{Timeline} Assets and \doxylink{namespace_timeline}{Timeline} instances, record basic animation, animate humanoids, and use other \doxylink{namespace_timeline}{Timeline} features.

The \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2smpl__about}{Samples section}{0} includes a description of the samples offered by the \doxylink{namespace_timeline}{Timeline} package.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2index_autotoc_md4646}{}\doxysection{\texorpdfstring{Installing Timeline}{Installing Timeline}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2index_autotoc_md4646}
