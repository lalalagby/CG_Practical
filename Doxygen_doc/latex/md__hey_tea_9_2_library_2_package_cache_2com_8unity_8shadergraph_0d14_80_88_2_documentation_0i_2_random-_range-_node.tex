Returns a pseudo-\/random number value based on input {\bfseries{Seed}} that is between the minimum and maximum values defined by inputs {\bfseries{Min}} and {\bfseries{Max}} respectively.

Whilst the same value in input {\bfseries{Seed}} will always result in the same output value, the output value itself will appear random. Input {\bfseries{Seed}} is a {\bfseries{Vector 2}} value for the convenience of generating a random number based on a UV input, however for most cases a {\bfseries{Float}} input will suffice.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_random-_range-_node_autotoc_md3675}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_random-_range-_node_autotoc_md3675}
