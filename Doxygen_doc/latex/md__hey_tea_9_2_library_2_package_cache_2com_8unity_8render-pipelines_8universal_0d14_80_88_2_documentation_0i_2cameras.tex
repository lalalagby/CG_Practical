 A Camera in \doxylink{namespace_unity}{Unity} works like a camera in the real world\+: it captures a view of objects in 3-\/dimensional space and flattens that view to display it on a 2-\/dimensional surface.

Cameras in the Universal Render Pipeline (URP) are based on \doxylink{namespace_unity}{Unity}\textquotesingle{}s standard Camera functionality, but with some significant differences. The most notable ways in which Cameras in URP differ from standard \doxylink{namespace_unity}{Unity} Cameras are\+:


\begin{DoxyItemize}
\item The \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do1a250abe3402902ce70ae3f5ae00577b}{Universal Additional Camera Data}{0} component, which extends the Camera component\textquotesingle{}s functionality and allows URP to store additional Camera-\/related data
\item The \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do281102a196ad8cdcf90806a18445bf3f}{Render Type}{0} setting, which defines the two types of Camera in URP\+: Base and Overlay
\item The \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2camera-stacking}{Camera Stacking}{0} system, which allows you to layer the output of multiple Cameras into a single combined output
\item The \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2_volumes}{Volume}{0} system, which allows you to apply \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do9e1490445e45fd9eef56dbe3e8613a42}{post-\/processing effects}{0} to a Camera based on a given Transform\textquotesingle{}s position within your Scene
\item The \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_doc7c35c176c1ad0ea77080cb33c453ec9}{Camera component}{0}, which exposes URP-\/specific properties in the Inspector
\end{DoxyItemize}

For a general introduction to how Cameras work in \doxylink{namespace_unity}{Unity}, and examples of common Camera workflows, see \href{https://docs.unity3d.com/Manual/CamerasOverview.html}{\texttt{ the Unity Manual section on Cameras}}.
