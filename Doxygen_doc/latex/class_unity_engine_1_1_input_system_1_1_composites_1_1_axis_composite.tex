\doxysubsection{Detailed Description}
A single axis value computed from one axis that pulls in the \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_a46c9c4ee4d8ac87a156e551d222edbf2}{negative} direction (\doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_a2765c0f4656dafa515fb6d9c24d6b067}{min\+Value}) and one axis that pulls in the \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_aab856239378e494f61ea6fe85a5974cc}{positive} direction (\doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_ab0858ba01c486eada9784a2ff27d9cf5}{max\+Value}). 

The limits of the axis are determined by \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_a2765c0f4656dafa515fb6d9c24d6b067}{min\+Value} and \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_ab0858ba01c486eada9784a2ff27d9cf5}{max\+Value}. By default, they are set to {\ttfamily \mbox{[}-\/1..1\mbox{]}}. The values can be set as parameters.

If both axes are actuated at the same time, the behavior depends on \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_a3674788391b30c5ccf1f1dbb94a3c608}{which\+Side\+Wins}. By default, neither side will win (\doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_ad00f29814bd35c27ef2bcf7e088eac78a72e608760a087621cc6f67517b2de8eb}{Which\+Side\+Wins.\+Neither}) and the result will be 0 (or, more precisely, the midpoint between \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_a2765c0f4656dafa515fb6d9c24d6b067}{min\+Value} and \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_ab0858ba01c486eada9784a2ff27d9cf5}{max\+Value}). This can be customized to make the positive side win (\doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_ad00f29814bd35c27ef2bcf7e088eac78a3289297424e01eda5b788c083bbf3147}{Which\+Side\+Wins.\+Positive}) or the negative one (\doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_ad00f29814bd35c27ef2bcf7e088eac78affb9356ff2b7da85c75c92fa7ea03b8b}{Which\+Side\+Wins.\+Negative}).

This is useful, for example, in a driving game where break should cancel out accelerate. By binding \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_a46c9c4ee4d8ac87a156e551d222edbf2}{negative} to the break control(s) and \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_aab856239378e494f61ea6fe85a5974cc}{positive} to the acceleration control(s), and setting \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_a3674788391b30c5ccf1f1dbb94a3c608}{which\+Side\+Wins} to \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_ad00f29814bd35c27ef2bcf7e088eac78affb9356ff2b7da85c75c92fa7ea03b8b}{Which\+Side\+Wins.\+Negative}, if the break button is pressed, it will always cause the acceleration button to be ignored.

The actual {\itshape absolute} values of \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_a46c9c4ee4d8ac87a156e551d222edbf2}{negative} and \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_aab856239378e494f61ea6fe85a5974cc}{positive} are used to scale \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_a2765c0f4656dafa515fb6d9c24d6b067}{min\+Value} and \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_ab0858ba01c486eada9784a2ff27d9cf5}{max\+Value} respectively. So if, for example, \doxylink{class_unity_engine_1_1_input_system_1_1_composites_1_1_axis_composite_aab856239378e494f61ea6fe85a5974cc}{positive} is bound to \doxylink{class_unity_engine_1_1_input_system_1_1_gamepad_a51aa5a7eb8d09580ad0287e1655dd1de}{Gamepad.\+right\+Trigger} and the trigger is at a value of 0.\+5, then the resulting value is {\ttfamily max\+Value \texorpdfstring{$\ast$}{*} 0.\+5} (the actual formula is {\ttfamily mid\+Point + (max\+Value -\/ mid\+Point) \texorpdfstring{$\ast$}{*} positive}).
