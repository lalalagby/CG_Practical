NOTE\+: In versions prior to 10.\+0, Shader Graph assumed that HDR colors from the Color Node were in gamma space. Version 10.\+0 corrected this behavior, and Shader Graph now interprets HDR colors in linear space. HDR Color nodes that you created with older versions maintain the old behavior, but you can use the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_internal-_inspector}{Graph Inspector}{0} to upgrade them.
