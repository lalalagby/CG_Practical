Use the {\bfseries{World}} and {\bfseries{Absolute World}} space options to transform the coordinate space of \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_position-_node}{position}{0} values. The {\bfseries{World}} space option uses the Scriptable Render Pipeline default world space to convert position values. The {\bfseries{Absolute World}} space option uses absolute world space to convert position values in all Scriptable Render Pipelines.

If you use the {\bfseries{Transform Node}} to convert coordinate spaces that are not for position values, \doxylink{namespace_unity}{Unity} recommends that you use the {\bfseries{World}} space option. Using {\bfseries{Absolute World}} on values that do not represent position might result in unexpected behavior.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_transform-_node_autotoc_md4053}{}\doxysection{\texorpdfstring{Generated Code Example}{Generated Code Example}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_transform-_node_autotoc_md4053}
