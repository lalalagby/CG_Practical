 You can use the empty graph creation flow to create a new unassigned graph for use in your project.



\begin{DoxyNote}{Note}
To use the graph file, you must \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-attach-graph-machine}{attach it to a Script Machine or State Machine}{0}.
\end{DoxyNote}
For more information on other ways to create a graph file, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-create-graph}{Create a new graph file}{0}.

To create a new unassigned graph\+:


\begin{DoxyEnumerate}
\item \mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0cb8e1ada6ce98d1944fded7f2fbf6b76}{visual-\/scripting-\/window}{0}\mbox{]}
\item In the new Visual Scripting window, select one of the following options\+:
\begin{DoxyItemize}
\item To create a new Script Graph, select {\bfseries{Create new Script Graph}}.
\item To create a new State Graph, select {\bfseries{Create new State Graph}}.
\end{DoxyItemize}
\item Choose a location to save the new graph file.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Enter a name for the graph.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select {\bfseries{Save}}.

The new graph file automatically opens in a new window.
\end{DoxyEnumerate}

The new graph file should look similar to the following image\+:

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0b7d98a7eb0727f6727404486aa5159de_autotoc_md5276}{}\doxysection{\texorpdfstring{Next steps}{Next steps}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0b7d98a7eb0727f6727404486aa5159de_autotoc_md5276}
After you create a new graph, attach it to a Script Machine or State Machine to use it in your application. For more information, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-attach-graph-machine}{Attach a graph file to a Script Machine or State Machine}{0}.
