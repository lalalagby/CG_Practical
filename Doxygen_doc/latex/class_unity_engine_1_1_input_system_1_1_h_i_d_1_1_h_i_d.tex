\doxysubsection{Detailed Description}
A generic \doxylink{class_unity_engine_1_1_input_system_1_1_h_i_d_1_1_h_i_d}{HID} input device. 

This class represents a best effort to mirror the control setup of a \doxylink{class_unity_engine_1_1_input_system_1_1_h_i_d_1_1_h_i_d}{HID} discovered in the system. It is used only as a fallback where we cannot match the device to a specific product we know of. Wherever possible we construct more specific device representations such as \doxylink{class_unity_engine_1_1_input_system_1_1_gamepad}{Gamepad}.
