\doxysubsection{Detailed Description}
Class {\ttfamily \doxylink{class_unity_engine_1_1_rendering_1_1_universal_1_1_scriptable_renderer}{Scriptable\+Renderer}} implements a rendering strategy. It describes how culling and lighting works and the effects supported. 

A renderer can be used for all cameras or be overridden on a per-\/camera basis. It will implement light culling and setup and describe a list of {\ttfamily \doxylink{class_unity_engine_1_1_rendering_1_1_universal_1_1_scriptable_render_pass}{Scriptable\+Render\+Pass}} to execute in a frame. The renderer can be extended to support more effect with additional {\ttfamily \doxylink{class_unity_engine_1_1_rendering_1_1_universal_1_1_scriptable_renderer_feature}{Scriptable\+Renderer\+Feature}}. Resources for the renderer are serialized in {\ttfamily \doxylink{class_unity_engine_1_1_rendering_1_1_universal_1_1_scriptable_renderer_data}{Scriptable\+Renderer\+Data}}.

The renderer resources are serialized in {\ttfamily \doxylink{class_unity_engine_1_1_rendering_1_1_universal_1_1_scriptable_renderer_data}{Scriptable\+Renderer\+Data}}. \begin{DoxySeeAlso}{See also}
