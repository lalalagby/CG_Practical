\doxysubsection{Detailed Description}
Write accessor to a binding on either an \doxylink{class_unity_engine_1_1_input_system_1_1_input_action}{Input\+Action} or an \doxylink{class_unity_engine_1_1_input_system_1_1_input_action_map}{Input\+Action\+Map}. 

Both \doxylink{class_unity_engine_1_1_input_system_1_1_input_action_a44fdaf3ff9ffc22609a9df942372d9bb}{Input\+Action.\+bindings} and \doxylink{class_unity_engine_1_1_input_system_1_1_input_action_map_ae98277c3697a798a28f8b16e187217ff}{Input\+Action\+Map.\+bindings} are read-\/only. To modify bindings (other than setting overrides which you can do through Input\+Action\+Rebinding\+Extensions.\+Apply\+Binding\+Override(\+Input\+Action,int,\+Input\+Binding)), it is necessary to gain indirect write access through this structure.
