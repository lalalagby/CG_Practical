\doxysubsection{Detailed Description}
A 2D rectangular mask that allows for clipping / masking of areas outside the mask. 

The \doxylink{class_unity_engine_1_1_u_i_1_1_rect_mask2_d}{Rect\+Mask2D} behaves in a similar way to a standard \doxylink{class_unity_engine_1_1_u_i_1_1_mask}{Mask} component. It differs though in some of the restrictions that it has. A \doxylink{class_unity_engine_1_1_u_i_1_1_rect_mask2_d}{Rect\+Mask2D}\+: \texorpdfstring{$\ast$}{*}\+Only works in the 2D plane \texorpdfstring{$\ast$}{*}\+Requires elements on the mask to be coplanar. \texorpdfstring{$\ast$}{*}\+Does not require stencil buffer / extra draw calls \texorpdfstring{$\ast$}{*}\+Requires fewer draw calls \texorpdfstring{$\ast$}{*}\+Culls elements that are outside the mask area.
