 Use Noise properties in a Virtual Camera to simulate camera shake. \doxylink{namespace_cinemachine}{Cinemachine} includes a {\bfseries{Basic Multi Channel Perlin}} component, which adds Perlin noise to the movement of the Virtual Camera. {\bfseries{Perlin noise}} is a technique to compute random movement with a natural behavior.



The Basic Multi Channel Perlin component applies a noise profile. A noise profile is an Asset that defines the behavior of noise over time. \doxylink{namespace_cinemachine}{Cinemachine} includes a few noise profile assets. You can \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_noise_profiles}{edit these and create your own}{0}.

To apply noise\+:


\begin{DoxyEnumerate}
\item Select your Virtual Camera in the \href{https://docs.unity3d.com/Manual/UsingTheSceneView.html}{\texttt{ Scene}} view or \href{https://docs.unity3d.com/Manual/Hierarchy.html}{\texttt{ Hierarchy}} window.
\item In the \href{https://docs.unity3d.com/Manual/UsingTheInspector.html}{\texttt{ Inspector}}, use the {\bfseries{Noise}} drop-\/down menu to choose {\bfseries{Basic Multi Channel Perlin}}.
\item In {\bfseries{Noise Profile}}, choose an existing profile asset or \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_noise_profiles}{create your own profile}{0}.
\item Use {\bfseries{Amplitude Gain}} and {\bfseries{Frequency Gain}} to fine-\/tune the noise.
