\doxysubsection{Detailed Description}
The \doxylink{class_unity_engine_1_1_input_system_1_1_x_r_1_1_tracked_pose_driver}{Tracked\+Pose\+Driver} component applies the current pose value of a tracked device to the Transform of the Game\+Object. \doxylink{class_unity_engine_1_1_input_system_1_1_x_r_1_1_tracked_pose_driver}{Tracked\+Pose\+Driver} can track multiple types of devices including \doxylink{namespace_unity_engine_1_1_input_system_1_1_x_r}{XR} HMDs, controllers, and remotes. 

For \doxylink{class_unity_engine_1_1_input_system_1_1_x_r_1_1_tracked_pose_driver_a92fd01f0f82d15837f1aa9ba9c342f50}{position\+Input} and \doxylink{class_unity_engine_1_1_input_system_1_1_x_r_1_1_tracked_pose_driver_a15db945bff5286b3c829df7b7afa1e68}{rotation\+Input}, if an action is directly defined in the \doxylink{struct_unity_engine_1_1_input_system_1_1_input_action_property}{Input\+Action\+Property}, as opposed to a reference to an action externally defined in an \doxylink{class_unity_engine_1_1_input_system_1_1_input_action_asset}{Input\+Action\+Asset}, the action will automatically be enabled and disabled by this behavior during \doxylink{class_unity_engine_1_1_input_system_1_1_x_r_1_1_tracked_pose_driver_a17956ee28844e9545864a94228b3e360}{On\+Enable} and \doxylink{class_unity_engine_1_1_input_system_1_1_x_r_1_1_tracked_pose_driver_a80da1f4f2354ca9c2d84ab5b619b1f2b}{On\+Disable}. The enabled state for actions externally defined must be managed externally from this behavior.
