 The \doxylink{namespace_unity}{Unity} Test Framework (UTF) enables \doxylink{namespace_unity}{Unity} users to test their code in both {\bfseries{Edit Mode}} and {\bfseries{Play Mode}}, and also on target platforms such as \href{https://docs.unity3d.com/Manual/Standalone.html}{\texttt{ Standalone}}, Android, i\+OS, etc.

This package provides a standard test framework for users of \doxylink{namespace_unity}{Unity} and developers at \doxylink{namespace_unity}{Unity} so that both benefit from the same features and can write tests the same way.

UTF uses a \doxylink{namespace_unity}{Unity} integration of NUnit library, which is an open-\/source unit testing library for .Net languages. UTF currently uses NUnit version 3.\+5. For more information about NUnit, see the \href{http://www.nunit.org/}{\texttt{ official NUnit website}} and the \href{https://docs.nunit.org/}{\texttt{ NUnit documentation}}.

\begin{quote}
{\bfseries{Note}}\+: UTF is not a new concept or toolset; it is an adjusted and more descriptive naming for the toolset otherwise known as \doxylink{namespace_unity}{Unity} Test Runner, which is now available as this package. \end{quote}

This version of the \doxylink{namespace_unity}{Unity} Test Framework is compatible with the following versions of the \doxylink{namespace_unity}{Unity} Editor\+:


\begin{DoxyItemize}
\item 2019.\+2 and later.
\end{DoxyItemize}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2index_autotoc_md4222}{}\doxysubsection{\texorpdfstring{Known limitations}{Known limitations}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0i_2index_autotoc_md4222}
\doxylink{namespace_unity}{Unity} Test Framework version 1.\+0.\+18 includes the following known limitations\+:


\begin{DoxyItemize}
\item The {\ttfamily Unity\+Test} attribute does not support WSA platform.
\item The {\ttfamily Unity\+Test} attribute does not support \href{https://github.com/nunit/docs/wiki/Parameterized-Tests}{\texttt{ Parameterized tests}} (except for {\ttfamily Value\+Source}).
\item The {\ttfamily Unity\+Test} attribute does not support the {\ttfamily NUnit} \href{https://github.com/nunit/docs/wiki/Repeat-Attribute}{\texttt{ Repeat}} attribute.
\item Nested test fixture cannot run from the Editor UI.
\item When using the {\ttfamily NUnit} \href{https://github.com/nunit/docs/wiki/Retry-Attribute}{\texttt{ Retry}} attribute in Play\+Mode tests, it throws {\ttfamily Invalid\+Cast\+Exception}.
\item Async tests are not supported in the current version of UTF.
