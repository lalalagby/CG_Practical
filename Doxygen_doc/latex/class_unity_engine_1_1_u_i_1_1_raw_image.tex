\doxysubsection{Detailed Description}
Displays a Texture2D for the \doxylink{namespace_unity_engine_1_1_u_i}{UI} \doxylink{namespace_system}{System}. 

If you don\textquotesingle{}t have or don\textquotesingle{}t wish to create an atlas, you can simply use this script to draw a texture. Keep in mind though that this will create an extra draw call with each \doxylink{class_unity_engine_1_1_u_i_1_1_raw_image}{Raw\+Image} present, so it\textquotesingle{}s best to use it only for backgrounds or temporary visible graphics.
