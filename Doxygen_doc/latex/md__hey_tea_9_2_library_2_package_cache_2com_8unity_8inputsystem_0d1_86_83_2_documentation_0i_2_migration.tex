 This guide provides a list of APIs in {\ttfamily Unity\+Engine.\+Input} (and other related APIs in {\ttfamily \doxylink{namespace_unity_engine}{Unity\+Engine}}) and their corresponding APIs in the new Input \doxylink{namespace_system}{System}. Not all APIs have a corresponding version in the new API yet.

\texorpdfstring{$>$}{>}{\bfseries{Note}}\+: All of the new APIs are in the {\ttfamily \doxylink{namespace_unity_engine_1_1_input_system}{Unity\+Engine.\+Input\+System}} namespace. The namespace is omitted here for brevity. {\ttfamily \doxylink{namespace_unity_engine_1_1_input_system}{Unity\+Engine.\+Input\+System}} is referenced in full for easy disambiguation.

Note that it is possible to have code for the old and the new input system at the same time using conditional compilation. When the new input system is enabled in the player preferences (see \href{./Installation.md\#enabling-the-new-input-backends}{\texttt{ here}}), the {\ttfamily ENABLE\+\_\+\+INPUT\+\_\+\+SYSTEM} preprocessor directive is available.
