{\bfseries{Split Toning}} uses the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2_volumes}{Volume}{0} framework, so to enable and modify {\bfseries{Split Toning}} properties, you must add a {\bfseries{Split Toning}} override to a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2_volumes}{Volume}{0} in your Scene. To add {\bfseries{Split Toning}} to a Volume\+:


\begin{DoxyEnumerate}
\item In the Scene or Hierarchy view, select a Game\+Object that contains a Volume component to view it in the Inspector.
\item In the Inspector, navigate to {\bfseries{Add Override \texorpdfstring{$>$}{>} Post-\/processing}} and click on {\bfseries{Split Toning}}. URP now applies {\bfseries{Split Toning}} to any Camera this Volume affects.
\end{DoxyEnumerate}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do23575e1dd25d0b8b7028b86b30bd672d_autotoc_md2319}{}\doxysection{\texorpdfstring{Properties}{Properties}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do23575e1dd25d0b8b7028b86b30bd672d_autotoc_md2319}
When you adjust the color in the color picker for each property, you should only adjust the {\bfseries{Hue}} and {\bfseries{Saturation}}. {\bfseries{Value}} also changes the overall image brightness.
