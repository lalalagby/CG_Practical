Closed platform packages are not available in the package registry or the Package Manager.

To install a Closed platform package\+:


\begin{DoxyEnumerate}
\item Download the package from the relevant platform developer forum.
\item Use the Package Manager to install the package locally. For information on how to install packages locally, see \href{https://docs.unity3d.com/Manual/upm-ui-local.html}{\texttt{ Installing a local package}}. 
\end{DoxyEnumerate}
