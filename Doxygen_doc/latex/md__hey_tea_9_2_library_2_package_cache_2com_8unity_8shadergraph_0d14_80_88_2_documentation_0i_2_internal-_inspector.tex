The {\bfseries{Graph Inspector}} makes it possible for you to interact with any selectable graph elements and graph-\/wide settings for a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_graph-_asset}{Shader Graph Asset}{0}. You can use the {\bfseries{Graph Inspector}} to edit attributes and default values.

When you open a Shader Graph, the {\bfseries{Graph Inspector}} displays the {\bfseries{Graph Settings}} tab by default. Graph-\/wide settings for that specific Shader Graph appear in this tab.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_internal-_inspector_autotoc_md3369}{}\doxysection{\texorpdfstring{How to use}{How to use}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_internal-_inspector_autotoc_md3369}
Select a node in the graph to display settings available for that node in the {\bfseries{Graph Inspector}}. Settings available for that node appear in the {\bfseries{Node Settings}} tab of the Graph Inspector. For example, if you select a Property node either in the graph or the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_blackboard}{Blackboard}{0}, the {\bfseries{Node Settings}} tab displays attributes of the Property that you can edit.
