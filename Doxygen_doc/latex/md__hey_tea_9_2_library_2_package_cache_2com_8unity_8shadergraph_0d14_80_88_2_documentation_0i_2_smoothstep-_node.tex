Returns the result of a smooth Hermite interpolation between 0 and 1, if the value of input {\bfseries{In}} is between the values of inputs {\bfseries{Edge1}} and {\bfseries{Edge2}} respectively. Returns 0 if the value of input {\bfseries{In}} is less than the value of input {\bfseries{Edge1}} and 1 if greater than the value of input {\bfseries{Edge2}}.

The Smoothstep node is similar to the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_lerp-_node}{Lerp Node}{0} but there are two notable differences. Firstly, with the Smoothstep node, the user specifies the range and the return value is between 0 and 1. You can consider this the opposite of the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_lerp-_node}{Lerp Node}{0}. Secondly, the Smoothstep node uses smooth Hermite interpolation instead of linear interpolation, which means the interpolation gradually speeds up from the start and slows down toward the end. This interpolation is useful for creating natural-\/looking animation, fading, and other transitions.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_smoothstep-_node_autotoc_md3879}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_smoothstep-_node_autotoc_md3879}
