The On Input \doxylink{namespace_system}{System} Event Float node lists for a specific Input Action from a \doxylink{class_player}{Player} Input component. The node can output a single float value.

Use this node when you want to read user input and return a single value, such as an axis value or input from a trigger on a controller.

In the following example, an On Input \doxylink{namespace_system}{System} Event Float node uses the bindings assigned to the {\bfseries{Lift}} Input Action. When a user presses any key from the {\bfseries{Lift}} binding, Visual Scripting takes the float value it receives from the Input \doxylink{namespace_system}{System} and sends it as an input to the Vector 3 Create node\textquotesingle{}s {\bfseries{Y}} input port. At the same time, Visual Scripting triggers the Transform Set Position node and uses the output from the Vector 3 Create node to set a new position for the Script Machine\textquotesingle{}s Game\+Object.



For this example, {\bfseries{Lift}} uses a Right Trigger from a Gamepad input device as a binding. When a user presses the Right Trigger, the {\bfseries{Y}} value of the Game\+Object\textquotesingle{}s transform increases, which makes the Game\+Object move upwards in the scene.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0d46e41f06f5b302fe098eb42e914b158_autotoc_md5448}{}\doxysection{\texorpdfstring{Related nodes}{Related nodes}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0d46e41f06f5b302fe098eb42e914b158_autotoc_md5448}
