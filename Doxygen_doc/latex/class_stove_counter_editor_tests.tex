\doxysubsection{Detailed Description}
Unit tests for the \doxylink{class_stove_counter}{Stove\+Counter} class. 

This class contains unit tests for various interaction scenarios involving the \doxylink{class_stove_counter}{Stove\+Counter} class,

\begin{DoxyNote}{Note}
milk\+Tea\+Material\+Type contains six types\+: tea\+Base, basic\+Add, ingredients, fruit, none, un treat. \doxylink{class_hey_tea_object}{Hey\+Tea\+Object} includes the objects contained in tea\+Base, basic\+Add, ingredients, fruit, none, un treat.
\begin{DoxyItemize}
\item tea\+Base\+: tea, milk, milktea
\item basic\+Add\+: sugar, ice
\item ingredients\+: red bean cooked, pearl cooked
\item fruit\+: orange slice, grape slice, strawberry slice
\item none\+: pot, cup
\item un treat\+: bagged pearl, bagged red bean, bagged sugar, grape, orange, strawberry 
\end{DoxyItemize}
\end{DoxyNote}
