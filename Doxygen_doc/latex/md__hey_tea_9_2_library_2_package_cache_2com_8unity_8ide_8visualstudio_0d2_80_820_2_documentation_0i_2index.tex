The Visual Studio Editor package provides the \doxylink{namespace_unity}{Unity} Editor with support for Unity-\/specific features from the \href{https://docs.microsoft.com/en-us/visualstudio/gamedev/unity/get-started/visual-studio-tools-for-unity}{\texttt{ Visual Studio Tools for Unity}} extension in \href{https://visualstudio.microsoft.com/}{\texttt{ Visual Studio}} and \href{https://visualstudio.microsoft.com/vs/mac/}{\texttt{ Visual Studio for Mac}}. These include Intelli\+Sense auto-\/complete suggestions, C\# editing, and debugging.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ide_8visualstudio_0d2_80_820_2_documentation_0i_2index_autotoc_md1001}{}\doxysection{\texorpdfstring{Installation}{Installation}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ide_8visualstudio_0d2_80_820_2_documentation_0i_2index_autotoc_md1001}
This package is a built-\/in package and installed by default.

{\bfseries{Note}}\+: If you’re using a version of the \doxylink{namespace_unity}{Unity} Editor before 2019.\+4, you’ll need to install this package through the package manager.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ide_8visualstudio_0d2_80_820_2_documentation_0i_2index_autotoc_md1002}{}\doxysection{\texorpdfstring{Requirements}{Requirements}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ide_8visualstudio_0d2_80_820_2_documentation_0i_2index_autotoc_md1002}
This version of the Visual Studio Editor package is compatible with the following versions of the \doxylink{namespace_unity}{Unity} Editor\+:


\begin{DoxyItemize}
\item 2019.\+4 and later
\end{DoxyItemize}

To use this package, you must have the following third-\/party products installed\+:


\begin{DoxyItemize}
\item {\bfseries{On Windows}}\+: Visual Studio 2019 version 16.\+9 or newer with Visual Studio Tools for \doxylink{namespace_unity}{Unity} 4.\+0.\+9 or newer.
\item {\bfseries{On mac\+OS}}\+: Visual Studio for Mac 2019 version 8.\+9 or newer with Visual Studio Tools for \doxylink{namespace_unity}{Unity} 2.\+0.\+9 or newer.
\end{DoxyItemize}

For more information about using Visual Studio with \doxylink{namespace_unity}{Unity}, see \href{https://docs.microsoft.com/en-us/visualstudio/gamedev/unity/get-started/visual-studio-tools-for-unity}{\texttt{ Microsoft’s Visual Studio Tools for Unity documentation}}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ide_8visualstudio_0d2_80_820_2_documentation_0i_2index_autotoc_md1003}{}\doxysection{\texorpdfstring{Submitting issues}{Submitting issues}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ide_8visualstudio_0d2_80_820_2_documentation_0i_2index_autotoc_md1003}
