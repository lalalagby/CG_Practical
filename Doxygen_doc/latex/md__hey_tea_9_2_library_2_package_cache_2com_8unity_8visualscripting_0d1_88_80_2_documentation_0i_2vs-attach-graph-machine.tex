 To use a Script Graph or State Graph file in your project, you must attach it to a Script Machine or State Machine.

A Script Machine or State Machine is a component. Components attach to Game\+Objects, and help define their behavior. For more information on components and Game\+Objects, see \href{https://docs.unity3d.com/2021.1/Documentation/Manual/UsingComponents.html}{\texttt{ Using components}} or \href{https://docs.unity3d.com/2021.1/Documentation/Manual/GameObjects.html}{\texttt{ Game\+Objects}} in the \doxylink{namespace_unity}{Unity} User Manual.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-attach-graph-machine_autotoc_md5170}{}\doxysection{\texorpdfstring{Add a Script Machine or State Machine component to a Game\+Object}{Add a Script Machine or State Machine component to a Game\+Object}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-attach-graph-machine_autotoc_md5170}

\item \mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_01b8576ec7a772dec58bafc7460cd57fb}{open-\/hierarchy}{0}\mbox{]}
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item In the Hierarchy window, select a Game\+Object where you\textquotesingle{}d like to add a Script Machine or State Machine.
\item \mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0e9d7825430e3771479edc4004f004d3d}{open-\/inspector-\/window}{0}\mbox{]} ~\newline

\item In the Game\+Object\textquotesingle{}s Inspector window, select {\bfseries{Add Component}}.

The Components menu opens.
\item Do one of the following\+:
\begin{DoxyItemize}
\item To add a Script Machine or State Machine, in the Components menu, go to {\bfseries{Visual Scripting}} and select {\bfseries{Script Machine}} or {\bfseries{State Machine}}.
\item Use the search bar to find the Script Machine or State Machine component.
\end{DoxyItemize}
\end{DoxyEnumerate}

The new Script Machine or State Machine component appears in the Inspector window for the Game\+Object\+:

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-attach-graph-machine_autotoc_md5171}{}\doxysection{\texorpdfstring{Attach a graph file to the Script Machine or State Machine}{Attach a graph file to the Script Machine or State Machine}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-attach-graph-machine_autotoc_md5171}

\begin{DoxyEnumerate}
\item In the Inspector window, locate your Script Machine or State Machine component.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Set the {\bfseries{Source}} to {\bfseries{Graph}}.
\item Do one of the following\+:
\begin{DoxyItemize}
\item In the {\bfseries{Graph}} field, select the object picker (circle icon) and choose a compatible graph file from your project.
\item Click and drag a file from your Project window and drop it into the {\bfseries{Graph}} field.
\end{DoxyItemize}
\end{DoxyEnumerate}
