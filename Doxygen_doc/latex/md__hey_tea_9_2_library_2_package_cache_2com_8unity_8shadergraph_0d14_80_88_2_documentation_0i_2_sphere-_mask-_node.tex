Creates a sphere mask originating from input {\bfseries{Center}}. The sphere is calculated using \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_distance-_node}{Distance}{0} and modified using the {\bfseries{Radius}} and {\bfseries{Hardness}} inputs. Sphere mask functionality works in both 2D and 3D spaces, and is based on the vector coordinates in the {\bfseries{Coords}} input. These vector coordinates can either be 3D like world space position, or 2D like UV coordinates.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sphere-_mask-_node_autotoc_md3948}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sphere-_mask-_node_autotoc_md3948}
