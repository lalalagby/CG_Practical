 The Subgraph Dropdown node is a node representation of a Dropdown property. It allows you to create a custom dropdown menu on a Subgraph node in its parent Shader Graph. You can specify the number of options that appear in the dropdown menu, and their names.

After you create a Dropdown property and add a Dropdown node to a Subgraph, the Subgraph node in any parent Shader Graph displays with a dropdown control\+:

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sub-_graph-_dropdown-_node_autotoc_md3982}{}\doxysection{\texorpdfstring{Create Node menu category}{Create Node menu category}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sub-_graph-_dropdown-_node_autotoc_md3982}
The Subgraph Dropdown node isn\textquotesingle{}t accessible from the Create Node menu.

To add a Subgraph Dropdown node to a Subgraph\+:


\begin{DoxyEnumerate}
\item In the Shader Graph window, open a Subgraph.
\item In the Blackboard, select {\bfseries{Add}} (+) and select {\bfseries{Dropdown}}.
\item Enter a name for your new Dropdown property, and press Enter.
\item Select your Dropdown property and drag it onto your graph to create a new Subgraph Dropdown node.
\item Select your new Dropdown node in your graph or the Dropdown property in the Blackboard and open the Graph Inspector.
\item Select the {\bfseries{Node Settings}} tab.
\item In the {\bfseries{Entries}} table, select {\bfseries{Add to the list}} (+) to add a new option to your dropdown. Each Entry adds a corresponding input port to your node. To remove an Entry, select its handle in the list and select {\bfseries{Remove selection from the list}} (-\/).
\item (Optional) In the {\bfseries{Default}} list, select the default Entry that you want Shader Graph to select on your property.
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
The Subgraph Dropdown node\textquotesingle{}s number of input ports and their names directly correspond to the settings you specify in the Graph Inspector\textquotesingle{}s {\bfseries{Node Settings}} tab. The node always has one output port.
\end{DoxyNote}
A Subgraph Dropdown node\textquotesingle{}s input ports always have the {\bfseries{Dynamic\+Vector}} type. This means that you can make a connection to an input port from any node that outputs a float, Vector 2, Vector 3, Vector 4, or Boolean value. For more information, see \href{Data-Types.md\#dynamic-data-types}{\texttt{ Dynamic Data Types}}.
