Defines a {\bfseries{Sampler State}} for sampling textures. It should be used in conjunction with sampling \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node}{Nodes}{0} such as the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sample-_texture-2_d-_node}{Sample Texture 2D Node}{0}. You can set a filter mode with the dropdown parameter {\bfseries{Filter}} and a wrap mode with the dropdown parameter {\bfseries{Wrap}}.

When using a separate {\bfseries{Sample State Node}} you can sample a {\bfseries{Texture 2D}} twice, with different sampler parameters, without defining the {\bfseries{Texture 2D}} itself twice.

Not all filtering, wrap, and Anisotropic filtering modes are available on all platforms.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sampler-_state-_node_autotoc_md3793}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sampler-_state-_node_autotoc_md3793}
