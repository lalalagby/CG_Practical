Provides access to the mesh vertex or fragment\textquotesingle{}s {\bfseries{View Direction}} vector. This is the vector from the vertex or fragment to the camera. Select a {\bfseries{Space}} to modify the coordinate space of the output value.

Prior to version 11.\+0, the {\bfseries{View Direction Node}} works differently in HDRP than in URP. In URP, it only stored Object space vectors normalized. HDRP stores all vectors normalized.

From 11.\+0 onwards, this node stores all vectors normalized in both the {\bfseries{High-\/\+Definition Render Pipeline}} and the {\bfseries{Universal Render Pipeline}}.
