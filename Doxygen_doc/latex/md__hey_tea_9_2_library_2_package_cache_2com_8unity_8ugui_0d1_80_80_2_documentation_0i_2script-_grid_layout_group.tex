Unlike other layout groups, the Grid Layout Group ignores the minimum, preferred, and flexible size properties of its contained layout elements and instead assigns a fixed size to all of them which is defined with the Cell Size property of the Grid Layout Group itself.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_grid_layout_group_autotoc_md4875}{}\doxysubsection{\texorpdfstring{Grid Layout Group and auto layout}{Grid Layout Group and auto layout}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_grid_layout_group_autotoc_md4875}
There are special considerations to be aware of when using the Grid Layout Group as part of an auto layout setup, such as using it with a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_content_size_fitter}{Content Size Fitter}{0}.

The auto layout system calculates the horizontal and vertical sizes independently. This can be at odds with the Grid Layout Group, where the number of rows depends on the number of columns and vice versa.

For any given number of cells, there are different combinations of row count and column count that can make the grid fit its content. In order to aid the layout system, you can specify that you intent the table to have a fixed number of columns or rows by using the Constraint property.

Here are suggested ways of using the Layout \doxylink{namespace_system}{System} with a Content Size Fitter\+:\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_grid_layout_group_autotoc_md4876}{}\doxysubsubsection{\texorpdfstring{Flexible width and fixed height}{Flexible width and fixed height}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_grid_layout_group_autotoc_md4876}
To setup a grid with a flexible width and fixed height, where the grid expands horizontally as more elements are added, you can set these properties as follows\+:


\begin{DoxyItemize}
\item Grid Layout Group Constraint\+: Fixed Row Count
\item Content Size Fitter Horizontal Fit\+: Preferred Size
\item Content Size Fitter Vertical Fit\+: Preferred Size or Unconstrained
\end{DoxyItemize}

If unconstrained Vertical Fit is used, it\textquotesingle{}s up to you to give the grid a height that is big enough to fit the specified row count of cells.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_grid_layout_group_autotoc_md4877}{}\doxysubsubsection{\texorpdfstring{Fixed width and flexible height}{Fixed width and flexible height}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_grid_layout_group_autotoc_md4877}
To setup a grid with a fixed width and flexible height, where the grid expands vertically as more elements are added, you can set these properties as follows\+:


\begin{DoxyItemize}
\item Grid Layout Group Constraint\+: Fixed Column Count
\item Content Size Fitter Horizontal Fit\+: Preferred Size or Unconstrained
\item Content Size Fitter Vertical Fit\+: Preferred Size
\end{DoxyItemize}

If unconstrained Horizontal Fit is used, it\textquotesingle{}s up to you to give the grid a width that is big enough to fit the specified column count of cells.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_grid_layout_group_autotoc_md4878}{}\doxysubsubsection{\texorpdfstring{Both flexible width and height}{Both flexible width and height}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_grid_layout_group_autotoc_md4878}
If you want a grid with both a flexible width and height you can do that, but you will have no control over the specific number of rows and columns. The grid will attempt to make the row and column count approximately the same. You can set these properties as follows\+:


\begin{DoxyItemize}
\item Grid Layout Group Constraint\+: Flexible
\item Content Size Fitter Horizontal Fit\+: Preferred Size
\item Content Size Fitter Vertical Fit\+: Preferred Size 
\end{DoxyItemize}
