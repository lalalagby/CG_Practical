 An Environment Library is an Asset that contains a list of environments that you can use in \href{Look-Dev.html}{\texttt{ Look Dev}} to simulate different lighting conditions. Each environment uses an HDRI (High Dynamic Range Image) for its skybox and also includes properties that you can use to fine-\/tune the environment.

To create an Environment Library Asset, either\+:


\begin{DoxyItemize}
\item Select {\bfseries{Assets \texorpdfstring{$>$}{>} Create \texorpdfstring{$>$}{>} Rendering Environment Library (Look Dev)}}.
\item Open \href{Look-Dev.html}{\texttt{ Look Dev}} and click the {\bfseries{New Library}} button.
\end{DoxyItemize}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8core_0d14_80_88_2_documen924317195c4b4be112e271118697d1f9_autotoc_md1796}{}\doxysection{\texorpdfstring{Creating and editing an environment}{Creating and editing an environment}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8core_0d14_80_88_2_documen924317195c4b4be112e271118697d1f9_autotoc_md1796}
After you create an Environment Library, you can add environments to it which you can then use in Look Dev. To create environments or edit their properties, use the Look Dev window itself. To create and edit environments, you need to open an Environment Library in Look Dev. To do this, either\+:


\begin{DoxyItemize}
\item Go to the Look Dev window (menu\+: {\bfseries{Window \texorpdfstring{$>$}{>} Rendering \texorpdfstring{$>$}{>} Look Dev}}) and drag your Environment Library from your Project window into the sidebar.
\item In your Project window, click on your Environment Library Asset. Then, in the Inspector, click the {\bfseries{Open in Look\+Dev window}} button.
\end{DoxyItemize}

If you already have environments in the Environment Library, you can see a list of them in the sidebar. When you click on any of the HDRI previews for an environment, a box appears at the bottom of the Environment Library list. This contains the selected environment\textquotesingle{}s properties for you to edit.

To add, remove, or duplicate environments, use the toolbar at the bottom of the Environment Library list, which contains the following buttons.

To import an HDRI Texture into the \doxylink{namespace_unity}{Unity} Editor, load an \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}.hdr\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} or \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}.exr\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} file into your \doxylink{namespace_unity}{Unity} Project like you would any other image. In the Texture Importer Inspector window, set {\bfseries{Texture Type}} to {\bfseries{Default}}, set {\bfseries{Texture Shape}} to {\bfseries{Cube}}, and set {\bfseries{Convolution Type}} to {\bfseries{None}}.

When you want to test an HDRI Texture Asset or a skybox cube map Material, drag and drop it into the Look Dev view.
