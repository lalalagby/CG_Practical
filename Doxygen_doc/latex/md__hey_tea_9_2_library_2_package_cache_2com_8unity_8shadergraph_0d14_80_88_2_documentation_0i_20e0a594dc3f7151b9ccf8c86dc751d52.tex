 This node calculates \href{https://docs.unity3d.com/Packages/com.unity.render-pipelines.high-definition@14.0/manual/WaterSystem-caustics.html}{\texttt{ water caustics}}.

This node outputs caustics as monochrome in the red channel. If you connect the output of this node to a {\bfseries{Base Color}} block, all the channels are red. To prevent this, split the output and use only the red channel. You can\textquotesingle{}t apply a tint to caustics.

Caustics don\textquotesingle{}t have an effect above the water unless you script this behavior. For example\+:


\begin{DoxyItemize}
\item If your scene contains a boat that sits in water, HDRP doesn\textquotesingle{}t project caustics on the part of the boat\textquotesingle{}s hull that\textquotesingle{}s above water.
\item A swimming pool inside a room doesn\textquotesingle{}t bounce caustics off the walls or ceiling.
\end{DoxyItemize}

The High Definition Render Pipeline (HDRP) uses this node in the default water shader graph.
