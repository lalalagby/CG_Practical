\doxysubsection{Detailed Description}
A single character text input event. 

Text input does not fit the control-\/based input model well and thus is represented as its own form of input. A device that is capable of receiving text input (such as \doxylink{class_unity_engine_1_1_input_system_1_1_keyboard}{Keyboard}) receives text input events and should implement \doxylink{interface_unity_engine_1_1_input_system_1_1_low_level_1_1_i_text_input_receiver}{IText\+Input\+Receiver} in order for the input system to be able to relay these events to the device.
