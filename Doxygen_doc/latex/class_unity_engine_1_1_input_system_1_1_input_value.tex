\doxysubsection{Detailed Description}
Wraps around values provided by input actions. 

This is a wrapper around \doxylink{struct_unity_engine_1_1_input_system_1_1_input_action_1_1_callback_context}{Input\+Action.\+Callback\+Context} chiefly for use with Game\+Object messages (i.\+e. Game\+Object.\+Send\+Message(string,object)). It exists so that action callback data can be represented as an object, can be reused, and shields the receiver from having to know about action callback specifics.
