The {\ttfamily Test\+Runner\+Api} retrieves and runs tests programmatically from code inside the project, or inside other packages. {\ttfamily Test\+Runner\+Api} is a \href{https://docs.unity3d.com/ScriptReference/ScriptableObject.html}{\texttt{ Scriptable\+Object}}.

You can initialize the API like this\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\ \#}}
\DoxyCodeLine{var\ testRunnerApi\ =\ ScriptableObject.CreateInstance<TestRunnerApi>();}

\end{DoxyCode}
 \begin{quote}
{\bfseries{Note}}\+: You can subscribe and receive test results in one instance of the API, even if the run starts from another instance. \end{quote}
