This node provides access to an unnormalized version of the mesh vertex or fragment\textquotesingle{}s {\bfseries{View Direction}} vector. It does not normalize any of the values it stores. For a normalized option, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_view-_direction-_node}{View Direction Node}{0}.
