 \doxylink{namespace_unity}{Unity} Cloud Build is a \href{https://docs.unity3d.com/2022.2/Documentation/Manual/UnityCloudBuild.html\#automated-build-generation}{\texttt{ continuous integration}} that automatically creates multiplatform builds in the Cloud in minutes. You can point Cloud Build toward your version control system to\+:


\begin{DoxyItemize}
\item Automate new builds
\item Build faster
\item Catch problems earlier
\item Iterate on your builds more efficiently with agility.
\end{DoxyItemize}

To get started, see \href{https://docs.unity3d.com/2022.2/Documentation/Manual/UnityCloudBuildMeteredBilling.html}{\texttt{ Pay as you go with Cloud Build}}.
