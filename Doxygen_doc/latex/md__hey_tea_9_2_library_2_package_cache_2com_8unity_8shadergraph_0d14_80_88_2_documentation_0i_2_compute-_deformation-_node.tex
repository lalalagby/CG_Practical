This node lets you pass compute deformed vertex data to a vertex shader, and only works with the \href{https://docs.unity3d.com/Packages/com.unity.rendering.hybrid@latest/}{\texttt{ DOTS Hybrid Renderer}}. You must provide {\ttfamily Deformed\+Vertex\+Data} in the {\ttfamily \+\_\+\+Deformed\+Mesh\+Data} buffer. The node uses the {\ttfamily \+\_\+\+Compute\+Mesh\+Index} property to calculate where the {\ttfamily Deformed\+Vertex\+Data} associated with the current mesh are located in the {\ttfamily \+\_\+\+Deformed\+Mesh\+Data} buffer. To output data, you must either install both the DOTS Hybrid Renderer and DOTS Animation packages, or use a custom solution.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_compute-_deformation-_node_autotoc_md3069}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_compute-_deformation-_node_autotoc_md3069}
