 A {\bfseries{Rect\+Mask2D}} is a masking control similar to the {\bfseries{Mask}} control. The mask restricts the child elements to the rectangle of the parent element. Unlike the standard Mask control it has some limitations, but it also has a number of performance benefits.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_rect_mask2_d_autotoc_md4908}{}\doxysection{\texorpdfstring{Description}{Description}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_rect_mask2_d_autotoc_md4908}
A common use of a Rect\+Mask2D is to show small sections of a larger area. Using the Rect\+Mask2D to frame this area.

The limitations of Rect\+Mask2D control are\+:


\begin{DoxyItemize}
\item It only works in 2D space
\item It will not properly mask elements that are not coplanar
\end{DoxyItemize}

The advantages of Rect\+Mask2D are\+:


\begin{DoxyItemize}
\item It does not use the stencil buffer
\item No extra draw calls
\item No material changes
\item Fast performance 
\end{DoxyItemize}
