Shader Graph can display colors on nodes in your graph to improve readability. This feature uses {\bfseries{Color Modes}} to change which colors to display in the graph. Use the {\bfseries{Color Mode\+:}} drop-\/down menu in the top right corner of the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_graph-_window}{Shader Graph Window}{0} to change the {\bfseries{Color Modes}}.

This mode displays colors on the nodes based on their current precision. If you set a node to {\bfseries{Inherit Precision}}, the display color reflects the currently active precision. See \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_precision-_modes}{Precision Modes}{0} for more information about inheritance.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_color-_modes_autotoc_md3046}{}\doxysubsection{\texorpdfstring{User Defined Colors}{User Defined Colors}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_color-_modes_autotoc_md3046}
This mode displays colors on the nodes based on user preferences. In this mode, the user defines colors for each node. If a custom color is not set, the node displays in the default gray.

To set a custom color for a node, right-\/click on the target node to bring up the the context menu, and select {\bfseries{Color}}.

For each project, you can override preset colors in the {\bfseries{Category}} and {\bfseries{Precision}} modes. \doxylink{namespace_unity}{Unity} uses a {\ttfamily .uss} style sheet and Hex color codes to set colors. The default style sheet in your project is {\ttfamily Packages/com.\+unity.\+shadergraph/\+Editor/\+Resources/\+Styles/\+Color\+Mode.uss}.

The best practice is to create a copy of this file to override the presets. Under your project\textquotesingle{}s {\bfseries{Assets}} folder, create a new {\ttfamily Editor/\+Resources/\+Styles} folder structure, and place a copy of {\ttfamily Color\+Mode.\+uss} in the {\ttfamily Styles} folder. Change the Hex color codes in this {\ttfamily .uss} file to override the presets and use your own custom colors for the {\bfseries{Category}} and {\bfseries{Precision}} modes.
