The {\bfseries{CPU Architecture}} setting is only supported for Windows, mac\+OS, Linux and Android. \doxylink{namespace_unity}{Unity} builds a \doxylink{class_player}{Player} that supports the CPU architectures you\textquotesingle{}ve selected. \doxylink{namespace_burst}{Burst} generates a special dispatch into the module, so that the code generated detects the CPU the target platform uses and selects the appropriate CPU architecture at runtime.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2building-aot-settings_autotoc_md289}{}\doxysection{\texorpdfstring{Optimize For setting}{Optimize For setting}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2building-aot-settings_autotoc_md289}
