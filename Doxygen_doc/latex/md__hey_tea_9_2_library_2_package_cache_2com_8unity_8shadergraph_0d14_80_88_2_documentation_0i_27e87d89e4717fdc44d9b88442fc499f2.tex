 This node calculates water foam intensity.

This node outputs foam as monochrome in the red channel. If you connect the output of this node to a {\bfseries{Base Color}} block, all the channels are red. To prevent this, split the output and use only the red channel. You can\textquotesingle{}t apply a tint to foam.

The High Definition Render Pipeline (HDRP) uses this node in the default water shader graph.
