Constructs square matrices from the four input vectors {\bfseries{M0}}, {\bfseries{M1}}, {\bfseries{M2}} and {\bfseries{M3}}. This node can be used to generate matrices of types {\bfseries{Matrix 2x2}}, {\bfseries{Matrix 3x3}} and {\bfseries{Matrix 4x4}}.

The dropdown on the node can be used to select whether the inputs values specify the matrix rows or columns.


\begin{DoxyItemize}
\item {\bfseries{Row}} \+: Input vectors specify matrix rows from top to bottom.
\item {\bfseries{Column}} \+: Input vectors specify matrix columns from left to right.
\end{DoxyItemize}

Matrix outputs are taken from the top left corner of the construction of the inputs. This can be used to generate different dimension square matrices from different dimension vectors.

For example, connecting {\bfseries{Vector 2}} type values to inputs {\bfseries{M0}} and {\bfseries{M1}} will generate the desired matrix from the output {\bfseries{2x2}}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_matrix-_construction-_node_autotoc_md3468}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_matrix-_construction-_node_autotoc_md3468}
