 The \doxylink{namespace_burst}{Burst} Inspector window displays all the jobs and other \doxylink{namespace_burst}{Burst} compile targets in the project. To open the \doxylink{namespace_burst}{Burst} Inspector window, go to {\bfseries{Jobs}} \texorpdfstring{$>$}{>} {\bfseries{\doxylink{namespace_burst}{Burst}}} \texorpdfstring{$>$}{>} {\bfseries{Open Inspector}}.

The \doxylink{namespace_burst}{Burst} Inspector displays all the Jobs that it can compile. It also displays the generated intermediate and native assembly code.

When opening a new target job in the \doxylink{namespace_burst}{Burst} Inspector, it will try to focus the assembly directly related to the chosen bursted job. Furthermore, if branch flow arrows are shown, and they fill more than half of the assembly view, the inspector will scroll horizontally rightwards to focus the code instead of the branches.

The {\bfseries{Compile Targets}} pane on the left of the window displays an alphabetical list of the jobs in the project that \doxylink{namespace_burst}{Burst} can compile. By default jobs either in the \doxylink{namespace_unity}{Unity} namespace or with "{}.\+Generated"{} in the name are excluded. This can be changed via the toggles {\bfseries{Show \doxylink{namespace_unity}{Unity} Namespace}} and {\bfseries{Show "{}.\+Generated"{}}} respectively. Disabled jobs in the list don\textquotesingle{}t have the {\ttfamily \mbox{[}Burst\+Compile\mbox{]}} attribute.

The right output pane of the \doxylink{namespace_burst}{Burst} Inspector window displays options to view the assembly and intermediate code for the job you\textquotesingle{}ve selected in the {\bfseries{Compile Targets}} list. To expand or collapse elements of the code, select the colored boxes (some with ellipses) . By default. the \doxylink{namespace_burst}{Burst} Inspector automatically collapses blocks that it considers non-\/essential, such as most directives and data.

It is possible to select lines of assembly. This will highlight the selected line, by underlining it. If this line contains any registers, the usage of these registers will be highlighted throughout the code; note that implicit registers are ignored for this feature.

To select and copy the text in this pane, either click and drag with your mouse, or use Shift + arrow keys to select the text. To copy the text, either right-\/click and select {\bfseries{Copy Selection}}, or press Ctrl + C (Command + C on mac\+OS). Default behavior for the \doxylink{namespace_burst}{Burst} Inspector\textquotesingle{}s copy is to include underlying color tags. To change this, right-\/click with the mouse on the right pane, to open up the context menu, and untick {\bfseries{Copy Color Tags}}.
