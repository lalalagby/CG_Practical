\doxysubsection{Detailed Description}
A serializable property type that can either reference an action externally defined in an \doxylink{class_unity_engine_1_1_input_system_1_1_input_action_asset}{Input\+Action\+Asset} or define a new action directly on the property. 

This struct is meant to be used for serialized fields in {\ttfamily Mono\+Behaviour} and {\ttfamily Scriptable\+Object} classes. It has a custom property drawer attached to it that allows to switch between using the property as a reference and using it to define an action in place.
