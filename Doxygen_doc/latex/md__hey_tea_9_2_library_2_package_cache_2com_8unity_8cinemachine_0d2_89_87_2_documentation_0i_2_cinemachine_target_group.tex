 Use \doxylink{namespace_cinemachine}{Cinemachine} Target Group to treat multiple Game\+Objects as a single Look At target. Use a Target Group with the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_aim_group_composer}{Group Composer}{0} algorithm.

To create a Virtual Camera with a Target Group\+:


\begin{DoxyEnumerate}
\item In the \doxylink{namespace_unity}{Unity} menu, choose {\bfseries{Game\+Object \texorpdfstring{$>$}{>} \doxylink{namespace_cinemachine}{Cinemachine} \texorpdfstring{$>$}{>} Target Group Camera}}. ~\newline
\doxylink{namespace_unity}{Unity} adds a new Virtual Camera and Target Group to the Scene. The {\bfseries{Follow}} and {\bfseries{Look At}} targets in the Virtual Camera refer to the new Target Group.
\item In the \href{https://docs.unity3d.com/Manual/Hierarchy.html}{\texttt{ Hierarchy}}, select the new Target Group object.
\item In the \href{https://docs.unity3d.com/Manual/UsingTheInspector.html}{\texttt{ Inspector}}, click the + sign to add a new item to the group.
\item In the new item, assign a Game\+Object (you can drag and drop from the Hierarchy), and edit the {\bfseries{Weight}} and {\bfseries{Radius}} properties.
\item To add more Game\+Objects to the Target Group, repeat steps 3-\/4.
\end{DoxyEnumerate}
