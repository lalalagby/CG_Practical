\doxylink{namespace_samples}{Samples} a Cubemap with reflected vector and returns a Vector 4 color value for use in the shader. Requires View Direction ({\bfseries{View Dir}}) and {\bfseries{Normal}} inputs to sample the Cubemap. You can achieve a blurring effect by using the {\bfseries{LOD}} input to sample at a different Level of Detail. You can also use the {\bfseries{Sampler}} input to define a custom Sampler State.

If you experience texture sampling errors while using this node in a graph which includes Custom Function Nodes or Sub Graphs, you can resolve them by upgrading to version 10.\+3 or later.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2b6883b1c31419cb881c9a464dafcc7a5_autotoc_md3749}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2b6883b1c31419cb881c9a464dafcc7a5_autotoc_md3749}
