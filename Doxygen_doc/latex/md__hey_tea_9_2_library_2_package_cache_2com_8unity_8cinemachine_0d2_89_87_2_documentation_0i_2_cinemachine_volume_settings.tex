 Use the \doxylink{namespace_cinemachine}{Cinemachine} Volume Settings \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_414e015286be387e4fe85aa25dd01df7}{extension}{0} to attach an HDRP/\+URP Volume\+Settings profile to a Virtual Camera.

The \doxylink{namespace_cinemachine}{Cinemachine} Volume Settings extension holds a Volume Settings Profile asset to apply to a Virtual Camera when it is activated. If the camera is blending with another Virtual Camera, then the blend weight is applied to the Volume Settings effects also.

To add a Volume Settings profile to a Virtual Camera


\begin{DoxyEnumerate}
\item Select your Virtual Camera in the \href{https://docs.unity3d.com/Manual/UsingTheSceneView.html}{\texttt{ Scene}} view or \href{https://docs.unity3d.com/Manual/Hierarchy.html}{\texttt{ Hierarchy}} window.
\item In the \href{https://docs.unity3d.com/Manual/UsingTheInspector.html}{\texttt{ Inspector}}, choose {\bfseries{Add Extension \texorpdfstring{$>$}{>} Cinemachine\+Volume\+Settings}}, then configure the Profile asset to have the effects you want when this virtual camera is live.
