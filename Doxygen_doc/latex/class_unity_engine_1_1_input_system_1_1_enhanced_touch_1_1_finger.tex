\doxysubsection{Detailed Description}
A source of touches (\doxylink{struct_unity_engine_1_1_input_system_1_1_enhanced_touch_1_1_touch}{Touch}). 

Each \doxylink{class_unity_engine_1_1_input_system_1_1_touchscreen}{Touchscreen} has a limited number of fingers it supports corresponding to the total number of concurrent touches supported by the screen. Unlike a \doxylink{struct_unity_engine_1_1_input_system_1_1_enhanced_touch_1_1_touch}{Touch}, a \doxylink{class_unity_engine_1_1_input_system_1_1_enhanced_touch_1_1_finger}{Finger} will stay the same and valid for the lifetime of its \doxylink{class_unity_engine_1_1_input_system_1_1_touchscreen}{Touchscreen}.

Note that a \doxylink{class_unity_engine_1_1_input_system_1_1_enhanced_touch_1_1_finger}{Finger} does not represent an actual physical finger in the world. That is, the same \doxylink{class_unity_engine_1_1_input_system_1_1_enhanced_touch_1_1_finger}{Finger} instance might be used, for example, for a touch from the index finger at one point and then for a touch from the ring finger. Each \doxylink{class_unity_engine_1_1_input_system_1_1_enhanced_touch_1_1_finger}{Finger} simply corresponds to the Nth touch on the given screen.
