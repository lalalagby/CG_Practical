\doxysubsection{Detailed Description}
Adding uncooked ingredients to the Pot. 

\begin{DoxyAuthor}{Author}
Yong Wu, Bingyu Guo
\end{DoxyAuthor}


\begin{DoxyNote}{Note}
milk\+Tea\+Material\+Type contains six types\+: tea\+Base, basic\+Add, ingredients, fruit, none, un treat. \doxylink{class_hey_tea_object}{Hey\+Tea\+Object} includes the objects contained in tea\+Base, basic\+Add, ingredients, fruit, none, un treat.
\begin{DoxyItemize}
\item tea\+Base\+: tea, milk, milktea
\item basic\+Add\+: sugar, ice
\item ingredients\+: red bean cooked, pearl cooked
\item fruit\+: orange slice, grape slice, strawberry slice
\item none\+: pot, cup
\item un treat\+: bagged pearl, bagged red bean, bagged sugar, grape, orange, strawberry 
\end{DoxyItemize}
\end{DoxyNote}

Try to add \doxylink{class_hey_tea_object}{Hey\+Tea\+Object} to the Pot. 

This method checks if the ingredient can be added to the Pot based on the current number of ingredients, the type of the ingredient, and if a recipe exists for the given ingredient.

The function performs the following steps\+:
\begin{DoxyItemize}
\item Checks if the total number of ingredients in the Pot has reached the maximum allowed ({\ttfamily total\+Inpot\+Num}).
\item Iterates through the list of milk tea material quotas to find a matching material type.
\item Validates if a recipe exists for the given ingredient.
\item Attempts to instantiate an intermediate state object for the ingredient.
\item Adds the instantiated object to the appropriate quota.
\item Resets the layers of all transform objects in {\ttfamily hey\+Tea\+Object\+Transform\+List}.
\item Logs the success or failure of the operation.
\item Resets the current progress time to 0 on successful addition. 
\end{DoxyItemize}
