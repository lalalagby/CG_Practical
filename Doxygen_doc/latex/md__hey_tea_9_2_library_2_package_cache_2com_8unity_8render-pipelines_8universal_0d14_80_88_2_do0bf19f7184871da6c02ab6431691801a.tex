 The {\bfseries{Shadows Midtones Highlights}} effect separately controls the shadows, midtones, and highlights of the render. Unlike \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do74dc799918dfd3281105755d7df23c68}{Lift, Gamma, Gain}{0}, you can use this effect to precisely define the tonal range for shadows, midtones, and highlights.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do0bf19f7184871da6c02ab6431691801a_autotoc_md2312}{}\doxysection{\texorpdfstring{Using Shadows Midtones Highlights}{Using Shadows Midtones Highlights}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do0bf19f7184871da6c02ab6431691801a_autotoc_md2312}
{\bfseries{Shadows Midtones Highlights}} uses the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2_volumes}{Volume}{0} framework, so to enable and modify the shadows, midtones, or highlights of the render, you must add a {\bfseries{Shadows Midtones Highlights}} override to a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2_volumes}{Volume}{0} in your Scene. To add {\bfseries{Shadows Midtones Highlights}} to a Volume\+:


\begin{DoxyEnumerate}
\item In the Scene or Hierarchy view, select a Game\+Object that contains a Volume component to view it in the Inspector.
\item In the Inspector, navigate to {\bfseries{Add Override \texorpdfstring{$>$}{>} Post-\/processing}} and click on {\bfseries{Shadows Midtones Highlights}}. URP now applies {\bfseries{Shadows Midtones Highlights}} to any Camera this Volume affects.
