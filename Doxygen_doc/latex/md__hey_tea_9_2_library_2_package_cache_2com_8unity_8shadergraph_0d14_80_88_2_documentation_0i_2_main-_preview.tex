The {\bfseries{Main Preview}} displays a representation of the shader on the active {\bfseries{Render Pipeline}}. It updates in real-\/time and automatically updates to display any changes you make in the Shader Graph. The title bar of the {\bfseries{Main Preview}} displays the name of the current shader.

The {\bfseries{Main Preview}} can be moved to anywhere in the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_graph-_window}{Shader Graph Window}{0} and will automatically move with the nearest corner of that window.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_main-_preview_autotoc_md3435}{}\doxysubsection{\texorpdfstring{Preview Mesh}{Preview Mesh}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_main-_preview_autotoc_md3435}
You can rotate the preview mesh by holding left mouse button and dragging on the {\bfseries{Main Preview}} and you can scale it by using the scroll wheel. The preview mesh can be changed by right clicking on the {\bfseries{Main Preview}}. Here you can select from any primitive mesh types or select a custom mesh.
