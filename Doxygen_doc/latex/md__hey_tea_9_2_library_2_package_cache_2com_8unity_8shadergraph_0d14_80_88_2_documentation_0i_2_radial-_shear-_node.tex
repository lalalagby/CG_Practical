Applies a radial shear warping effect similar to a wave to the value of input {\bfseries{UV}}. The center reference point of the warping effect is defined by input {\bfseries{Center}} and the overall strength of the effect is defined by the value of input {\bfseries{Strength}}. Input {\bfseries{Offset}} can be used to offset the individual channels of the result.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_radial-_shear-_node_autotoc_md3667}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_radial-_shear-_node_autotoc_md3667}
