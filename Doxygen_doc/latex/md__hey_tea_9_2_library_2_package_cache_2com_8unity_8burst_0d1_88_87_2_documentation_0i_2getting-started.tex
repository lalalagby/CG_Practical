 \doxylink{namespace_burst}{Burst} is primarily designed to work with \doxylink{namespace_unity}{Unity}\textquotesingle{}s \href{https://docs.unity3d.com/Manual/JobSystem.html}{\texttt{ job system}}. To start using the \doxylink{namespace_burst}{Burst} compiler in your code, decorate a \href{https://docs.unity3d.com/ScriptReference/Unity.Jobs.IJob.html}{\texttt{ Job struct}} with the \href{xref:Unity.Burst.BurstCompileAttribute}{\texttt{ {\ttfamily \mbox{[}Burst\+Compile\mbox{]}}}} attribute. Add the {\ttfamily \mbox{[}Burst\+Compile\mbox{]}} attribute to the type and the static method you want \doxylink{namespace_burst}{Burst} to compile.

\doxylink{namespace_burst}{Burst} supports most C\# expressions and statements, with a few exceptions. For more information, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2csharp-language-support}{C\# language support}{0}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2getting-started_autotoc_md404}{}\doxysection{\texorpdfstring{Compilation}{Compilation}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2getting-started_autotoc_md404}
\doxylink{namespace_burst}{Burst} compiles your code \href{https://en.wikipedia.org/wiki/Just-in-time_compilation}{\texttt{ just-\/in-\/time (JIT)}} while in Play mode in the Editor, and \href{https://en.wikipedia.org/wiki/Ahead-of-time_compilation}{\texttt{ ahead-\/of-\/time (AOT)}} when your application runs in a \doxylink{class_player}{Player}. For more information on compilation, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2compilation}{Burst compilation}{0}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2getting-started_autotoc_md405}{}\doxysection{\texorpdfstring{Command line options}{Command line options}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2getting-started_autotoc_md405}
You can pass the following options to the \doxylink{namespace_unity}{Unity} Editor on the command line to control \doxylink{namespace_burst}{Burst}\+:


\begin{DoxyItemize}
\item {\ttfamily -\/-\/burst-\/disable-\/compilation} disables \doxylink{namespace_burst}{Burst}.
\item {\ttfamily -\/-\/burst-\/force-\/sync-\/compilation} force \doxylink{namespace_burst}{Burst} to compile synchronously. For more information, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8burst_0d1_88_87_2_documentation_0i_2compilation}{Burst compilation}{0}. 
\end{DoxyItemize}
