 This type of track can be used to adjust \doxylink{namespace_unity}{Unity}\textquotesingle{}s global {\ttfamily Time.\+time\+Scale} for the duration of the clip. The most common use would be to create bullet-\/time style effects. The sample demonstrates the following\+:


\begin{DoxyItemize}
\item Creating a custom Track\+Mixer Playable\+Behaviour that performs custom blending of clip values.
\item Setting and restoring \doxylink{namespace_unity}{Unity} global values in a Playable\+Behaviour.
\item How to support blending and extrapolation on custom clips.
\item Provide custom clip data that can be animated using the inline curve editor using a Playable\+Behaviour template.
\end{DoxyItemize}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2smpl__custom__time_autotoc_md4718}{}\doxysection{\texorpdfstring{Usage}{Usage}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8timeline_0d1_87_85_2_documentation_0i_2smpl__custom__time_autotoc_md4718}
Create a Time\+Dilation\+Track in timeline using the Add Track menu, found under the \doxylink{namespace_timeline_1_1_samples}{Timeline.\+Samples} submenu. Add clips to the track, and use the inspector to set scale values for the clip, or use the inline curve editor to animate the scale values. Clips can also be overlapped to create transitions between clips.
