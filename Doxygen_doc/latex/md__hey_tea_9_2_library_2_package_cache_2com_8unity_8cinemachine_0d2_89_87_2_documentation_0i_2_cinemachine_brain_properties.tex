 The \doxylink{namespace_cinemachine}{Cinemachine} Brain is a component in the \doxylink{namespace_unity}{Unity} camera itself. \doxylink{namespace_cinemachine}{Cinemachine} Brain monitors all active Virtual Cameras in the Scene. It chooses the next Virtual Camera to control the \doxylink{namespace_unity}{Unity} camera. It also controls the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_blending}{cut or blend}{0} from the current Virtual Camera to the next.

To add a \doxylink{namespace_cinemachine}{Cinemachine} Brain component to a \doxylink{namespace_unity}{Unity} camera, do {\bfseries{one}} of the following\+:


\begin{DoxyItemize}
\item \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_set_up_v_cam}{Add a Virtual Camera}{0}, or other \doxylink{namespace_cinemachine}{Cinemachine} object, to your Scene. \doxylink{namespace_unity}{Unity} adds a \doxylink{namespace_cinemachine}{Cinemachine} Brain component to the \doxylink{namespace_unity}{Unity} camera for you if there isn’t one already.
\item \href{https://docs.unity3d.com/Manual/UsingComponents.html}{\texttt{ Add}} a \doxylink{namespace_cinemachine}{Cinemachine} Brain component to the \doxylink{namespace_unity}{Unity} camera yourself.
\end{DoxyItemize}

{\bfseries{Tip}}\+: You can also control Virtual Cameras from \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_timeline}{Timeline}{0}. \doxylink{namespace_timeline}{Timeline} overrides the decisions that \doxylink{namespace_cinemachine}{Cinemachine} Brain makes.

\doxylink{namespace_cinemachine}{Cinemachine} Brain holds the following key properties\+:


\begin{DoxyItemize}
\item {\bfseries{Blend Settings}}\+: A list that defines how to blend from one Virtual Camera to another. For example, add an item to the list for a 4 second blend from vcam1 to vcam2 then add another item for a 1 second blend from vcam2 back to vcam1. If a blend between two cameras isn’t defined, \doxylink{namespace_cinemachine}{Cinemachine} Brain uses its default blend.
\item {\bfseries{Layer Filter}}\+: \doxylink{namespace_cinemachine}{Cinemachine} Brain uses only those Virtual Cameras that pass the culling mask of the \doxylink{namespace_unity}{Unity} camera. You can set up \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_multiple_cameras}{split-\/screen environments}{0} by using the culling mask to filter layers.
\item {\bfseries{Event Dispatching}}\+: \doxylink{namespace_cinemachine}{Cinemachine} Brain fires events when it changes shot. It fires an event when a Virtual Camera goes live. It also fires an event when it cuts from one Virtual Camera to another. Use the latter event to reset temporal post effects.
\end{DoxyItemize}
