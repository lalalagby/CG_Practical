 To use nodes generated from a custom C\# script in a project after you rename a member, class, struct, type, or enum, add the {\ttfamily \mbox{[}Renamed\+From\mbox{]}} attribute to the relevant API element in the script file. For more information on the {\ttfamily \mbox{[}Renamed\+From\mbox{]}} attribute, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-refactoring}{Refactor a C\# script with Visual Scripting}{0}.

To add the attribute to a C\# script\+:


\begin{DoxyEnumerate}
\item 

\mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_02d03661124e0bd86dfb18db85fc8ceba}{open-\/project-\/window}{0}\mbox{]}


\item 

In the Project window, double-\/click the C\# script file you want to refactor. \doxylink{namespace_unity}{Unity} opens the file in the program you specified in your preferences, under {\bfseries{External Script Editor}}.

\doxysubsubsubsubsection*{NOTE}



For more information on script editors in \doxylink{namespace_unity}{Unity}, see \href{https://docs.unity3d.com/Manual/ScriptingToolsIDEs.html}{\texttt{ Integrated development environment (IDE) support}} in the \doxylink{namespace_unity}{Unity} User Manual.

  
\item 

In your external editor, do the following\+:


\begin{DoxyEnumerate}[label=\enumalphalphcnt*]

\item 

Add the {\ttfamily \mbox{[}Renamed\+From\mbox{]}} attribute above the definition of the part of the script you want to rename.


\item 

Add the element\textquotesingle{}s old name as a string to the {\ttfamily \mbox{[}Renamed\+From\mbox{]}} attribute, as its parameter. For example\+:


\begin{DoxyPre}{\ttfamily 
using \doxylink{namespace_unity_engine}{UnityEngine}; 
using \doxylink{namespace_unity_1_1_visual_scripting}{Unity.VisualScripting}; 

[RenamedFrom("{}Character"{})]
public class \doxylink{class_player}{Player} : MonoBehaviour
\{
    [RenamedFrom("{}InflictDamage"{})]
    public void TakeDamage(int damage)
    \{
        //...
    \}
\}
}\end{DoxyPre}
 
\end{DoxyEnumerate}
\item 

\mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2snippets_2vs-save-script}{save-\/script}{0}\mbox{]}


\item 

\mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2snippets_2vs-return-unity}{return-\/unity}{0}\mbox{]}


\item 

\mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_025c361676c0bf0aee90afedf32d5b844}{regen-\/node-\/library}{0}\mbox{]}


\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
If you change the namespace or namespaces used in your script, you must include the old namespace or namespaces to use the {\ttfamily \mbox{[}Renamed\+From\mbox{]}} attribute.
\end{DoxyNote}
