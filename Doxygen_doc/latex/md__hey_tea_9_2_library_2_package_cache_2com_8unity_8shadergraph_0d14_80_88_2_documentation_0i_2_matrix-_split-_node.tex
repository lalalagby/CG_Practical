Splits a square matrix defined by input {\bfseries{In}} into vectors. Output vector dimension is defined by the dimension of the input matrix.

The dropdown on the node can be used to select whether the output values are taken from the rows or columns of the input matrix.


\begin{DoxyItemize}
\item {\bfseries{Row}} \+: Output vectors are composed of matrix rows from top to bottom.
\item {\bfseries{Column}} \+: Output vectors are composed of matrix columns from left to right.
\end{DoxyItemize}

An input matrix of type {\bfseries{Matrix 2x2}} or {\bfseries{Matrix 3x3}} will return 0 values in the rows (or columns, depending on dropdown selection) that are beyond their dimension.

For example, connecting {\bfseries{Matrix 2x2}} type to input {\bfseries{In}} will return the correct {\bfseries{Vector 2}} type outputs to output slots {\bfseries{M0}} and {\bfseries{M1}}, leaving outputs {\bfseries{M2}} and {\bfseries{M3}} to return 0 values.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_matrix-_split-_node_autotoc_md3477}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_matrix-_split-_node_autotoc_md3477}
