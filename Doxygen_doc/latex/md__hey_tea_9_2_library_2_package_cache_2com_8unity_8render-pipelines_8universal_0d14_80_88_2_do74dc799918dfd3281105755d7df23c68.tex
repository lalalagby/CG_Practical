 This effect allows you to perform three-\/way color grading. The {\bfseries{Lift Gamma Gain}} trackballs follow the \href{https://en.wikipedia.org/wiki/ASC_CDL}{\texttt{ ASC CDL}} standard. When you adjust the position of the point on the trackball, it shifts the hue of the image towards that color in the given tonal range. Use the different trackballs to affect different ranges within the image. Adjust the slider under the trackball to offset the color lightness of that range.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do74dc799918dfd3281105755d7df23c68_autotoc_md2302}{}\doxysection{\texorpdfstring{Using Lift Gamma Gain}{Using Lift Gamma Gain}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do74dc799918dfd3281105755d7df23c68_autotoc_md2302}
{\bfseries{Lift Gamma Gain}} uses the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2_volumes}{Volume}{0} framework, so to enable and modify the lift, gamma, or gain of the render, you must add a {\bfseries{Lift Gamma Gain}} override to a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2_volumes}{Volume}{0} in your Scene. To add {\bfseries{Lift Gamma Gain}} to a Volume\+:


\begin{DoxyEnumerate}
\item In the Scene or Hierarchy view, select a Game\+Object that contains a Volume component to view it in the Inspector.
\item In the Inspector, navigate to {\bfseries{Add Override \texorpdfstring{$>$}{>} Post-\/processing}} and click on {\bfseries{Lift Gamma Gain}}. URP now applies {\bfseries{Lift Gamma Gain}} to any Camera this Volume affects.
\end{DoxyEnumerate}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do74dc799918dfd3281105755d7df23c68_autotoc_md2303}{}\doxysection{\texorpdfstring{Properties}{Properties}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do74dc799918dfd3281105755d7df23c68_autotoc_md2303}

{\bfseries{Lift}}   &Use this to control the dark tones. This has a more exaggerated effect on shadows.
\begin{DoxyItemize}
\item Use the trackball to select which color URP should shift the hue of the dark tones to.
\item Use the slider to offset the color lightness of the trackball color.
\end{DoxyItemize}\\\cline{1-2}
{\bfseries{Gamma}}   &Use this to control the mid-\/range tones with a power function.
\begin{DoxyItemize}
\item Use the trackball to select which color URP should use to shift the hue of the mid-\/tones to.
\item Use the slider to offset the color lightness of the trackball color.
\end{DoxyItemize}\\\cline{1-2}
{\bfseries{Gain}}   &Use this to increase the signal and make highlights brighter.
\begin{DoxyItemize}
\item Use the trackball to select which color that URP uses to shift the hue of the highlights to.
\item Use the slider to offset the color lightness of the trackball color.
\end{DoxyItemize}\\\cline{1-2}
