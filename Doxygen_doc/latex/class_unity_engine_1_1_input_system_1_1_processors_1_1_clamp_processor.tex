\doxysubsection{Detailed Description}
Clamp a floating-\/point input to between \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_clamp_processor_ab66841ff542bb80b9602e3a71fa80582}{min} and \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_clamp_processor_a5422b6fe7cea61651f31243bd7d36aa7}{max}. This is equivalent to {\ttfamily Mathf.\+Clamp(value, min, max)}. 

This processor is registered (see Input\+System.\+Register\+Processor$<$\+T$>$) under the name "{}clamp"{} by default.

Note that no normalization is performed. If you want to re-\/normalize the input value after clamping, add a \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_normalize_processor}{Normalize\+Processor}. Alternatively, add a \doxylink{class_unity_engine_1_1_input_system_1_1_processors_1_1_axis_deadzone_processor}{Axis\+Deadzone\+Processor} which both clamps and normalizes.
