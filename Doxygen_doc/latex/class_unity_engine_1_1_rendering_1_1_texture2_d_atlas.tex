\doxysubsection{Detailed Description}
A generic Atlas texture of 2D textures. An atlas texture is a texture collection that collects multiple sub-\/textures into a single big texture. Sub-\/texture allocation for \doxylink{class_unity_engine_1_1_rendering_1_1_texture2_d_atlas}{Texture2\+DAtlas} is static and will not change after initial allocation. Does not add mipmap padding for sub-\/textures.
