 This attribute is an alternative to the standard {\ttfamily Ignore} attribute in \href{http://www.nunit.org/}{\texttt{ NUnit}}. It allows for ignoring tests only under a specified condition. The condition evaluates during {\ttfamily On\+Load}, referenced by ID.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0fd4537c0503509846e4c58090226ad8f_autotoc_md4235}{}\doxysection{\texorpdfstring{Example}{Example}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8test-framework_0d1_81_833_2_documentation_0fd4537c0503509846e4c58090226ad8f_autotoc_md4235}
