The Universal Render Pipeline implements Physically Based Rendering (PBR).

The pipeline provides pre-\/built shaders that can simulate real world materials.

PBR materials provide a set of parameters that let artists achieve consistency between different material types and under different lighting conditions.

The URP \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2lit-shader}{Lit shader}{0} is suitable for modeling most of the real world materials. The \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_dobad11f4bc095c28bb5b8a1698bb5a9dc}{Complex Lit shader}{0} is suitable for simulating advanced materials that require more complex lighting evaluation, such as the clear coat effect.

URP provides the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_doe4da8ababb5b180f91a96b1c5f6e66e8}{Simple Lit shader}{0} as a helper to convert non-\/\+PBR projects made with the Built-\/in Render Pipeline to URP. This shader is non-\/\+PBR and is not supported by Shader Graph.

If you don’t need real-\/time lighting, or would rather only use \href{https://docs.unity3d.com/Manual/LightMode-Baked.html}{\texttt{ baked lighting}} and sample global illumination, choose a Baked Lit Shader.

If you don’t need lighting on a Material at all, you can choose the Unlit Shader.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do172a57c066d9bcc5364ac933caa8ecae_autotoc_md2484}{}\doxysection{\texorpdfstring{SRP Batcher compatibility}{SRP Batcher compatibility}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do172a57c066d9bcc5364ac933caa8ecae_autotoc_md2484}
To ensure that a Shader is SRP Batcher compatible\+:
\begin{DoxyItemize}
\item Declare all Material properties in a single CBUFFER called {\ttfamily Unity\+Per\+Material}.
\item Declare all built-\/in engine properties, such as {\ttfamily unity\+\_\+\+Object\+To\+World} or {\ttfamily unity\+\_\+\+World\+Transform\+Params}, in a single CBUFFER called {\ttfamily Unity\+Per\+Draw}.
\end{DoxyItemize}

For more information on the SRP Batcher, see the page \href{https://docs.unity3d.com/Manual/SRPBatcher.html}{\texttt{ Scriptable Render Pipeline (SRP) Batcher}}.
