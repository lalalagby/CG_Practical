A \doxylink{namespace_unity}{Unity} shader source file contains one or more \href{https://docs.unity3d.com/Manual/SL-SubShader.html}{\texttt{ Sub\+Shader}} blocks. When rendering a mesh, \doxylink{namespace_unity}{Unity} selects the first Sub\+Shader that is compatible with the GPU on the target device.

A Sub\+Shader block can optionally contain a Sub\+Shader Tags block. Use the {\ttfamily Tags} keyword to declare a Sub\+Shader Tags block.


\begin{DoxyCode}{0}
\DoxyCodeLine{Tags\ \{\ "{}RenderType"{}\ =\ "{}Opaque"{}\ "{}RenderPipeline"{}\ =\ "{}UniversalPipeline"{}\ \}}

\end{DoxyCode}


A Sub\+Shader Tag with a name of {\ttfamily Render\+Pipeline} tells \doxylink{namespace_unity}{Unity} which render pipelines to use this Sub\+Shader with, and the value of {\ttfamily Universal\+Pipeline} indicates that \doxylink{namespace_unity}{Unity} should use this Sub\+Shader with URP.

To execute the same shader in different render pipelines, create multiple Sub\+Shader blocks with different {\ttfamily Render\+Pipeline} tag values. To execute a Sub\+Shader block in HDRP, set the {\ttfamily Render\+Pipeline} tag to {\ttfamily HDRender\+Pipeline}, to execute it in the Built-\/in Render Pipeline, set {\ttfamily Render\+Pipeline} to an empty value.
