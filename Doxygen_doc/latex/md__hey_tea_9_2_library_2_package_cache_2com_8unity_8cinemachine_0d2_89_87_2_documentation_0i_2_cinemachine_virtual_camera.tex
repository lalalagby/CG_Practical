 The Cinemacine Virtual Camera is a component that you add to an empty Game\+Object. It represents a Virtual Camera in the \doxylink{namespace_unity}{Unity} Scene.

Use the {\bfseries{Aim}}, {\bfseries{Body}}, and {\bfseries{Noise}} properties to specify how the Virtual Camera animates position, rotation, and other properties. The Virtual Camera applies these settings to the \doxylink{namespace_unity}{Unity} Camera when \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_brain_properties}{Cinemachine Brain}{0} or \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_cinemachine_timeline}{Timeline}{0} transfers control of the \doxylink{namespace_unity}{Unity} camera to the Virtual Camera. You can also use \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2handles}{Scene Handles}{0} to modify some common properties.

At any time, each Virtual Camera may be in one of these states\+:


\begin{DoxyItemize}
\item {\bfseries{Live}}\+: The Virtual Camera actively controls a \doxylink{namespace_unity}{Unity} camera that has a \doxylink{namespace_cinemachine}{Cinemachine} Brain. When a \doxylink{namespace_cinemachine}{Cinemachine} Brain blends from one Virtual Camera to the next, both Virtual Cameras are live. When the blend is complete, there is only one live Virtual Camera.
\item {\bfseries{Standby}}\+: The Virtual Camera doesn’t control the \doxylink{namespace_unity}{Unity} camera. However, it still follows and aims at its targets, and updates at every frame. A Virtual Camera in this state is activated and has a priority that is the same as or lower than the live Virtual Camera.
\item {\bfseries{Disabled}}\+: The Virtual Camera doesn’t control the \doxylink{namespace_unity}{Unity} camera and doesn’t actively follow or aim at its targets. A Virtual Camera in this state doesn’t consume processing power. To disable a Virtual Camera, deactivate its game object. The Virtual Camera is present but disabled in the Scene. However, even though the game object is deactivated, the virtual camera can still control the \doxylink{namespace_unity}{Unity} camera if the virtual camera is participating in a blend, or if it is invoked by \doxylink{namespace_timeline}{Timeline}.
\end{DoxyItemize}
