 The Thread\+Map\+Detail node adds tileable thread map detail information to a fabric material. The node outputs a thread map that you can apply to a fabric material.



\mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2snippets_2nodes-subgraph-node}{nodes-\/subgraph-\/node}{0}\mbox{]}

A thread map is a Texture with 4 channels. Like a detail map, a thread map contains information about ambient occlusion, the normal x-\/axis and normal y-\/axis, and smoothness.
