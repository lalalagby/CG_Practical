{\bfseries{Bias}} 

Float 

Bias 

\doxysubsubsubsubsection*{NOTE}



The {\bfseries{Bias}} Input port only displays if {\bfseries{Mip Sampling Mode}} is {\bfseries{Bias}}. For more information, see \href{\#additional-node-settings}{\texttt{ Additional node settings}}.

 If {\bfseries{Use Global Mip Bias}} is enabled, \doxylink{namespace_unity}{Unity} adds this Bias amount to the Global Mip Bias for a texture\textquotesingle{}s mip calculation. If {\bfseries{Global Mip Bias}} is disabled, \doxylink{namespace_unity}{Unity} uses this Bias amount instead of the Global Mip Bias.
