Creates a mask from values in input {\bfseries{In}} equal to input {\bfseries{Mask Color}}. Input {\bfseries{Range}} can be used to define a wider range of values around input {\bfseries{Mask Color}} to create the mask. Colors within this range will return 1, otherwise the node will return 0. Input {\bfseries{Fuzziness}} can be used to soften the edges around the selection similar to anti-\/aliasing.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_color-_mask-_node_autotoc_md3039}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_color-_mask-_node_autotoc_md3039}
