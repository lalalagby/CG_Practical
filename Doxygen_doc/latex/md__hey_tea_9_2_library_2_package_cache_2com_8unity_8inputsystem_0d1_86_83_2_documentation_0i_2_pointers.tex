The coordinates within \doxylink{class_player}{Player} code are in the coordinate space of the \doxylink{class_player}{Player} window.

Within Editor code, the coordinates are in the coordinate space of the current \href{https://docs.unity3d.com/ScriptReference/EditorWindow.html}{\texttt{ {\ttfamily Editor\+Window}}}. If you query \href{../api/UnityEngine.InputSystem.Pointer.html\#UnityEngine_InputSystem_Pointer_position}{\texttt{ {\ttfamily Pointer.\+current.\+position}}} in \href{https://docs.unity3d.com/ScriptReference/EditorWindow.OnGUI.html}{\texttt{ {\ttfamily Unity\+Editor.\+Editor\+Window.\+On\+GUI}}}, for example, the returned 2D vector will be in the coordinate space of your local GUI (same as \href{https://docs.unity3d.com/ScriptReference/Event-mousePosition.html}{\texttt{ {\ttfamily Unity\+Engine.\+Event.\+mouse\+Position}}}).
