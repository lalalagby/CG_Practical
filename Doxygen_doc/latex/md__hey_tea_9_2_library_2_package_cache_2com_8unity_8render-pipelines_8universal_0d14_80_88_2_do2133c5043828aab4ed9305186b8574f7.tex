 Unity’s \href{https://docs.unity3d.com/Manual/ScriptableRenderPipeline.html}{\texttt{ Scriptable Render Pipeline (SRP)}} includes the {\bfseries{Lens Flare Data}} asset. You can use this asset to control the appearance of \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_doae565ffde777e364ece04ab423fabd6e}{Lens Flares}{0} in your scene. This is the SRP equivalent of the Built-\/in Render Pipeline\textquotesingle{}s \href{https://docs.unity3d.com/Manual/class-Flare.html}{\texttt{ Flare}} asset, which is incompatible with SRPs.

For examples of how to use Lens Flares, see the \href{../../package-sample-urp-package-samples.md\#lens-flares}{\texttt{ Lens Flare samples in URP Package Samples}}.

To create a Lens Flare Data asset, select {\bfseries{Assets \texorpdfstring{$>$}{>} Create \texorpdfstring{$>$}{>} Lens Flare (SRP)}}. To use this asset, assign it to the {\bfseries{Lens Flare Data}} property of a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_doae565ffde777e364ece04ab423fabd6e}{Lens Flare (SRP) component}{0}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do2133c5043828aab4ed9305186b8574f7_autotoc_md2496}{}\doxysection{\texorpdfstring{Properties}{Properties}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do2133c5043828aab4ed9305186b8574f7_autotoc_md2496}
