 Use the \doxylink{namespace_cinemachine}{Cinemachine} Storyboard \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8cinemachine_0d2_89_87_2_documentation_0i_2_414e015286be387e4fe85aa25dd01df7}{extension}{0} to let artists, producers, and directors contribute to your game development. \doxylink{namespace_cinemachine}{Cinemachine} Storyboard places a still image in screen space over the output of the \doxylink{namespace_unity}{Unity} camera.

Storyboard simplifies animatics for your team. Start with still images to pre-\/visualize terrain, layout, movement, lighting, timing, and so on. Following the intentions of the Storyboard image, developers incrementally add the assets, Game\+Objects, and settings that implement the Scene.

Use the properties in the Storyboard component to hide and show the image to compare it to the actual rendering of the \doxylink{namespace_unity}{Unity} camera.

Storyboards can be muted at a global level -\/ completely disabling them. To toggle this, go to the \doxylink{namespace_cinemachine}{Cinemachine} Storyboard component and enable {\itshape Storyboard Global Mute}.
