\doxysubsection{Detailed Description}
Wraps around mechanisms for listening in on the \doxylink{struct_unity_engine_1_1_input_system_1_1_low_level_1_1_input_event}{Input\+Event} stream made available through Input\+System.\+on\+Event. 

This struct can be used to add (op\+\_\+\+Addition) or remove (op\+\_\+\+Subtraction) callbacks directly to/from the event pipeline.

Alternatively, it can be used as an {\ttfamily IObservable} to \doxylink{struct_unity_engine_1_1_input_system_1_1_low_level_1_1_input_event_listener_ab8f1959922427d4d252fb6a7fac24f0f}{Subscribe} observers to the event stream. See Observable for extension methods to set up various observer mechanisms.
