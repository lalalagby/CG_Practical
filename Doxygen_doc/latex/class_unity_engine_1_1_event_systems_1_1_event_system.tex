\doxysubsection{Detailed Description}
Handles input, raycasting, and sending events. 

The \doxylink{class_unity_engine_1_1_event_systems_1_1_event_system}{Event\+System} is responsible for processing and handling events in a \doxylink{namespace_unity}{Unity} scene. A scene should only contain one \doxylink{class_unity_engine_1_1_event_systems_1_1_event_system}{Event\+System}. The \doxylink{class_unity_engine_1_1_event_systems_1_1_event_system}{Event\+System} works in conjunction with a number of modules and mostly just holds state and delegates functionality to specific, overrideable components. When the \doxylink{class_unity_engine_1_1_event_systems_1_1_event_system}{Event\+System} is started it searches for any Base\+Input\+Modules attached to the same Game\+Object and adds them to an internal list. On update each attached module receives an Update\+Modules call, where the module can modify internal state. After each module has been Updated the active module has the Process call executed.\+This is where custom module processing can take place.
