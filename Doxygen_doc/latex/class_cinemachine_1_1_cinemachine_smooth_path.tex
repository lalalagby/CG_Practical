\doxysection{Cinemachine.\+Cinemachine\+Smooth\+Path Class Reference}
\hypertarget{class_cinemachine_1_1_cinemachine_smooth_path}{}\label{class_cinemachine_1_1_cinemachine_smooth_path}\index{Cinemachine.CinemachineSmoothPath@{Cinemachine.CinemachineSmoothPath}}


Defines a world-\/space path, consisting of an array of waypoints, each of which has position and roll settings. Bezier interpolation is performed between the waypoints, to get a smooth and continuous path. The path will pass through all waypoints, and (unlike \doxylink{class_cinemachine_1_1_cinemachine_path}{Cinemachine\+Path}) first and second order continuity is guaranteed.

Bezier interpolation is performed between the waypoints, to get a smooth and continuous path. The path will pass through all waypoints, and (unlike \doxylink{class_cinemachine_1_1_cinemachine_path}{Cinemachine\+Path}) first and second order continuity is guaranteed.
