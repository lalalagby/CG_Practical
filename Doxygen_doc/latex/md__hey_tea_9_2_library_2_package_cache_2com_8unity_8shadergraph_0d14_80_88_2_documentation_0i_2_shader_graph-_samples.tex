The Shader Graph package offers sample Assets, which you can download through {\bfseries{Package Manager}}. When you import these samples, \doxylink{namespace_unity}{Unity} places the files in your Project\textquotesingle{}s Asset folder. The files contain examples that demonstrate how to use Shader Graph features.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader_graph-_samples_autotoc_md3857}{}\doxysection{\texorpdfstring{Add samples}{Add samples}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader_graph-_samples_autotoc_md3857}
To add samples to your Project, go to {\bfseries{Window}} \texorpdfstring{$>$}{>} {\bfseries{Package Manager}}. Locate {\bfseries{Shader Graph}} in the list of available packages, and select it. Under the package description, there is list of available samples. Click the {\bfseries{Import into Project}} button next to the sample you wish to add.



\doxylink{namespace_unity}{Unity} places imported samples in your Project\textquotesingle{}s Asset folder under {\bfseries{Assets}} \texorpdfstring{$>$}{>} {\bfseries{\doxylink{namespace_samples}{Samples}}} \texorpdfstring{$>$}{>} {\bfseries{Shader Graph}} \texorpdfstring{$>$}{>} {\bfseries{\mbox{[}version number\mbox{]}}} \texorpdfstring{$>$}{>} {\bfseries{\mbox{[}sample name\mbox{]}}}. The example below shows the samples for {\bfseries{Procedural Patterns}}.
