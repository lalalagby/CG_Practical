This node lets you apply Linear Blend Vertex Skinning, and only works with the \href{https://docs.unity3d.com/Packages/com.unity.rendering.hybrid@latest/}{\texttt{ DOTS Hybrid Renderer}}. You must provide skinned matrices in the {\ttfamily \+\_\+\+Skin\+Matrices} buffer.
