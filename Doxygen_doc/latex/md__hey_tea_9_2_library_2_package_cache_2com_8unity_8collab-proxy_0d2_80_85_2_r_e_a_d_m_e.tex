This is the package to add Collaborate support to the \doxylink{namespace_unity}{Unity} Editor. Unlike its predecessor Collab\+Proxy, this package has completely switched the UI to using UIElements. There is no more CEF, JS, or HTML.

The project is exclusively targeting .Net\+Standard 2.\+0 and will not work with the legacy Mono runtime.

The minimum supported version of the \doxylink{namespace_unity}{Unity} Editor is 2020.\+1a13.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8collab-proxy_0d2_80_85_2_r_e_a_d_m_e_autotoc_md892}{}\doxysection{\texorpdfstring{Development}{Development}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8collab-proxy_0d2_80_85_2_r_e_a_d_m_e_autotoc_md892}

\item {\ttfamily Editor/\+User\+Interface/\+Collaborate\+Window.\+cs} code for the window itself.
