\begin{DoxyEnumerate}
\item Create a new Asset by selecting {\bfseries{Assets \texorpdfstring{$>$}{>} Create \texorpdfstring{$>$}{>} Shader Graph \texorpdfstring{$>$}{>} URP \texorpdfstring{$>$}{>} Sprite Lit Shader Graph}}. The Shader Graph Asset is then created in the Asset window.

 ~\newline

\item Double-\/click the new Asset to open the {\bfseries{Shader Graph}}.  ~\newline

\item Create three {\bfseries{Sample Texture 2D}} Nodes by right-\/clicking on the Shader Graph window and selecting {\bfseries{Create Node}}, then search for and select the {\bfseries{Sample Texture 2D}} option.  ~\newline

\item Change the {\bfseries{Type}} of one of the Nodes to {\bfseries{Normal}}.  ~\newline

\item Attach the RGBA(4) {\bfseries{Output Slot}} of the {\bfseries{Default Type}} Nodes as shown below. Note that you should attach the {\bfseries{Normal Type}} Node\textquotesingle{}s Output Slot to the {\bfseries{Normal(\+Tangent Space)(3)}} Input Slot.  ~\newline

\item Create three {\bfseries{Texture 2D}} properties by selecting the {\bfseries{+}} on the \href{http://docs.unity3d.com/Packages/com.unity.shadergraph@12.0/manual/Blackboard.html}{\texttt{ Blackboard}}, and then select {\bfseries{Texture 2D}}. Name them \textquotesingle{}Main\+Tex\textquotesingle{}, \textquotesingle{}Mask\+Tex\textquotesingle{}, and \textquotesingle{}Normal\+Map\textquotesingle{} for this example.   ~\newline

\item Drag each of the {\bfseries{Texture 2D}} properties onto the editor window. Attach each of the properties to the {\bfseries{Input Slots}} of the Sample Texture 2D Nodes as shown below. Note that the \textquotesingle{}Normal\+Map\textquotesingle{} property must be attached to the {\bfseries{Normal Type}} Node only.  ~\newline

\item Select {\bfseries{Save Asset}} to save the Shader. 
\end{DoxyEnumerate}

You can now apply the newly built Shader to materials.
