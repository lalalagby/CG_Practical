Find an \doxylink{class_unity_engine_1_1_input_system_1_1_input_action}{Input\+Action} in the collection by its \doxylink{class_unity_engine_1_1_input_system_1_1_input_action_a18e68eda84c66e46fc7908fe67b8ffce}{Input\+Action.\+name} or by its \doxylink{class_unity_engine_1_1_input_system_1_1_input_action_a20c55a69282fe033ba0ca4ce706e4a68}{Input\+Action.\+id} (in string form). 


\begin{DoxyParams}{Parameters}
{\em action\+Name\+Or\+Id} & Name of the action as either a "{}map/action"{} combination (e.\+g. "{}gameplay/fire"{}) or a simple name. In the former case, the name is split at the \textquotesingle{}/\textquotesingle{} slash and the first part is used to find a map with that name and the second part is used to find an action with that name inside the map. In the latter case, all maps are searched in order and the first action that has the given name in any of the maps is returned. Note that name comparisons are case-\/insensitive.\\
\hline
\end{DoxyParams}
Alternatively, the given string can be a GUID as given by \doxylink{class_unity_engine_1_1_input_system_1_1_input_action_a20c55a69282fe033ba0ca4ce706e4a68}{Input\+Action.\+id}.


\begin{DoxyParams}{Parameters}
{\em throw\+If\+Not\+Found} & If {\ttfamily true}, instead of returning {\ttfamily null} when the action cannot be found, throw {\ttfamily Argument\+Exception}.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
The action with the corresponding name or {\ttfamily null} if no matching action could be found.
\end{DoxyReturn}
