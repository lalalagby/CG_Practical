When \doxylink{namespace_unity}{Unity} renders with GPU instancing, it assigns an {\bfseries{Instance ID}} to each geometry.

Use this node to capture {\bfseries{Instance ID}} values in \href{https://docs.unity3d.com/ScriptReference/Graphics.DrawMeshInstanced.html}{\texttt{ {\ttfamily Graphics.\+Draw\+Mesh\+Instanced}}} API calls.

When \doxylink{namespace_unity}{Unity} does not render with GPU instancing, this ID is 0.

When \doxylink{namespace_unity}{Unity} uses dynamic instancing, instance IDs might not be consistent across multiple frames.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_instance-_i_d-_node_autotoc_md3361}{}\doxysection{\texorpdfstring{Ports}{Ports}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_instance-_i_d-_node_autotoc_md3361}
