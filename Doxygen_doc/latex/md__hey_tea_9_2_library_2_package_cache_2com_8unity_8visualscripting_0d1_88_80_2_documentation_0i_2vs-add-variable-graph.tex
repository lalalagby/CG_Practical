 You can create and add a variable to a Script Graph in one of two ways\+: use the Graph window\textquotesingle{}s Blackboard, or add a node to a graph.

\begin{DoxyNote}{Note}
You can\textquotesingle{}t add a Flow variable to a graph from the Blackboard. For more information on Flow variables, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-variables}{Variables}{0}.
\end{DoxyNote}
\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-add-variable-graph_autotoc_md5161}{}\doxysection{\texorpdfstring{Create and add a new variable through the Blackboard}{Create and add a new variable through the Blackboard}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-add-variable-graph_autotoc_md5161}
To create a new variable in the Blackboard and add it to a graph\+:


\begin{DoxyEnumerate}
\item With a graph \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-open-graph-edit}{open in the Graph window}{0}, open \href{vs-interface-overview.md\#the-blackboard}{\texttt{ the Blackboard}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item In the Blackboard, select the scope for the variable you want to add\+: {\bfseries{Graph}}, {\bfseries{Object}}, {\bfseries{Scene}}, {\bfseries{App}}, or {\bfseries{Saved}}. For more information on variable scopes and when you can use each scope in your graph, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-variables}{Variables}{0}.
\item In the \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}(New Variable Name)\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} field, enter a name for the new variable.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Do one of the following\+:
\begin{DoxyItemize}
\item Press Enter.
\item Select the {\bfseries{Add Variable}} (+) button.
\end{DoxyItemize}
\item In the {\bfseries{Type}} list, select the data type for your variable. For more information on types, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-types}{Object types}{0}.
\item In the {\bfseries{Value}} field, enter or modify the default value for your variable.
\item To add the node to your graph, click and drag from the Handle (=) on the variable\textquotesingle{}s definition in the Blackboard into the Graph Editor.
\end{DoxyEnumerate}

Visual Scripting adds a new Get Variable node for your variable to your graph.

\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-add-variable-graph_autotoc_md5162}{}\doxysection{\texorpdfstring{Create and add a new variable through the Graph Editor}{Create and add a new variable through the Graph Editor}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-add-variable-graph_autotoc_md5162}
To create a Flow variable or another new variable directly in a graph\+:


\begin{DoxyEnumerate}
\item \mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_06fd8d5e00402366f8aaeca25bc9a4448}{with-\/graph-\/open-\/ff}{0}\mbox{]}
\item Select the {\bfseries{Variables}} category.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Select the scope for the variable you want to add\+: {\bfseries{Flow}}, {\bfseries{Graph}}, {\bfseries{Object}}, {\bfseries{Scene}}, {\bfseries{Application}}, or {\bfseries{Saved}}. For more information on variable scopes and when you can use each scope in a graph, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-variables}{Variables}{0}.
\item Select the {\bfseries{Set {\ttfamily \texorpdfstring{$<$}{<}Scope\texorpdfstring{$>$}{>}} Variable}} node, where {\ttfamily \texorpdfstring{$<$}{<}Scope\texorpdfstring{$>$}{>}} is the scope you selected in the previous step. Visual Scripting adds a new Set Variable node to the graph.
\item In the {\bfseries{Name}} field, enter a name for the new variable.
\item (Object variables only) In the {\bfseries{Game\+Object}} field, indicated by a Game\+Object icon on the node, specify the Game\+Object where you want to create the variable. Do one of the following\+:
\begin{DoxyItemize}
\item Select the object picker (circle icon) and select a Game\+Object.
\item Attach a node to the field\textquotesingle{}s data input port that outputs a Game\+Object.
\item Leave the field as the default value of {\bfseries{This}} to use the Game\+Object where you attached the Script Graph to a Script Machine.
\end{DoxyItemize}
\item To set a default value for your variable, connect another node that outputs the value you want to the Set Variable node\textquotesingle{}s data input port.
\end{DoxyEnumerate}
