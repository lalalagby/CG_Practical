\doxysubsection{Detailed Description}
Interface to navigate through Timelines and Sub\+Timelines for the \doxylink{namespace_unity_editor_1_1_timeline}{Timeline} window. 

\doxylink{class_unity_editor_1_1_timeline_1_1_timeline_navigator}{Timeline\+Navigator} gives you access to the \doxylink{namespace_unity_editor_1_1_timeline}{Timeline} window breadcrumbs functionality. Use it to programmatically dig into Sub\+Timelines, navigate to parent Timelines or navigate \doxylink{namespace_unity_editor_1_1_timeline}{Timeline} Window breadcrumbs.
