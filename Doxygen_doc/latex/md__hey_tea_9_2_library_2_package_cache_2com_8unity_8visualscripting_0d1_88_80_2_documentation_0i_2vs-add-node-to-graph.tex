 All logic in a Script Graph starts with a node.

\begin{DoxyRemark}{Remarks}
Depending on the method you used to create your Script Graph, you might already have two Event nodes in your graph\+: Start and Update. For more information on these nodes, see \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-events-reference}{Events node}{0}.
\end{DoxyRemark}
To add a node to a Script Graph\+:


\begin{DoxyEnumerate}
\item \mbox{[}!include\doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_06fd8d5e00402366f8aaeca25bc9a4448}{with-\/graph-\/open-\/ff}{0}\mbox{]}
\item In the fuzzy finder, enter a search term into the Search bar or select a category from the list to find related nodes. Categories have an arrow (\texorpdfstring{$>$}{>}) at the end of their entry in the fuzzy finder.
\item Select an entry in the fuzzy finder to add that node to your Script Graph. Visual Scripting adds the node to your Script Graph at the location where you opened the fuzzy finder.
\end{DoxyEnumerate}
