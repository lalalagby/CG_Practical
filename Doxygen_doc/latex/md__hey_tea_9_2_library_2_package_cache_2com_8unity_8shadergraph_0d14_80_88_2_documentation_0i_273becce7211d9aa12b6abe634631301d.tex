\doxylink{namespace_samples}{Samples} a {\bfseries{Texture 2D}} and returns a {\bfseries{Vector 4}} color value for use in the shader. You can override the {\bfseries{UV}} coordinates using the {\bfseries{UV}} input and define a custom {\bfseries{Sampler State}} using the {\bfseries{Sampler}} input. Use the {\bfseries{LOD}} input to adjust the level of detail of the sample.

To use the {\bfseries{Sample Texture 2D LOD Node}} to sample a normal map, set the {\bfseries{Type}} dropdown parameter to {\bfseries{Normal}}.

This \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node}{Node}{0} is useful for sampling a {\bfseries{Texture}} in the vertex \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_stage}{Shader Stage}{0} as the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_sample-_texture-2_d-_node}{Sample Texture 2D Node}{0} is unavailable in this \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_shader-_stage}{Shader Stage}{0}.

On platforms that do not support this operation, opaque black is returned instead.
