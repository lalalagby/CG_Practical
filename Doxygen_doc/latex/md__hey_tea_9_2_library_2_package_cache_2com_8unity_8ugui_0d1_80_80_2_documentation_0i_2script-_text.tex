Some controls (such as \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_button}{Buttons}{0} and \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_toggle}{Toggles}{0}) have textual descriptions built-\/in. For controls that have no implicit text (such as \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_slider}{Sliders}{0}), you can indicate the purpose using a label created with a Text control. Text is also useful for lists of instructions, story text, conversations and legal disclaimers.

The Text control offers the usual parameters for font size, style, etc, and text alignment. When the {\itshape Rich Text} option is enabled, markup elements within the text will be treated as styling information, so you can have just a single word or short section in boldface or in a different color, say (see the page about \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2_styled_text}{Rich Text}{0} for details of the markup scheme).\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_text_autotoc_md4937}{}\doxysection{\texorpdfstring{Hints}{Hints}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8ugui_0d1_80_80_2_documentation_0i_2script-_text_autotoc_md4937}
