Returns a {\bfseries{Dielectric Specular}} F0 value for a physically based material. The material to use can be selected with the {\bfseries{Material}} dropdown parameter on the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_node}{Node}{0}.

A {\bfseries{Common}} {\bfseries{Material}} type defines a range between 0.\+034 and 0.\+048 s\+RGB values. The value between this range can be selected with the {\bfseries{Range}} parameter. This {\bfseries{Material}} type should be used for various materials such as plastics and fabrics.

You can use {\bfseries{Custom}} material type to define your own physically based material value. The output value in this case is defined by its index of refraction.
