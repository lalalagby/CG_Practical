 To switch between states in a State Graph, you must use a Script Graph called a transition. ~\newline


For more information on State Graphs, see \href{vs-graph-types.md\#state-graphs}{\texttt{ State Graphs}}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-creating-transition_autotoc_md5303}{}\doxysection{\texorpdfstring{Create a transition with an embedded Script Graph}{Create a transition with an embedded Script Graph}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-creating-transition_autotoc_md5303}
To create a new transition to another state with an embedded graph in a State Graph\+:


\begin{DoxyEnumerate}
\item With a State Graph \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8visualscripting_0d1_88_80_2_documentation_0i_2vs-open-graph-edit}{open in the Graph window}{0}, do one of the following\+:
\begin{DoxyItemize}
\item Right-\/click on the state where you want to make a transition, then in the context menu, select {\bfseries{Make Transition}}.
\item Select the state where you want to make a transition, then press Ctrl+click and drag away from your selected state.
\end{DoxyItemize}
\item Do one of the following\+:
\begin{DoxyItemize}
\item Select or release while on an existing state in your State Graph to connect the states with a transition.
\item Select or release while on an empty space in the Graph Editor to automatically create a new blank Script State at the end of your transition.
\end{DoxyItemize}
\item Select your transition node.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item Open the \href{vs-interface-overview.md\#the-graph-inspector}{\texttt{ Graph Inspector}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item In the Graph Inspector, set the {\bfseries{Source}} to {\bfseries{Embed}}.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item In the \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}(Title)\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} field, enter a title for your transition\textquotesingle{}s Script Graph.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item In the \texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*}(Summary)\texorpdfstring{$\ast$}{*}\texorpdfstring{$\ast$}{*} field, enter a brief descriptive summary of your transition\textquotesingle{}s Script Graph.
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
If you choose to use an embedded transition Script Graph, Visual Scripting automatically provides the Trigger Transition node you need for the graph.
\end{DoxyNote}
