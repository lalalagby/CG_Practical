 The Universal Render Pipeline (URP) comes with a set of samples to help you get started.

A sample is a set of assets that you can import into your \doxylink{namespace_unity}{Unity} project and use as a base to build upon or learn how to use a feature. A package sample can contain anything from a single C\# script to multiple scenes.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2package-samples_autotoc_md2246}{}\doxysection{\texorpdfstring{Importing package samples}{Importing package samples}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2package-samples_autotoc_md2246}
Before you import any package samples for URP, be aware that they require your project to be URP-\/compatible. A project is URP-\/compatible if you \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do0e54e280482c9e2055dcf6e7a3ab1ea3}{created it from a template}{0} or manually \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do725301a77635678c75535ca740a1ac66}{installed and set up URP in it}{0}. If the project is not URP-\/compatible, errors can occur when you import a package sample.

To import package samples, use the \href{https://docs.unity3d.com/Manual/upm-ui.html}{\texttt{ Unity Package Manager window}}\+:


\begin{DoxyEnumerate}
\item Go to {\bfseries{Window}} \texorpdfstring{$>$}{>} {\bfseries{Package Manager}} and, in the \href{https://docs.unity3d.com/Manual/upm-ui-list.html}{\texttt{ packages list view}}, select {\bfseries{Universal RP}}.
\item In the package \href{https://docs.unity3d.com/Manual/upm-ui-details.html}{\texttt{ details view}}, find the {\bfseries{\doxylink{namespace_samples}{Samples}}} section.
\item Find the sample you want to import and click the {\bfseries{Import}} button next to it.
\end{DoxyEnumerate}

\doxylink{namespace_unity}{Unity} imports URP package samples into {\ttfamily Assets/\+Samples/\+Universal RP/\texorpdfstring{$<$}{<}package version\texorpdfstring{$>$}{>}/\texorpdfstring{$<$}{<}sample name\texorpdfstring{$>$}{>}}.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2package-samples_autotoc_md2247}{}\doxysection{\texorpdfstring{Opening package samples}{Opening package samples}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2package-samples_autotoc_md2247}
To open a package sample\+:


\begin{DoxyEnumerate}
\item Go to {\ttfamily Assets/\+Samples/\+Universal RP/\texorpdfstring{$<$}{<}package version\texorpdfstring{$>$}{>}/}. Here there is a folder for each URP package sample you have imported.
\item Find the folder that contains the package sample you want and open it. The folder has the same name that the package sample has in the \doxylink{namespace_unity}{Unity} Package Manager window.
\end{DoxyEnumerate}\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2package-samples_autotoc_md2248}{}\doxysection{\texorpdfstring{Package samples list}{Package samples list}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2package-samples_autotoc_md2248}
