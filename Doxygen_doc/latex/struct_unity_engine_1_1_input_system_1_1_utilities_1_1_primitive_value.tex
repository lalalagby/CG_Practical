\doxysubsection{Detailed Description}
A union holding a primitive value. 

This structure is used for storing things such as default states for controls (see \doxylink{struct_unity_engine_1_1_input_system_1_1_layouts_1_1_input_control_layout_1_1_control_item_a5d68782615098767cf2f0b9022400c66}{Layouts.\+Input\+Control\+Layout.\+Control\+Item.\+default\+State}). It can store one value of any primitive, non-\/reference C\# type (bool, char, int, float, etc).
