\doxysubsection{Detailed Description}
Specifies the type of object that should be bound to a \doxylink{class_unity_engine_1_1_timeline_1_1_track_asset}{Track\+Asset}. 

Use this attribute when creating Custom Tracks to specify the type of object the track requires a binding to.
