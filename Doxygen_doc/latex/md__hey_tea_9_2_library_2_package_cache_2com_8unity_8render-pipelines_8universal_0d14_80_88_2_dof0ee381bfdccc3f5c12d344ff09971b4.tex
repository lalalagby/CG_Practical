{\bfseries{Name}}   &Name of the Full Screen Pass Renderer Feature.    \\\cline{1-2}
{\bfseries{Pass Material}}   &The Material the Renderer Feature uses to render the effect.    \\\cline{1-2}
{\bfseries{Injection Point}}   &Select when the effect is rendered\+:
\begin{DoxyItemize}
\item {\bfseries{Before Rendering Transparents}}\+: Add the effect after the skybox pass and before the transparents pass.
\item {\bfseries{Before Rendering Post Processing}}\+: Add the effect after the transparents pass and before the post-\/processing pass.
\item {\bfseries{After Rendering Post Processing}}\+: Add the effect after the post-\/processing pass and before After\+Rendering pass.
\end{DoxyItemize}{\bfseries{After Rendering Post Processing}} is the default setting.    \\\cline{1-2}
{\bfseries{Requirements}}   &Select one or more of the following passes for the Renderer Feature to use\+:
\begin{DoxyItemize}
\item {\bfseries{None}}\+: Add no additional passes.
\item {\bfseries{Everything}}\+: Adds all additional passes available (Depth, Normal, Color, and Motion).
\item {\bfseries{Depth}}\+: Adds a depth prepass to enable the use of depth values.
\item {\bfseries{Normal}}\+: Enables the use of normal vector data.
\item {\bfseries{Color}}\+: Copies color data of a screen to the \+\_\+\+Blit\+Texture texture inside the shader.
\item {\bfseries{Motion}}\+: Enables the use of motion vectors.
\end{DoxyItemize}{\bfseries{Color}} is the default setting.    \\\cline{1-2}
{\bfseries{Pass Index}}   &Select a specific pass inside the Pass Material\textquotesingle{}s shader for the Pass Material to use.~\newline
~\newline
This option is hidden by default. To access this option, click \&\#8942; in the Renderer Feature section of the Inspector and select {\bfseries{Show Additional Properties}}.   \\\cline{1-2}
