What the \doxylink{namespace_timeline}{Timeline} window shows depends on what you select in either the Project window or the Scene view.

For example, if you select a Game\+Object that is associated with a \doxylink{namespace_timeline}{Timeline} Asset, the \doxylink{namespace_timeline}{Timeline} window shows the tracks and clips from the \doxylink{namespace_timeline}{Timeline} Asset and the Game\+Object bindings from the \doxylink{namespace_timeline}{Timeline} instance.

{\itshape With no Game\+Object selected, the \doxylink{namespace_timeline}{Timeline} window provides instructions}

To use the \doxylink{namespace_timeline}{Timeline} window to view a previously created \doxylink{namespace_timeline}{Timeline} Asset, select the \doxylink{namespace_timeline}{Timeline} Asset in the Project window and open the \doxylink{namespace_timeline}{Timeline} window. The \doxylink{namespace_timeline}{Timeline} window shows the tracks and clips associated with the \doxylink{namespace_timeline}{Timeline} Asset, but without the track bindings to Game\+Objects in the Scene. In addition, the \doxylink{namespace_timeline}{Timeline} Playback Controls are disabled and there is no \doxylink{namespace_timeline}{Timeline} Playhead.



{\itshape \doxylink{namespace_timeline}{Timeline} Asset selected in the Project window shows its tracks and clips, but with no track bindings. The \doxylink{namespace_timeline}{Timeline} Playback Controls are disabled.}
