 URP Package \doxylink{namespace_samples}{Samples} is a \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_documentation_0i_2package-samples}{package sample}{0} for the Universal Render Pipeline (URP). It contains example shaders, C\# scripts, and other assets you can build upon, use to learn how to use a feature, or use directly in your application. For information on how to import URP Package \doxylink{namespace_samples}{Samples} into your project, see \href{package-samples.md\#importing-package-samples}{\texttt{ Importing package samples}}.

Each example uses its own \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8render-pipelines_8universal_0d14_80_88_2_do6a4fd81cefe986341b093c28ca9c1f30}{URP Asset}{0} so, if you want to build an example scene, \href{InstallURPIntoAProject.md\#set-urp-active}{\texttt{ add the example\textquotesingle{}s URP Asset to your Graphics settings}}. If you don\textquotesingle{}t do this, \doxylink{namespace_unity}{Unity} might strip shaders or render passes that the example uses.
