 To use the package, go to {\bfseries{Edit \texorpdfstring{$>$}{>} Preferences \texorpdfstring{$>$}{>} External Tools}}, click on the {\bfseries{External Script Editor}} dropdown menu and select your version of {\bfseries{\doxylink{namespace_rider}{Rider}}}. After the window reloads, new settings that control production of .csproj files become available.
