\doxysubsection{Detailed Description}
The direct converter is similar to a regular converter, except that it targets specifically only one type. \doxylink{class_unity_1_1_visual_scripting_1_1_this}{This} means that it can be used without performance impact when discovering converters. It is strongly recommended that you derive from \doxylink{class_unity_1_1_visual_scripting_1_1_full_serializer_1_1fs_direct_converter}{fs\+Direct\+Converter}\{TModel\}. 

Due to the way that direct converters operate, inheritance is {\itshape not} supported. Direct converters will only be used with the exact Model\+Type object.
