{\bfseries{Coroutine}} 

Toggle 

Enable {\bfseries{Coroutine}} if you want Visual Scripting to run this node and any of its connected nodes as a coroutine. Coroutine nodes don\textquotesingle{}t execute all their code in a single frame, so they can spread an effect over several frames. Coroutines can also help optimize your code. For more information on coroutines, see the \doxylink{namespace_unity}{Unity} User Manual section on \href{https://docs.unity3d.com/Manual/Coroutines.html}{\texttt{ Coroutines}}.
