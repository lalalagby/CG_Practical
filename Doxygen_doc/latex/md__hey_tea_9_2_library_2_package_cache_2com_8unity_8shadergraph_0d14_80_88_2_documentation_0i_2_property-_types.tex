{\bfseries{Property Types}} are the types of \href{https://docs.unity3d.com/Manual/SL-Properties.html}{\texttt{ Property}} than can be defined on the \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_blackboard}{Blackboard}{0} for use in the {\bfseries{Graph}}. These \href{https://docs.unity3d.com/Manual/SL-Properties.html}{\texttt{ Properties}} are exposed to the \href{https://docs.unity3d.com/Manual/UsingTheInspector.html}{\texttt{ Inspector}} for \href{https://docs.unity3d.com/Manual/class-Material.html}{\texttt{ Materials}} that use the shader.

Each property has an associated {\bfseries{Data Type}}. See \doxysectlink{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_data-_types}{Data Types}{0} for more information.\hypertarget{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_property-_types_autotoc_md3648}{}\doxysection{\texorpdfstring{Common Parameters}{Common Parameters}}\label{md__hey_tea_9_2_library_2_package_cache_2com_8unity_8shadergraph_0d14_80_88_2_documentation_0i_2_property-_types_autotoc_md3648}
