\doxysubsection{Detailed Description}
Use this attribute to customize the appearance of a \doxylink{class_unity_engine_1_1_timeline_1_1_marker}{Marker}. 

Specify the style to use to draw a \doxylink{class_unity_engine_1_1_timeline_1_1_marker}{Marker}. 

How to create a custom style rule\+: 1) Create a \textquotesingle{}common.\+uss\textquotesingle{} USS file in an Editor folder in a Style\+Sheets/\+Extensions folder hierarchy. Example of valid folder paths\+:
\begin{DoxyItemize}
\item Assets/\+Editor/\+Style\+Sheets/\+Extensions
\item Assets/\+Editor/\+Markers/\+Style\+Sheets/\+Extensions
\item Assets/\+Timeline/\+Editor/\+My\+Markers/\+Style\+Sheets/\+Extensions Rules in \textquotesingle{}dark.\+uss\textquotesingle{} are used if you use the Pro Skin and rules in \textquotesingle{}light.\+uss\textquotesingle{} are used otherwise.
\end{DoxyItemize}

2)In the USS file, create a styling rule to customize the appearance of the marker.
