\doxysubsection{Detailed Description}
Base class the Sprite Editor Window custom module inherits from. 

Sprite Editor Window functionality can be extended by providing custom module. By inheriting from \doxylink{class_unity_editor_1_1_u2_d_1_1_sprites_1_1_sprite_editor_module_base}{Sprite\+Editor\+Module\+Base}, user will be able to activate the module\textquotesingle{}s functionality from Sprite Editor Window.
