Splits the input vector {\bfseries{In}} into four {\bfseries{Float}} outputs {\bfseries{R}}, {\bfseries{G}}, {\bfseries{B}} and {\bfseries{A}}. These output vectors are defined by the individual channels of the input {\bfseries{In}}; red, green, blue and alpha respectively.
